## 编码规则

型号格式：`A1DCJ1000-00-00`
- 第1位=款式，第2位=迭代，第3-5位=机型代码，第4位数字=规格
- 后缀：-00-00=成品，-01-00=组合件，-00-01=零件，AD=单相220V/60Hz出口

## 电压规格

| 后缀 | 含义 |
|------|------|
| 无 | 默认电压 |
| C | 单相220V |
| 三相380V带零线 | 三相380V |
| AD | 单相220V/60Hz（出口规格）|
