# 南美市场开发信示例

本目录包含针对南美市场（巴西、智利、阿根廷）的外贸开发信模板，供 HOLO Industrial Equipment Mfg Co., Ltd（温州红龙工业设备）使用。

## 文件说明

| 文件 | 用途 | 语言 |
|------|------|------|
| `cold-email-day0-pt.md` | 首次开发信 | 葡萄牙语（巴西） |
| `cold-email-day0-es.md` | 首次开发信 | 西班牙语（智利/阿根廷） |
| `quote-email-es.md` | 报价跟进邮件 | 西班牙语 |
| `followup-sequence.md` | Day3/7/14 跟进序列说明 | 双语 |

## 使用方法

1. 复制模板文件，重命名为实际客户公司名
2. 将 `[公司名]` `[决策人姓名]` `[具体产品需求]` 等占位符替换为真实信息
3. 根据客户所在国家选择对应语言版本
4. 邮件签名固定使用 Wike Chen 联系方式

## 南美市场简报

### 巴西（Brazil）

- **市场地位**：南美最大市场，工业设备需求强劲
- **语言**：葡萄牙语（Portuguese）
- **沟通偏好**：WhatsApp-first（WhatsApp 极为普及，电话/语音消息效果好）
- **税务要点**：ICMS（州税）、ST（Substituicao Tributaria）复杂，报价需预留税务空间
- **时区**：GMT-3，比中国晚 10-12 小时
- **建议**：开发信后 24-48 小时内通过 WhatsApp 跟进

### 智利（Chile）

- **市场特点**：南美最开放市场，贸易壁垒低，USD 接受度高
- **语言**：西班牙语（Spanish）
- **货币**：CLP 披索，可接受 USD 报价降低风险
- **沟通偏好**：Email + WhatsApp 并行，商务邮件规范
- **时区**：GMT-3/4
- **建议**：智利客户决策较快，适合作为南美突破口

### 阿根廷（Argentina）

- **市场特点**：外汇管制严格，ARS 比索波动剧烈
- **语言**：西班牙语（Spanish）
- **货币风险**：USD/ARS 汇率波动大，建议 USD 报价或设置汇率有效期
- **支付建议**：优先 L/C USD 或部分预付，避免汇率损失
- **沟通偏好**：Email + WhatsApp，WhatsApp 优先
- **时区**：GMT-3
- **建议**：报价有效期设短（7天），减少汇率风险
