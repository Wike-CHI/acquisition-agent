## Step 1.5 联系方式获取优先级 ⚠️

**铁律：没有邮箱就不能进入开发信步骤！**

| 优先级 | 方法 | 等级 |
|--------|------|------|
| ⭐⭐⭐⭐⭐ | 官网"Contact Us"表单 | S级：最可靠 |
| ⭐⭐⭐⭐⭐ | 特易详情页"联系方式" | A级：决策人邮箱 |
| ⭐⭐⭐⭐ | LinkedIn个人资料 | B级（需二次验证）|
| ⭐⭐⭐ | 官网Contact页面 | C级：通用邮箱 |
| ⭐⭐ | 邮箱猜测+Hunter验证 | D级：需验证 |

### 验证铁律（3选1必须满足）

1. ✅ 时效性检查：信息发布时间必须是6个月内
2. ✅ 多源验证：LinkedIn + 官网 + 特易（取最新的）
3. ✅ 决策人确认：职位必须是Purchasing Manager/Procurement Director

## Step 3 ICP评分（6维度×100分）

| 维度 | 权重 | 评分依据 |
|------|------|----------|
| 行业匹配度 | 20% | 传送带相关采购占比 ≥50%=10分 |
| 采购能力 | 20% | 海关金额 S≥$1M/A≥$500K/B≥$100K |
| 采购频率 | 20% | 交易次数+最新日期 高频=10分 |
| 客户类型 | 15% | 制造商=10/经销商=8/终端=5 |
| 决策周期 | 15% | 企业规模推断 |
| 决策人联系 | 10% | LinkedIn/官网联系方式 |

**客户等级**：A级≥70分（重点开发）| B级50-69分 | C级30-49分 | D级<30分

**发送门槛**：ICP评分≥75分才发邮件（铁律2）

## Step 4 LinkedIn决策人搜索 ⚠️

**第一步工具是 Exa，不是 LinkedIn MCP！**

```
1. Exa 搜索 → site:linkedin.com + 关键词
2. 提取信息 → 姓名、职位、公司、连接数
3. 优先级排序 → 高/中/低
4. 生成报告 → 决策人报告模板
5. 行动计划 → 连接消息 + 电话脚本
```

## Step 6 开发信打磨流程（v2.0）

**强制打磨流程**：
```
生成 → 第1轮润色(humanize-ai-text) → 评分
  ↓
< 9.0分 → 第2轮润色 → 第2轮评分
  ↓
≥ 9.0分 → 最终版本 ✅
```

**评分标准**（满分10分）：
- 个性化程度 2.0分
- 相关性 2.0分
- 简洁性 2.0分
- 语法质量 2.0分
- 去AI味 2.0分

## 回退方案速查

| 失败场景 | 回退方案 |
|---------|---------|
| 客户发现失败 | 换关键词 → 换渠道 → 保存失败记录 |
| 查重错误 | 本地缓存 → 标记"未查重" |
| 背调失败 | 换渠道 → 标记"信息不足" |
| 决策人找不到 | Exa → Jina Reader → 标记"需人工" |
| 邮件发送失败 | 换邮箱 → 重试队列(1h→2h→4h) |
