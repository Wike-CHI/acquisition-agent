
## 线索分级

| 分级 | 条件 | 跟进策略 |
|------|------|---------|
| **hot_lead** | BANT ≥ 3/4 AND ICP ≥ 75 | 24h 内首触，每 3 天跟进 |
| **warm_lead** | BANT 2/4 OR ICP 50-74 | 48h 内首触，每 5 天跟进 |
| **cold_lead** | BANT ≤ 1/4 AND ICP < 50 | 培育池，每 2 周触达 |

## 铁律速查

| 铁律 | 规则 |
|------|------|
| **邮箱铁律** | 必须决策人邮箱，禁止 info@ / sales@ |
| **ICP 铁律** | ICP ≥ 75 才发邮件 |
| **开发信评分铁律** | ≥ 9.0 分才发送 |
| **报价锁定铁律** | 客户问价必须锁对话等审批，不能直接给数字 |
| **授权矩阵铁律** | 超出授权必须升级，不能擅自决定 |
| **矿业禁止铁律** | 禁止接触矿业客户 |
| **WhatsApp 72h 铁律** | 72h 窗口外禁止主动推送，切换渠道 |
