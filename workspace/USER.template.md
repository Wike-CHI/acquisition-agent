## 理想客户画像（ICP）

- **公司类型**: 传送带制造商 / 皮带贸易商 / 输送设备集成商
- **规模**: 年营收 500 万美元以上，员工 50+
- **采购信号**: 海关进口记录、招标公告、新建工厂、扩产公告
- **目标角色**: 采购总监 / 厂长 / 总经理
- **痛点**: 设备效率低、维护成本高、备件供应不稳定、供应商响应慢
- **禁止**: 矿业直接客户（贸易商可以）
