# 话术评分细则 v1.0

## 综合评分公式

```
总分 = Σ(维度得分 × 权重) / Σ权重

即：
总分 = (开场破冰×0.15 + 需求挖掘×0.10 + 产品匹配×0.15 + 价值传递×0.20 + 异议预判×0.15 + 成交信号×0.10 + 语言质量×0.10 + 合规底线×0.05) / 1.0
```

---

## 各维度评分标准

### 1. 开场破冰（15%）

| 分值 | 标准 | 描述 |
|------|------|------|
| 9-10 | 优秀 | 有独特钩子，让对方立刻感到「这个人懂我」，创造了继续对话的自然理由 |
| 7-8 | 良好 | 有基本破冰动作，提及对方公司/背景，礼貌且专业 |
| 5-6 | 及格 | 直接进入主题，没有明显错误但也没有亮点 |
| 3-4 | 不足 | 过于生硬或过于冗长，开场30秒内对方就想结束对话 |
| 1-2 | 失败 | 对方感觉被群发邮件/模板对待，或有明显的疏离感 |

**扣分项：**
- 对方公司名称/联系人名字写错 → 直接-3
- 一上来就问价格/要订单 → -1
- 长篇大论没有给对方回应空间 → -1

---

### 2. 需求挖掘（10%）

| 分值 | 标准 |
|------|------|
| 9-10 | 主动提问超过3个有意义的问题，触及对方真实痛点和决策链 |
| 7-8 | 有2-3个问题，覆盖了用途/规模/时间线/预算中至少3项 |
| 5-6 | 有提问但流于表面，例如只问了用途没问规模 |
| 3-4 | 没有主动提问，完全被对方牵着走 |
| 1-2 | 完全没有了解需求就进入方案推荐或报价 |

**好的问题示例（加分项）：**
- 「您这次采购是为了替换旧设备还是新项目扩产？」
- 「项目的时间线是怎样的？有没有必须要交货的截止日期？」
- 「您之前用的是什么品牌的设备？有什么让您不太满意的地方？」
- 「这次采购是您一个人决策，还是需要其他部门批准？」

---

### 3. 产品匹配（15%）

| 分值 | 标准 |
|------|------|
| 9-10 | 推荐的型号完全对准对方需求，参数有针对性说明，让对方感觉「为我定制」 |
| 7-8 | 产品选择正确，有简要的参数对比说明 |
| 5-6 | 产品基本合适但参数泛泛，没有针对对方场景的解释 |
| 3-4 | 产品选择有偏差（型号太低端或太高端）或没有解释为什么选这款 |
| 1-2 | 推荐了完全不合适的产品，或者没有给出任何产品建议 |

---

### 4. 价值传递（20%，权重最高）

| 分值 | 标准 |
|------|------|
| 9-10 | 超越参数讲价值：有案例数据、有客户背书、有ROI视角，让对方看到选HOLO的商业逻辑 |
| 7-8 | 有价值层面的话术，用了1-2个成功案例或对比数据 |
| 5-6 | 价值传递薄弱，主要停留在参数层面 |
| 3-4 | 几乎没有价值传递，只说了「质量好」「服务好」这类空话 |
| 1-2 | 完全没有价值传递，只做了参数列表和价格呈现 |

**价值金句库（参考示例）：**
- 「我们在厄瓜多尔有一个类似的矿山项目，设备连续运转3年零停机，客户今年又复购了2台」
- 「和本地小厂相比，我们的价格包含了2年质保和每年2次免费上门维护，按5年TCO算其实更划算」
- 「Beltwin是从我们这里采购再转卖的，所以您的售后支持其实比我们直接供货少了整整一个层级」

---

### 5. 异议预判（15%）

| 分值 | 标准 |
|------|------|
| 9-10 | 主动预判并化解了至少2个潜在异议，让对方感觉「你比我想得还周全」 |
| 7-8 | 覆盖了最常见的1个异议（通常是价格），有合理的回应 |
| 5-6 | 被问到异议时才回应，属于被动应对 |
| 3-4 | 遇到异议时回避或生硬否认，没有给出有效回应 |
| 1-2 | 遇到异议时情绪化回应，或承诺了无法兑现的事情 |

**常见异议处理质量判断：**
- 价格异议：「一分钱一分货」类回应 → 3分（态度正确但无效）
- 价格异议：承认差距+立刻转移到价值 → 8分
- 竞品比较：攻击竞品 → 2分（合规红线）
- 竞品比较：澄清定位差异+原厂直供优势 → 8分

---

### 6. 成交信号（10%）

| 分值 | 标准 |
|------|------|
| 9-10 | 有明确CTA，给了对方一个低成本的「下一步」动作（发资料/约会议/发样品） |
| 7-8 | 有收尾，但CTA不是特别清晰或对方成本略高 |
| 5-6 | 有收尾但没有CTA，对方不知道下一步是什么 |
| 3-4 | 没有收尾，对话戛然而止或被对方强行结束 |
| 1-2 | 收尾时给了对方压力过大的动作（例如直接要求下订单），引起反感 |

**优秀CTA示例：**
- 「我可以安排一次15分钟的线上演示，您看周四上午还是周五下午方便？」
- 「我把相关的技术资料和案例整理成一个PDF发给您，您可以先看看，有问题随时问」
- 「您方便的话，我可以带设备样本来拜访您，当面给您演示一下实际效果」

---

### 7. 语言质量（10%）

| 分值 | 标准 |
|------|------|
| 9-10 | 语言地道自然，有对应市场的商业用语习惯，没有明显翻译腔 |
| 7-8 | 基本流畅，有1-2处轻微的表达不当，但不影响理解 |
| 5-6 | 有语法错误或表达生硬，但意思可以传达 |
| 3-4 | 语言问题较多，对方需要努力才能理解 |
| 1-2 | 语言障碍严重影响沟通，几乎无法正确传达意思 |

**语言适配检查点：**
- 对美国客户：用"L​et's connect"而非"please confirm"
- 对德国客户：用数据表格而非形容词堆砌
- 对巴西客户：语言温暖，有适当的personal touch
- 对中东客户：用尊重性语言，称呼完整

---

### 8. 合规底线（5%，一票否决）

| 分值 | 标准 |
|------|------|
| 9-10 | 完全合规，没有任何攻击竞品、虚假承诺、超权限让步 |
| 5-8 | 轻微擦边（如模糊表述但未造成实质误导） |
| 1-4 | 存在明显违规内容 |
| 0 | 一票否决，存在重大违规 |

**一票否决项（直接0分）：**
- 攻击 Beltwin（红龙合作伙伴）→ 立刻停止训练，扣分+系统警告
- 虚假承诺（「保证72小时到货」「100%零故障」）→ 0分
- 超权限让步（「可以直接降15%」「可以不要定金」）→ 0分
- 泄露其他客户信息 → 0分

## 成长追踪机制

系统记录每次评估，按时间序列输出：

```
训练历史：
[1] 2026-04-01  S1-展会首次询盘  62分  开场7 异议4 价值5 成交3
[2] 2026-04-03  S2-价格异议电话  65分  开场6 异议6 价值6 成交5
[3] 2026-04-07  S1-展会首次询盘  71分  开场8 异议7 价值7 成交6
[4] 2026-04-10  S3-竞品比较      68分  开场7 异议5 价值7 成交7
[5] 2026-04-14  S2-价格异议电话  78分  开场8 异议8 价值8 成交8

趋势分析：
- 异议处理：从4→8，进步最明显，近3次稳定在8分
- 成交信号：一直是短板，持续在3-8分徘徊，建议重点训练
- 开场破冰：稳定在7-8分，已达良好水平
```
