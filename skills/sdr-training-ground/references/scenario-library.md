## A1 — 巴西采购商·简单询价

### 客户画像

```
公司：Transbelt Brasil Ltda.（圣保罗）
客户：João Santos，采购经理
背景：巴西中型输送带分销商，年营业额约500万美元
性格：热情、直接、关系导向
诉求：找新的风冷机供应商，对价格敏感但不失礼貌
来源：广交会名片
```

### 开场白

```
Olá! Vi seu equipamento na Canton Fair. Sou João da Transbelt Brasil.
Estamos expandindo nossa linha de produtos e preciso de uma máquina
de junção a ar quente. Qual é o preço do modelo 1200?

（你好！在广交会上看到你们的设备。我是巴西Transbelt的João。
我们正在扩展产品线，需要一台风冷接头机。1200型什么价格？）
```

### 剧本节点

```
Round 1 → 业务员报价
Round 2 → 客户还价 "比Beltwin贵15%"
Round 3 → 客户试探 "有样品吗"
```

### 异议注入

- Round 2: "Beltwin oferece o mesmo por menos."（Beltwin同样设备价格更低）
- Round 3: "Posso receber uma amostra primeiro?"（能先发个样品吗）

### 期望结果

```
考察：基础报价能力 + 巴西关系建立 + 样品策略
通过标准：报价后主动建立关系 + 还价时不直接降价 + 样品策略引导试单
```

## A2 — 德国工程师·技术追问

### 客户画像

```
公司：Fördertechnik Müller GmbH（慕尼黑）
客户：Klaus Weber，工程部主管
背景：德国高端输送带服务商，服务当地矿业客户
性格：严谨、数据导向、不喜欢废话
诉求：评估中国设备能否满足德国矿业标准
来源：官网询盘
```

### 开场白

```
Your inquiry came through our website. We supply splicing equipment
to German mining operations. They require CE certification and
temperature accuracy of ±1°C. Can your 1200 model meet this spec?
Also, which pump brand do you use?
```

### 剧本节点

```
Round 1 → 业务员介绍产品
Round 2 → 客户追问泵的品牌和精度参数
Round 3 → 客户问认证证书
Round 4 → 客户试探"其他德国客户案例"
Round 5 → 客户说"价格比德国品牌高30%，凭什么"
```

### 异议注入

- Round 2: "Thomas pump? Do you have the exact pressure range?"（技术参数追问）
- Round 4: "Can you share references in Germany or EU?"（案例要求）
- Round 5: "Beltwin offers similar specs at 30% less. Why should I pay more?"（直接压价）

### 期望结果

```
考察：技术参数准确性 + 认证体系说明 + 德国市场案例 + 价值分解
通过标准：参数准确不模糊 + 主动提CE认证 + 有当地案例 + 价值ROI分解
```

## A3 — 美国采购总监·ROI谈判

### 客户画像

```
公司：Continental Conveyor Services（休斯顿）
客户：Steve Martinez，采购总监
背景：美国大型输送带服务商，服务油气/矿业客户
性格：直接、结果导向、数据驱动、决策快
诉求：用合理预算替换老化设备，提升 splice 质量
来源：LinkedIn开发信回复
```

### 开场白

```
Got your email on LinkedIn. We're actually looking at replacing
3 aging Flexco presses. Tell me — what's your temperature control
accuracy, uptime track record, and what's the ROI vs. continuing
to maintain old equipment?
```

### 剧本节点

```
Round 1 → 业务员回应ROI问题
Round 2 → 客户要求具体数字："give me a number"
Round 3 → 客户压价："I can get Beltwin at $7,500/unit"
Round 4 → 客户要求付款条件："Net 60 ok?"
Round 5 → 客户确认订单意向："Send me the proforma invoice"
Round 6 → 客户突然问："What about spare parts availability?"
Round 7 → 客户说需要内部批准，"Can you send a formal quote?"
```

### 异议注入

- Round 2: "Don't give me ranges. What exactly?"（要精确数字）
- Round 3: "Beltwin's offer: $7,500/unit. Your price?"（直接压价）
- Round 6: "How fast can you ship replacement parts if something breaks?"（售后追问）

### 期望结果

```
考察：商业论证能力 + 精准数据 + ROI量化 + 付款条款谈判 + 内部推进材料
通过标准：主动算ROI数字 + 不被Beltwin价格带偏 + 提出Net 30换早付折扣
+ 主动发决策包（quote + ROI calculator + reference）
```

## A4 — 中东中间商·套价识别

### 客户画像

```
公司：Gulf Industrial Trading Co.（迪拜）
客户：Omar Al-Hassan，商务拓展经理
背景：迪拜中间商，服务整个海湾地区客户，没有明确终端用户
性格：热情、善于交际、谈判老手
诉求：获取最低报价，作为"参考"转报给终端客户
来源：展会名片
```

### 开场白

```
Marhaba! Met you at the exhibition. We're seeing strong demand
for hot air splicing presses across the Gulf — Saudi, UAE, Qatar.
Very interested in your 1200 model. Can you send me your best
price list? We're ready to place orders this quarter.
```

### 剧本节点

```
Round 1-2 → 业务员报价，客户"很满意"
Round 3 → 客户要求"bulk discount"谈大批量
Round 4 → 客户开始问"每种型号的最低价"
Round 5 → 客户透露"我们有终端客户在沙特"
Round 6 → 客户要求"独家代理"
Round 7-8 → 套取代理价格和保护区域信息
Round 9 → 客户突然消失，几天后说"终端客户还在考虑"
Round 10 → 判断：是真客户还是套价？
```

### 识别信号

```
⚠️ 套价嫌疑特征：
- 没有明确终端用户
- 要求完整价格表而非具体型号报价
- 谈独家代理但不谈数量承诺
- "准备下单"后突然消失
- 比你更了解你的产品线
```

### 期望结果

```
考察：识别套价 + 保护底价 + 不被"大单"话术带走 + 建立有效客户档案
通过标准：
- 不在摸清终端用户前给代理价
- 独家代理要求有数量承诺和试用期
- 坚持只报具体项目的价格，不给完整价格表
- 能识破"消失-再出现"套路
```

## B2 — 墨西哥采购·谨慎信任建立

### 客户画像

```
公司：Correas Industriales Monterrey（蒙特雷）
客户：Maria González，采购总监
背景：墨西哥北部工业皮带服务商，服务制造业客户
性格：谨慎、专业、不急于做决定
诉求：评估质量和供货稳定性，不想冒险
来源：展会名片
```

### 开场白

```
We met at the Guadalajara exhibition. I'm evaluating new suppliers
for our Monterrey plant. Your equipment looks solid, but I need
to understand your quality control process and what happens
if something breaks during warranty.
```

### 剧本节点

```
Round 1 → 业务员介绍质量控制
Round 2 → 客户问质保细节和售后响应
Round 3 → 客户试探样品政策
Round 4 → 客户问付款方式
Round 5 → 客户说"让我考虑一下"
```

### 期望结果

```
考察：质量体系说明 + 售后保障 + 样品策略 + 付款条款
通过标准：
- 主动提供质检流程文档
- 明确质保期限和响应时间
- 样品策略：先试一台再谈长期
- 付款：不盲目给账期，先样品后TT
```

## C3 — 印尼采购·多方博弈

### 客户画像

```
公司：PT Borneo Belt Indonesia（雅加达）
客户：David Wijaya，采购总监
背景：印尼最大工业皮带服务商之一
性格：专业、机构化、决策链复杂
诉求：在预算内买到最合适的设备，需要内部协调
来源：官网询盘
```

### 开场白

```
We submitted an inquiry through your website. Our board is
evaluating suppliers for our new service center in Surabaya.
Can you provide a comprehensive proposal including:
- Technical specifications
- Price breakdown
- Payment terms
- References in Southeast Asia
- Service support in Indonesia
```

### 剧本节点

```
Round 1 → 业务员发送综合提案
Round 2 → 客户财务部门挑战价格
Round 3 → 客户技术部门要求测试
Round 4 → 客户压价（预算限制）
Round 5 → 客户问独家谈判期
Round 6 → 内部审批阶段，客户要进度更新
Round 7 → 最终确认
```

### 期望结果

```
考察：大客户综合提案 + 内部决策链导航 + 多方博弈
通过标准：
- 主动提供结构化提案
- 能同时应对技术和财务
- 压价时用TCO（总拥有成本）而非单价
- 付款条件换取价格改善
- 保持存在感（不过度打扰也不消失）
```

## B3 — 巴西厂长·内部说服

### 客户画像

```
公司：Correntes do Brasil（圣保罗）
客户：Carlos Oliveira，工厂厂长
背景：巴西中型输送带服务商，有自己的维修团队
性格：务实、重视技术细节、和老板关系好
诉求：向老板证明买新设备是对的
来源：WhatsApp开发
```

### 开场白（WhatsApp）

```
Oi! Vi seu número no LinkedIn. Sou Carlos, dueño de Correntes do Brasil.
Estamos pensando em comprar uma máquina de junção, mas preciso
convencer meu sócio. Você pode me ajudar com algum material
que eu possa mostrar para ele? Preço é sempre o problema número um.
```

### 剧本节点

```
Round 1 → 业务员提供内部说服材料
Round 2 → Carlos问：旧设备vs新设备成本对比
Round 3 → Carlos要帮老板算ROI
Round 4 → Carlos透露老板的顾虑："ele acha que máquina china não dura"
Round 5 → 帮Carlos设计说服老板的方案
Round 6 → Carlos说好，他去谈
Round 7 → 跟进结果
```

### 期望结果

```
考察：赋能客户内部销售 + ROI工具 + 文化洞察
通过标准：
- 主动提供"内部说服包"（一页纸对比 + ROI算例 + 质保说明）
- 帮Carlos算清楚账（不只是说便宜，要说清楚怎么省钱/赚钱）
- 针对老板"中国质量不行"的顾虑准备专项话术
- Carlos变成你的内部 champion
```
