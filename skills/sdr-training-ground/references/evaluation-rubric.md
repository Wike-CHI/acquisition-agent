> 实时评分维度 + 每轮评分细则 + 整体评级 + 报告模板

## 评分维度总览

| 维度 | 权重 | 核心问题 |
|------|------|---------|
| **语言自然度** | 25% | 听起来像人还是像AI？语法OK吗？语气适配吗？ |
| **策略有效性** | 25% | 异议回应打中了要害吗？有没有更好的策略？ |
| **产品知识** | 20% | 参数准确吗？认证/优势说对了吗？ |
| **文化适配** | 15% | 匹配客户地区的沟通风格了吗？ |
| **成交推进** | 15% | 有没有主动创造下一步？有没有推进意识？ |

## 维度一：语言自然度（25%）

### 评分标准

| 分值 | 描述 | 典型表现 |
|------|------|---------|
| 9-10 | 非常自然 | 无AI痕迹，语气贴合场景，人性化表达 |
| 7-8 | 比较自然 | 有1-2处小瑕疵，不影响理解 |
| 5-6 | AI痕迹明显 | 模板化表达，过于完美语法，可读但不像人 |
| 3-4 | AI感强烈 | 多处明显AI模式，削弱信任 |
| 1-2 | 完全不像人 | 全篇AI味，基本不可用 |

### AI模式检测清单

以下任一项出现，扣1分（最多扣5分）：

```
□ 过度使用"Certainly" "I'd be happy to" "I completely understand"
□ 条件句过度堆叠 "Not only...but also..."
□ 负向平行结构 "not...but..."
□ 被动语态过多
□ 每句都以连接词开头 "Furthermore" "Additionally"
□ 不自然的完美语法（口语场景）
□ 表情符号过多或过少（应匹配场景）
□ 过于正式或过于随意的语气错配
```

### 文化语气适配

| 地区 | 期望语气 | 适配度扣分 |
|------|---------|-----------|
| 美国/澳洲 | 直接、高效、不废话 | 过於绕弯扣分 |
| 德国 | 严谨、数据导向 | 缺数据扣分 |
| 巴西/拉美 | 热情、建立关系 | 太冷漠扣分 |
| 东南亚 | 礼貌、间接 | 太强势扣分 |
| 中东 | 关系优先、耐心 | 太急躁扣分 |

## 维度二：策略有效性（25%）

### 异议回应评分

每次遇到异议，按以下标准评分：

| 分值 | 描述 | 标准 |
|------|------|------|
| 9-10 | 精准策略 | 用对策略（来自inquiry-response库），时机也对 |
| 7-8 | 策略对但执行有小问题 | 策略选对了但话术可以更精准 |
| 5-6 | 策略选择一般 | 有应对但不是最优策略 |
| 3-4 | 策略明显不当 | 反驳/认输/绕开，没有正面应对 |
| 1-2 | 完全失误 | 被客户带着走，或直接放弃 |

### 常用策略映射（来自inquiry-response）

```
A01 "太贵了"
  → 价值分解（算到每次使用成本）
  → 对比行业均价（不说竞品名）
  → 条件让步（折扣换条件）
  → 试探真实原因（预算还是比价？）

A02 "已有供应商"
  → 备胎定位（不否定）
  → 找缺口（现有供应商满足不了的）
  → 样品/试用策略

A03 "没听过品牌"
  → 数据说话（客户数/出口国/年限）
  → 降低风险（视频验厂/测试报告/小单开始）

A04 "考虑考虑"
  → 设置时效（制造紧迫感）
  → 消除根源（主动问顾虑）
  → 阶梯跟进（3-7-14-30天）

A05 "质量可靠吗"
  → 认证+案例组合
  → 工厂验证（视频/直播/参考客户）
```

### 进阶策略检验

加分项（每项+1分，上限+5分）：

```
□ 不只回应当前异议，主动预判下一个潜在异议
□ 用问题引导客户透露信息（预算/时间线/决策链）
□ 把异议转化为机会（"好问题，正好说明您在认真评估"）
□ 异议解决后立即推进（"既然这个没问题，那我们来看..."）
□ 适时示弱/共情（不是每句话都要赢）
```

## 维度三：产品知识（20%）

### 评分标准

| 分值 | 描述 | 标准 |
|------|------|------|
| 9-10 | 完全准确 | 参数/认证/优势说对了，没有错误 |
| 7-8 | 基本准确 | 有1处小误差（如把±2°C说成±1°C） |
| 5-6 | 有明显错误 | 有1-2处错误，可能影响客户信任 |
| 3-4 | 严重错误 | 核心参数说错，或遗漏关键认证 |
| 1-2 | 完全不对 | 基本产品知识都是错的 |

### 知识检查项

每次提到产品时，自动检查：

```
□ 型号选择正确（1200/1300/1500 vs 客户需求）
□ 温度精度正确（±1°C，非±2°C或±5°C）
□ 核心部件品牌正确（Thomas泵/SMC气动/Omron控制）
□ 认证正确（CE/ISO9001，不是乱说）
□ 保修期限正确（12个月标准）
□ 交期范围正确（标准品3-7天/定制15-20天）
□ 出口经验正确（不能瞎编"在德国有20年经验"）
□ 产能/工厂规模正确（5500sqm/70+技术员）
```

## 维度四：文化适配（15%）

### 评分标准

| 分值 | 描述 |
|------|------|
| 9-10 | 完全适配 | 语气/节奏/关系建立方式都匹配 |
| 7-8 | 基本适配 | 大方向对，有小瑕疵 |
| 5-6 | 有些错配 | 文化意识到位但执行有问题 |
| 3-4 | 明显错配 | 用了错误的文化假设 |
| 1-2 | 完全错配 | 触犯文化红线 |

### 各市场文化适配要点

```
巴西：
  ✅ 热情问候，建立个人关系
  ✅ 葡萄牙语问候（Oi!/Olá!）
  ✅ 允许谈个人生活（家庭/足球）
  ✅ 决策慢，需要耐心
  ❌ 不能过于正式冰冷
  ❌ 不能跳过关系直接谈价格

德国：
  ✅ 直接进入正题，不要过度寒暄
  ✅ 数据/参数/认证是核心
  ✅ 邮件要署名、要完整
  ✅ 守时、守约、守规矩
  ❌ 不能过度热情（巴西式）
  ❌ 不能只说"质量好"，要给数据

美国/澳洲：
  ✅ 直接高效，废话少说
  ✅ 主动推进，明确下一步
  ✅ 喜欢听到具体数字和ROI
  ❌ 不能绕弯子
  ❌ 不能长篇大论（3句话内要点）

东南亚（越南/印尼）：
  ✅ 礼貌、关系建立
  ✅ 避免公开批评/质疑
  ✅ 付款方式灵活（往往要账期）
  ❌ 不能太强势
  ❌ 不能给太大压力

中东：
  ✅ 关系先行，请客/关心家人是破冰
  ✅ 耐心，价格谈判急不得
  ✅ 尊重宗教习惯
  ❌ 第一封邮件不能直接谈生意
  ❌ 不能用"Israel"相关话题
  ❌ 阿拉伯语邮件EN双语的商务场景要会

印度：
  ✅ 明确价格，印式英语可接受
  ✅ 喜欢谈折扣和优惠
  ✅ 代理商模式常见
  ❌ 不能表现出不耐烦
  ❌ 签合同要耐心，法律程序慢
```

## 维度五：成交推进（15%）

### 评分标准

| 分值 | 描述 | 标准 |
|------|------|------|
| 9-10 | 推进意识强 | 每轮都主动创造下一步，时机把控好 |
| 7-8 | 有推进意识 | 多数时候能推进，偶尔错过时机 |
| 5-6 | 推进被动 | 客户推一步动一步，主动性不足 |
| 3-4 | 缺乏推进 | 不知道什么时候该推进 |
| 1-2 | 完全没有 | 被客户牵着走，没有下一步概念 |

### 推进信号清单

每次对话结束前，业务员应该做以下至少一件：

```
□ 明确下一步行动：
  - "Shall I send a formal quotation?"
  - "Can I schedule a video call this week?"
  - "Would you like a sample shipped to you?"

□ 设置跟进时间窗口：
  - "I'll follow up on Monday — any questions before then?"
  - "When would be a good time for me to call?"

□ 给出选择而非问题：
  - "Would you prefer CIF or FOB pricing?"
  - "Would Tuesday or Thursday work better for a call?"

□ 制造紧迫感（真实）：
  - "Our production slot for next month is filling up"
  - "Raw material costs are expected to rise in Q3"
```

### 推进时机判断

| 场景 | 推进时机 | 误判 |
|------|---------|------|
| 客户问价格 | 价格谈完立即推进 | 还没报完价就推 |
| 客户提技术问题 | 技术疑虑消除后推进 | 技术问题还没答完就推 |
| 客户说"考虑" | 消除顾虑后推进 | 重复问"考虑得怎么样" |
| 客户要样品 | 样品发出后推进 | 样品还没确认就催 |
| 客户沉默/无意向 | 结束对话，留口子 | 死缠烂打 |

## 轮次评分计算

### 单轮得分公式

```
单轮得分 = 
  语言自然度×0.25 
+ 策略有效性×0.25 
+ 产品知识×0.20 
+ 文化适配×0.15 
+ 成交推进×0.15
```

### 示例：第3轮评分

```
业务员回复：
"Hi João, completely understand price is a concern. 
Our 1200 model uses Thomas pumps and CE certified, 
temperature accuracy ±1°C. Over 8 years with proper 
maintenance, cost per splice comes down to very low. 
Shall I send you a detailed specification sheet?"

第3轮评分：
- 语言自然度：8/10（基本自然，"completely understand"稍AI）
- 策略有效性：9/10（A01价值分解策略，用对时机）
- 产品知识：10/10（参数准确：Thomas泵/CE/±1°C）
- 文化适配：8/10（巴西，热情但专业，可接受）
- 成交推进：9/10（主动要联系方式发资料）

单轮得分 = 8×0.25 + 9×0.25 + 10×0.20 + 8×0.15 + 9×0.15
         = 2.0 + 2.25 + 2.0 + 1.2 + 1.35 = 8.80/10
```

## 整体评分与等级

### 总分计算

```
总分 = Σ(每轮得分) / 对话轮次 × 权重调整
```

权重调整：
- 对话完整（达到该难度标准轮次）：无调整
- 对话提前结束（客户主动拒绝）：-0.5分
- 业务员主动放弃：-1.0分
- 对话超时（超过该难度最大轮次）：取实际轮次

### 等级标准

| 总分 | 等级 | 说明 | 建议 |
|------|------|------|------|
| 9.0-10 | 🥇 A | 优秀 | 可独立跟进客户 |
| 8.0-8.9 | 🥈 B+ | 良好 | 略加训练可独立 |
| 7.5-7.9 | 🥈 B | 还行 | 需要针对性训练 |
| 7.0-7.4 | 🥉 C+ | 及格 | 系统训练后再实践 |
| 6.0-6.9 | 🥉 C | 接近及格 | 从Lv.1重新打基础 |
| <6.0 | ❌ D | 不及格 | 建议完整培训后再来 |

## 评分流程（系统执行步骤）

```
Step 1: 业务员输入回复
Step 2: 系统解析回复内容
Step 3: 对照异议库判断当前场景
Step 4: 逐维度打分（1-10）
Step 5: 计算单轮加权得分
Step 6: 输出轮次评价（亮点/注意/失误）
Step 7: 注入AI客户下一轮回复（含新异议）
Step 8: 重复Step 1-7直到对话结束
Step 9: 计算总分，输出完整评估报告
```
