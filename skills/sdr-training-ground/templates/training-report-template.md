# 训练评估报告模板

```
================================================================================
                    SDR TRAINING GROUND — 评估报告
================================================================================

【基本信息】
训练场景：{scenario_code} — {scenario_name}
难度等级：Lv.{level} ({difficulty_name})
对话轮次：{total_rounds}轮 {status}
总体评分：{overall_score}/10 {grade_emoji} {grade_text}
训练日期：{training_date}
训练用时：{training_duration}分钟

【雷达图】（文字版）
{radar_chart_text}

【轮次得分明细】
{rounds_detail}

【亮点清单】✅
{strengths_list}

【改进清单】⚠️
{improvements_list}

【重点改进项】
{priority_items}

【参考资料推荐】
{reference_materials}

【下一步训练建议】
{next_steps}

================================================================================
                           报告生成时间：{report_time}
                    sdr-training-ground v{version}
================================================================================
```

---

## 字段说明

| 字段 | 说明 | 示例 |
|------|------|------|
| `{scenario_code}` | 场景代码 | A2, B3, C4 |
| `{scenario_name}` | 场景名称 | 德国工程师·技术追问 |
| `{level}` | 难度等级 | 1-4 |
| `{difficulty_name}` | 难度描述 | 进阶/高手/地狱 |
| `{total_rounds}` | 对话总轮次 | 5 |
| `{status}` | 完成状态 | （完整）/（客户主动拒绝）/（业务员放弃） |
| `{overall_score}` | 总分（1位小数） | 8.45 |
| `{grade_emoji}` | 等级emoji | 🥇/🥈/🥉/❌ |
| `{grade_text}` | 等级文字 | A/B+/C |
| `{radar_chart_text}` | 雷达图文字版 | 五维得分 |
| `{rounds_detail}` | 每轮得分列表 | Round 1: 8.20... |
| `{strengths_list}` | 亮点列表（3-5条） | 1. 产品参数准确... |
| `{improvements_list}` | 改进列表（3-5条） | 1. 推进时机... |
| `{priority_items}` | 重点改进（最高优先1条） | 🔴 [最高优先]... |
| `{reference_materials}` | 推荐学习资料 | inquiry-response... |
| `{next_steps}` | 下一步建议 | 建议继续Lv.2... |

---

## 雷达图文字版生成规则

```
                        语言自然度
                        {score}/10
                            ▲
                            │
    文化适配                │              策略有效性
      {score}/10 ◄─────────┼─────────────► {score}/10
      ◄────────────────────┼─────────────────►
                            │
                            ▼
                        产品知识
                        {score}/10

                成交推进：{score}/10
```

## 等级与颜色对应

| 等级 | Emoji | 颜色 | 说明 | 建议 |
|------|-------|------|------|------|
| A (9.0-10) | 🥇 | 绿色 | 优秀 | 可独立跟进客户 |
| B+ (8.0-8.9) | 🥈 | 蓝色 | 良好 | 略加训练可独立 |
| B (7.5-7.9) | 🥈 | 蓝色 | 还行 | 需要针对性训练 |
| C+ (7.0-7.4) | 🥉 | 黄色 | 及格 | 系统训练后再实践 |
| C (6.0-6.9) | 🥉 | 黄色 | 接近及格 | 从Lv.1重新打基础 |
| D (<6.0) | ❌ | 红色 | 不及格 | 建议完整培训后再来 |

```
                        语言自然度
                        8.5/10

                            │

                            ▼

                        9.0/10

                成交推进：7.5/10

1. 产品知识扎实：质保条款、核心部件品牌、认证信息均准确

3. 样品策略提出时机合适：在建立一定信任后提出，不早不晚

【改进清单】⚠️
1. 成交推进不足（7.5/10）：第4轮解决付款问题后没有立即推进

🔴 [最高优先] 推进时机把控
   问题：解决了客户疑虑但没有立即推进
   练习：墨西哥客户在"质量信任建立后"是最佳推进时机

🟡 [中等优先] 拉美邮件风格
   问题：当前邮件偏正式，缺乏拉美关系的温度
   练习：邮件开头加葡萄牙语问候，结尾加个人署名

【参考资料推荐】
- inquiry-response/references/multilingual-objections.md (PT-BR章节)
- cultural-psychology.md (拉美市场章节)
- sdr-humanizer (消息拟人化技能)

【下一步训练建议】
建议继续 B3（巴西厂长·内部说服），重点练习：
1. 赋能客户内部销售（帮客户说服其老板）
2. 葡语问候和拉美关系建立
3. Carlos场景的WhatsApp节奏把控

================================================================================
                           报告生成时间：2026-04-14 14:30
                    sdr-training-ground v1.0.0
================================================================================
```
