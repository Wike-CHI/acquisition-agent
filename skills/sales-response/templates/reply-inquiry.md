# 询盘回复模板

> 适用场景：客户主动询问价格、型号、交货期等业务信息。

---

## 标准询盘回复框架（PAS 公式）

```
P - Acknowledge the inquiry (回应/感谢)
A - Address the main question (直接回答)
S - Strengthen with value (补充价值)
```

---
