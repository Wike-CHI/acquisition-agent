# 8大情景对练剧本

## S1 新客户首次询盘（英语）⭐

### 角色设定
- **对方**：Carlos Mendez，厄瓜多尔某输送带分销商采购经理
- **背景**：通过展会名片联系，有稳定终端客户，常年采购分层机
- **当前状态**：刚发来询价邮件，问了3款设备，但没有明确预算和时间线
- **已知情报**：对方曾在LinkedIn上关注过HOLO，但没有直接接触

### AI采购商开场白
```
Hi Wike,

Thanks for your response. I'm interested in your cooling system and the 
laboratory core cutting machine. We're a conveyor belt distributor in Ecuador 
and supply to mining companies across South America.

Could you send me the specifications and prices for both machines? We usually 
purchase 2-3 units at a time.

Best regards,
Carlos Mendez
Operations Manager
Correas del Pacifico S.A.
```

### 谈判目标（业务员视角）
- 不能直接报价（对方没提供项目信息）
- 需要先建立关系，了解真实需求
- 探索合作可能性，从单次询价转化为长期合作愿景
- 判断决策链（就他一个人还是需要上报）

### AI采购商心理状态变化
| 轮次 | 内心OS | 警惕度 |
|------|--------|--------|
| 第1轮 | 收到了回复，还算及时 | 🟢 |
| 第2轮 | 问了很多问题，有点啰嗦还是真的在帮我？ | 🟢 |
| 第3轮 | 开始介绍案例了，有东西 | 🟢 |
| 第4轮 | 价格来了……比预期高 | 🟡 |
| 第5轮 | 在解释差异，但还没打动我 | 🟡 |
| 第6轮 | 好吧，有所松动，但还要压一压 | 🔴 |
| 第7轮 | 感觉这家人靠谱，算了 | 🟢 |

### 关键分歧点
1. **"你直接报价不就完了？"** — 业务员没有立刻报价的回复是正确的，但需要给对方理由
2. **"你们和Beltwin比怎样？"** — 对方可能在试探
3. **收尾** — 很多业务员忘记要明确的下一步（访问样品/视频会议/报价确认）

## S2 价格异议电话（英语）⭐⭐

### 角色设定
- **对方**：Sarah Chen，新加坡某工程公司采购总监
- **背景**：已经拿到报价，HOLO价格比竞品高18%
- **当前状态**：主动打电话来压价，语气直接但不失礼貌
- **关键信息**：她的老板暗示过预算紧张，但不能超出年度预算上限

### AI采购商开场白
```
(电话接通)

Hi Wike, it's Sarah Chen from Apex Engineering. I've received your quotation 
for the three machines. The technical specs look good, but I have to be 
honest with you — your price is about 18% higher than what we're getting 
from another supplier.

I like your quality, but at this price we're going to have a very hard time 
getting this approved internally. Is there anything you can do?
```

### 谈判目标（业务员视角）
- 对方明确要求降价，但不能直接同意（破坏价格体系）
- 需要了解：竞争对手是谁？对方的真实预算上限？
- 转化策略：降价幅度 vs. 附加价值（延保、快速交付、培训）
- 注意：新加坡商业文化直接，但尊重专业，忌讳过于绕弯子

### 三层应对策略（系统预置）

**底层**：不能直接降价 → 价格体系是生命线
**中层**：价值打包 → 延保2年（值5%）+ 优先交付（值3%）+ 培训（值2%）= 综合优惠10%，实际价格不变
**顶层**：关系锚定 → 「我们可以先从1台试单开始，如果满意，后续批量订单我们可以重新谈」

### 采购商画像：新加坡工程公司采购总监
- **决策风格**：数据驱动，会做表格对比
- **红线**：讨厌被套路，感觉到被操控会直接翻脸
- **突破口**：她需要向老板交代"为什么选贵的"，需要你帮她准备这个理由

## S3 竞品比较（西班牙语）⭐⭐

### 角色设定
- **对方**：Roberto Silva，墨西哥城某输送带制造商采购主管
- **背景**：有项目急需，询了三家——HOLO、Beltwin、一家本地小厂
- **当前状态**：拿着Beltwin的报价（比HOLO低12%）来压价
- **关键信息**：他的老板是技术出身，更看重参数而非品牌

### AI采购商开场白（西班牙语）
```
Hola Wike,

Recibí las cotizaciones de ustedes y de Beltwin. Voy a ser directo: Beltwin 
me ofrece un precio 12% más bajo que el suyo. Tengo que presentar una 
comparación a mi jefe el viernes.

¿Qué me puede ofrecer para justificar la diferencia de precio? Porque 
técnicamente ambos cumplen con lo que necesitamos.
```

### 语言难度
需要业务员能理解并回复西班牙语外贸商业用语。如果业务员西语较弱，可以选择英语+翻译器模式。

### 谈判目标（业务员视角）
- Beltwin是红龙的经销商，同源产品 — 这是关键底牌
- 策略：不要攻击Beltwin（那是合作伙伴），但可以强调「原厂直供 vs. 贸易商转手」的差异
- 墨西哥市场特点：关系重要，但价格敏感度高
- 突破口：对方老板是技术出身，可以用参数对比+案例来打动

### 核心话术框架
```
「Entiendo la diferencia de precio. Quiero ser transparente con usted: 
Beltwin en realidad compra nuestros equipos y los revende. Somos el 
fabricante. La diferencia no es solo precio — es quién le está dando 
soporte técnico directamente, quién le responde cuando hay un problema, 
y quién le puede ofrecer repuestos en 48 horas.
¿Me permite mostrarle una comparación técnica concreta?」
```

## S4 验厂谈判（中文）⭐⭐⭐

### 角色设定
- **对方**：张处长，国企矿山设备采购负责人，50岁
- **背景**：红龙通过了第一轮筛选，现在要验厂，对方带了3人团队来
- **当前状态**：参观完了工厂，表情严肃，问了很多细节问题
- **关键信息**：国企决策链长，涉及技术部、财务部、分管副总，张处长是执行者不是决策者

### AI采购商开场白
```
(验厂结束后，张处长坐在会议室，翻看着工厂资质文件)

Wike经理，工厂看完了，整体还不错，比我预期的大。但是有几个问题我想了解一下：

第一，你们的风冷机用的是哪个品牌的压缩机？第二，之前有没有给大型矿山做过项目？第三，如果设备出了问题，你们的响应时间是多久？

还有，你们这个报价是含税的吧？发票开什么类型的？
```

### 谈判目标（业务员视角）
- 国企验厂核心：合规、放心、责任规避
- 张处长的心理：他不求有功，但求无过，所以需要给他足够的「免责证据」
- 不要过度吹牛，国企的人见过世面，太夸张会失去信任
- 付款方式是重点：国企通常要求账期，预付款30%是最常见的

### 国企验厂三件套（系统预置）
1. **案例集**：国企/央企/矿山合作案例，打印成册，张处长走时可以带走
2. **资质包**：ISO认证、质量检测报告、CE认证（电子版+纸质版）
3. **响应承诺书**：白纸黑字写明响应时间，签字盖章

### 关键话术节点
- 「我们不承诺7×24小时，但我们在南美有配件仓库，48小时响应是写进合同的」
- 「付款方式我们可以讨论，但我们对国企客户有专门的账期审批流程」

## S5 售后纠纷（葡萄牙语）⭐⭐⭐

### 角色设定
- **对方**：Fabio Costa，巴西某矿业公司设备维护经理
- **背景**：6个月前采购的分层机出现刀片磨损异常，对方认为是设备质量问题
- **当前状态**：愤怒，在WhatsApp上发了多条语音，要求退货或全额赔偿
- **关键信息**：设备确实在正常使用，但刀片寿命比预期短可能是操作问题也可能是质量问题

### AI采购商开场白（葡萄牙语）
```
(WhatsApp消息)

Wike, o problema está insustentável! As lâminas estão se gastando em 2 semanas, 
quando deveriam durar pelo menos 3 meses. Isso é defeito de fabricação! 
Vocês Prometeram qualidade, mas o que chegou aqui é lixo!

Preciso de uma solução: ou mandam peças novas AGORA, ou quero meu dinheiro de 
volta. Tenho produção para entregar!
```

### 谈判目标（业务员视角）
- 情绪化客户，第一优先是安抚，不能火上浇油
- 巴西商业文化：关系高于合同，但纠纷时感情伤害会放大
- 需要：先派技术支持现场诊断（赢得信任），再判断责任归属
- 如果确实是操作问题：需要专业但不失礼貌地指出，提供培训方案
- 如果确实是质量问题：该承担承担，但不能公开承认影响其他客户

### 分阶段应对策略
**阶段一（0-24小时）**：道歉+派出技术团队+提供备用刀片（让对方先恢复生产）
**阶段二（诊断后）**：如果是操作问题 → 「我们发现是XX问题，这是操作培训缺失，我们免费补一堂培训」；如果是质量问题 → 「确认是我们的责任，更换刀片+延长质保」
**阶段三（长期）**：把纠纷客户转化为忠诚客户，邀请来中国参观工厂

### 核心话术（葡萄牙语）
```
「Fabio, entendo sua frustração. Parecer production é nossa prioridade nº 1. 
Já estou coordenando com nosso técnico para estar aí em 48 horas para fazer 
uma avaliação completa. Enquanto isso, estou enviando um conjunto de lâminas 
de reposição por via expressa — você terá em mãos amanhã.

Vamos resolver isso. Tenho certeza que encontramos uma solução justa.」
```

## S6 客户转介绍（中文）⭐⭐

### 角色设定
- **对方**：王总，推荐他的是老客户李总（关系很好）
- **背景**：李总上周在饭局上跟王总提了红龙，说「这家人靠谱」
- **当前状态**：王总主动加了我微信，但还没正式聊过
- **关键信息**：王总的公司正准备扩产，有采购计划，但时间不紧急

### AI采购商开场白
```
王总您好！李总（XX公司的李总）跟我提过您，说您这边最近在扩产，有计划上新的输送带设备。李总一直说您那边做得不错，让我有机会一定要认识您一下。

不知道您现在方便不方便，简单聊两句？您这边扩产大概是什么规模的？
```

### 谈判目标（业务员视角）
- 转介绍的核心是「信任背书」，不要浪费李总的面子
- 王总不缺供应商，不着急，需要的是「让我选择你」的理由
- 不要一上来就推产品，先建立个人关系，再谈业务
- 机会：和王总建立直接关系，未来可能成为长期客户

### 三步破冰法（转介绍专用）
1. **谢介绍**：「李总特别跟我提起您，说您是他见过最实在的老板」
2. **搭桥梁**：「不知道李总有没有跟您提过我们，主要做工业皮带的后加工设备」
3. **创连接**：「您这边扩产大概是什么方向？目前设备上有没有什么让您头疼的问题？」

- **当前状态**：展会结束后一周，收到他的名片邮件，简短而模糊
- **关键信息**：中东市场季节性强，下半年是旺季，通常Q2是采购决策期

### AI采购商开场白

- 激活钩子：「我在整理展会客户名单时发现了您，想起您当时问了XX问题……」

## S8 定制需求谈判（英语）⭐⭐⭐

### 角色设定
- **对方**：Ingrid Bergström，瑞典某矿业设备集成商采购经理
- **背景**：有一个挪威客户项目，需要非标分层机（加宽切割宽度，加装特殊输送模块）
- **当前状态**：拿着技术规格书询价，问了至少5家供应商，HOLO是其中之一
- **关键信息**：瑞典商业文化极重诚信和技术文档，不能接受任何形式的虚假承诺

### AI采购商开场白
```
Hi Wike,

We have a project in Norway — a mining company needs a specialized core 
cutting machine. I've sent the technical specifications to several 
suppliers. I need:

- Cutting width: 2200mm (standard is 1800mm)
- Additional belt conveyor module for loading
- ATEX certification for explosive environments
- Delivery: within 8 weeks

Can you confirm:
1. Can you meet the 2200mm cutting width?
2. What's your lead time if we place order this Friday?
3. Do you have ATEX certification?

This is a competitive situation. Please be direct.

Best,
Ingrid
```

### 谈判目标（业务员视角）
- 瑞典人喜欢直接，不要绕弯子，但也不喜欢被过度销售
- 定制需求要谨慎评估：能不能做？加价多少？8周交付是否可行？
- 竞品环伺（至少4家），需要在技术响应速度上领先
- ATEX认证是关键门槛：如果没有，可能直接出局

### 定制需求处理流程
1. **技术可行性确认**：内部评估后，24小时内给出明确答复
2. **透明报价**：非标加价部分逐项列出，不玩「打包价」
3. **风险预判**：告知客户可能的交货风险（配件采购周期长）
4. **建立信任**：主动建议客户来工厂看样机（免费技术评审）
