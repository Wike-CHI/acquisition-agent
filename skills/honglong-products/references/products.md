## 十二、编码规则详解

**格式：** `A1DCJ1000-00-00(S)`

| 位置 | 含义 | 说明 |
|------|------|------|
| 第1位字母 | 款式 | A-Z，区分不同设计 |
| 第2位数字 | 迭代 | 1-9，如2=二代，3=三代 |
| 第3-5位字母 | 机型 | DCJ/FLJ/SLJ/CQJ/XDT等 |
| 第4位数字 | 规格 | 如1000=1米 |
| 第5位字母/后缀 | 定制 | 电压、客户要求等 |

**后缀规则：**
- `-00-00` = 成品
- `-01-00` = 组合件/焊接件（半成品）
- `-00-01` = 零件（半成品）
- `-W` = 原材料
- `混色` = 定制颜色
- `WG-01` = 测试件
- `W-DJ-0001` = 外购件
- `CS-` = 常规机改进
- `D1/D2/D3/D4` = 定制版本

**机型代码：**
