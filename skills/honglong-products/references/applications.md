## 一、输送带基础知识

### 1.1 什么是输送带？

**输送带（Conveyor Belt）** 是工业生产中用于传输物料的连续带状设备。

**核心应用行业：**
| 行业 | 应用场景 |
|------|---------|
| 矿业 | 煤炭、金属矿石、砂石运输 |
| 水泥 | 原材料、成品运输 |
| 港口 | 集装箱、散货装卸 |
| 电力 | 燃煤输送 |
| 食品 | 食品原料、成品加工线 |
| 物流 | 快递分拣、仓储运输 |
| 机场 | 行李传输系统 |

### 1.2 输送带的结构

```
┌─────────────────────────────────────┐
│           覆盖胶（上表面）            │  耐磨层
├─────────────────────────────────────┤
│            骨架材料                   │  承受拉力
├─────────────────────────────────────┤
│           覆盖胶（下表面）            │  与滚筒接触
└─────────────────────────────────────┘
```

### 1.3 常见输送带类型

| 类型 | 材质 | 特点 | 适用场景 |
|------|------|------|---------|
| **PVC输送带** | 聚氯乙烯 | 成本低、防滑 | 食品、轻工业、物流 |
| **PU输送带** | 聚氨酯 | 耐磨、耐油、食品级 | 食品加工、医药 |
| **橡胶输送带** | 天然/合成橡胶 | 耐高温、耐磨 | 矿业、冶金、港口 |
| **PVK输送带** | PVC+织物 | 高强度 | 重型物流 |
| **同步带** | 橡胶/聚氨酯+钢丝 | 精确传动 | 机械传动、数控设备 |

### 1.4 输送带为什么会损坏？

| 损坏类型 | 原因 | 后果 |
|---------|------|------|
| **断裂** | 磨损老化、过载、异物划伤 | 生产线停机 |
| **接头开胶** | 硫化不良、温度压力不够 | 接头处断裂风险 |
| **覆盖胶脱落** | 磨损、腐蚀 | 骨架暴露，加速损坏 |
| **跑偏** | 托辊不正、张力不均 | 皮带边缘磨损 |

## 二、为什么要硫化拼接？

### 2.1 输送带维修的两种方式

| 方式 | 说明 | 优缺点 |
|------|------|--------|
| **机械接头** | 用夹板/螺栓连接 | ❌ 强度低、易脱落、不耐磨损 |
| **硫化接头** | 高温高压粘合 | ✅ 强度接近原带、耐磨、平滑 |

### 2.2 硫化拼接的核心原理

**硫化 = 橡胶的"焊接"**

```
硫化过程：
1. 加热 → 橡胶软化（140-150℃）
2. 加压 → 材料融合
3. 保压冷却 → 形成牢固粘接

结果：
- 接头强度达到原带的 85-95%
- 接头处平滑，不伤滚筒
- 使用寿命长
```

### 2.3 接头失败的后果

| 后果 | 影响 |
|------|------|
| 接头断裂 | 生产线停机，可能造成物料洒落 |
| 返工维修 | 增加人工成本和停机时间 |
| 频繁维修 | 影响生产效率，增加维护成本 |
| 安全隐患 | 接头脱落可能造成事故 |

## 三、风冷机 vs 水冷机：选型指南

### 3.1 对比表

| 维度 | 风冷机 | 水冷机 |
|------|--------|--------|
| **原理** | 风机冷却 | 水循环冷却 |
| **加热温度** | 140-150℃ | 140-150℃ |
| **适用皮带** | PVC/PU/薄橡胶带 | 厚橡胶带、重型输送带 |
| **皮带厚度** | ≤10mm | >10mm |
| **优势** | 便携、快速启动 | 温度稳定、适合厚带 |
| **劣势** | 温度波动大 | 需要水源、设备较重 |

### 3.2 选型决策树

```
皮带厚度 ≤ 10mm?
├── 是 → 风冷机
│   ├── 规格 300-1200 → 小型风冷
│   ├── 规格 1500-2100 → 标准风冷
│   └── 规格 2400-3600 → 大型风冷/箱体风冷
│
└── 否 → 水冷机
    ├── 规格 600-1600 → 不锈钢机身
    ├── 规格 2100-3200 → 井字型机身
    └── 规格 3600-4200 → 超大型水冷机
```

### 3.3 不同场景推荐

| 客户类型 | 推荐设备 | 原因 |
|---------|---------|------|
| PVC/PU输送带加工厂 | 风冷机 | 皮带薄，风冷足够 |
| 矿业输送带维修 | 水冷机 | 皮带厚，需要更强加热 |
| 同步带制造厂 | 风冷机（小型） | 规格小，精度要求高 |
| 代理商/维修商 | 风冷+水冷组合 | 应对不同客户需求 |

## 五、产品应用场景汇总

### 5.1 输送带硫化接头机

**用途**：将两根输送带永久性连接（硫化）

**目标客户**：
- 输送带加工厂（加工成品带）
- 矿业/水泥/港口（维修自己的输送带）
- 代理商/维修服务商（为客户提供服务）

**核心价值**：
- 替代进口设备（Flexco/Almex）
- 快速硫化，提高效率
- 稳定可靠，减少返工

### 5.2 输送带打齿机

**用途**：在输送带接头处加工"齿形"，增加粘接面积

**原理**：
```
打齿 = 在皮带端面切割出梯形齿
    ↓
齿与齿之间填充胶料
    ↓
硫化后形成"榫卯结构"
    ↓
接头强度大幅提升
```

**目标客户**：
- 同步带制造厂（高精度齿形）
- 重型输送带维修（高强度要求）

### 5.3 输送带裁切机

**用途**：精确裁切输送带到指定宽度

**目标客户**：
- 输送带加工厂（分裁卷料）
- 定制输送带制造商

### 5.4 输送带分层机

**用途**：将厚橡胶带分层（覆盖胶+骨架）

**应用场景**：
- 修补局部损坏（只换覆盖胶）
- 回收骨架材料

### 5.5 导条机

**用途**：在输送带上粘贴挡边导条

**应用**：
- 防止物料偏移
- 爬坡输送
- 开槽输送
