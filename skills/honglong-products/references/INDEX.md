### Q：客户问"风冷还是水冷？"

```
皮带厚度 ≤ 10mm? 
├── 是 → 风冷机
│   ├── 规格 300-1200 → 小型风冷
│   ├── 规格 1500-2100 → 标准风冷
│   └── 规格 2400-3600 → 大型风冷
│
└── 否 → 水冷机
    ├── 规格 600-1600 → 不锈钢机身
    ├── 规格 2100-3200 → 井字型机身
    └── 规格 3600-4200 → 超大型水冷机
```

## 📊 客户类型 → 推荐设备

| 客户类型 | 推荐设备 | 原因 |
|---------|---------|------|
| PVC/PU输送带加工厂 | 风冷机 | 皮带薄，风冷足够 |
| 矿业输送带维修 | 水冷机 | 皮带厚，需要更强加热 |
| 同步带制造厂 | 风冷机（小型） | 规格小，精度要求高 |
| 代理商/维修商 | 风冷+水冷组合 | 应对不同客户需求 |

## 🔑 核心概念速查

| 术语 | 解释 |
|------|------|
| **硫化** | 高温（140-150℃）+高压，使橡胶永久粘合 |
| **接头** | 两根输送带的连接处 |
| **接头强度** | 硫化后可达原带的 85-95% |
| **机械接头** | 夹板/螺栓连接，强度低，易脱落 |
| **风冷** | 风机冷却，适合薄皮带（≤10mm） |
| **水冷** | 水循环冷却，适合厚皮带（>10mm） |

**详细**：→ [§2 硫化原理](#-二为什么要硫化拼接)
