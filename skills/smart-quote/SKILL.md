# 红龙工业 - 智能报价技能 v2.0

## ⚡ 核心原则（业务员必读）

> **❌ 以前**：AI直接给价格 → 价格固化 → 不会变通
>
> **✅ 现在**：AI先背调查户口 → 给利润率范围 → 业务员灵活报价

---

## 报价流程（Step by Step）

### Step 1️⃣：获取客户公司名

**必须先问业务员**：
```
客户公司叫什么名字？（英文名）
目的国是哪里？
买什么产品？要多少量？
```

> ⚠️ 如果业务员只说"沙特客户"，一定要追问公司名！

### Step 2️⃣：检查是否已背调

**背调记录存储位置**：
1. **记忆文件**：检查 `MEMORY.md` 或当日 `memory/YYYY-MM-DD.md`
2. **CRM系统**：检查孚盟CRM中是否有该客户记录
3. **Pipeline客户池**：检查 `Pipeline客户池.xlsx`（如有）

**检查关键词**：
- 公司英文名
- 公司域名（bedrockco.sa → bedrockco.sa）
- 联系人邮箱

**判断逻辑**：
- **已有背调** → 读取ICP评分和关键信息，进入 Step 4
- **未背调** → 立即触发背调流程（调用 company-research）

### Step 3️⃣：客户背调（如需要）

使用 `company-research` 技能进行背调，重点获取：

| 维度 | 为什么要看 | 影响什么 |
|-----|-----------|---------|
| **企业规模** | 大公司付款好 | 可给较低利润率 |
| **行业匹配** | 制造业/木工/3D打印 | 直接影响产品选型 |
| **采购历史** | 是否有进口记录 | 判断付款能力 |
| **决策人** | 找到采购总监 | 影响沟通策略 |
| **ICP评分** | 综合评估 | 确定触达优先级 |

**背调完成后**，将结果保存到记忆文件（供下次使用）：
```markdown
## {公司名} 背调摘要
- 背调时间：{日期}
- 企业规模：{规模}（{员工数}人）
- 行业：{行业}
- ICP评分：{A/B/C/D}级（{分数}分）
- 付款能力：{描述}
- 报价有效期：{日期}
```

### 背调结果标准格式

从 `company-research` 获取后，提取以下字段映射到报价：

| 字段 | 背调来源 | 映射到 |
|-----|---------|--------|
| employee_count | LinkedIn/Crunchbase | 企业规模加成 |
| industry | 公司描述 | 行业匹配 |
| icp_score | 背调报告 | ICP加成 |
| payment_history | 海关记录/历史交易 | 付款方式加成 |
| customer_type | 公司业务描述 | 客户类型加成 |

### Step 4️⃣：根据背调结果确定利润率区间

**利润率决定因素**：

| 因素 | 好的（大公司/老客户） | 差的（小公司/新客户） |
|-----|---------------------|---------------------|
| **企业规模** | 1000+人 → 低利润率 | 50人以下 → 高利润率 |
| **ICP评分** | A级 → 可优惠 | C/D级 → 保利润 |
| **客户类型** | 经销商 → 中等 | 终端用户 → 可高 |
| **付款方式** | T/T预付 → 可优惠 | OA赊账 → 保利润 |
| **订单量** | 批量大 → 薄利多销 | 小批量 → 高利润 |

**利润率区间速查表**：

| 国家风险等级 | 国家示例 | 基础利润率 | 背调加成 |
|------------|---------|-----------|---------|
| 🟢 低风险 | 沙特、美国、加拿大 | 45-60% | +10%（A级客户）|
| 🟡 中风险 | 越南、墨西哥、巴西 | 35-50% | +5%（A级客户）|
| 🔴 高风险 | 印度、孟加拉、巴基斯坦 | 20-35% | 坚守底线 |

**给业务员的报价指引（不是固定价格表！）**：

## 背调加成规则

### 企业规模加成

| 规模 | 员工数 | 加成 |
|-----|-------|------|
| 微型 | 1-50 | +0% |
| 小型 | 50-200 | +5% |
| 中型 | 200-500 | +10% |
| 大型 | 500-1000 | +15% |
| 超大型 | 1000+ | +20% |

### ICP评分加成

| ICP等级 | 评分 | 加成 |
|--------|------|------|
| A级 | 75+ | +10% |
| B级 | 50-74 | +5% |
| C级 | 30-49 | +0% |
| D级 | <30 | -5%（谨慎）|

### 付款方式加成

| 付款方式 | 加成 |
|---------|------|
| T/T预付100% | +10% |
| T/T预付30% | +5% |
| L/C即期 | +0% |
| OA赊账 | -10% |
| 账期超60天 | -15% |

### 客户类型加成

| 客户类型 | 说明 | 加成 |
|---------|------|------|
| 终端用户 | 自己用 | +5% |
| 经销商 | 转卖 | -5% |
| 同行拿货 | 竞争对手 | -15% |

## 关键原则

### ⚠️ 红线（绝对不能破）

1. **孟加拉国利润率 ≥10%**
2. **印度利润率 ≥20%**
3. **马来西亚利润率 ≥20%**
4. **绝对不能低于成本价！**

### 💡 灵活空间

1. 利润率区间内，业务员可以灵活调整
2. 批量订单可以适当让利
3. 老客户可以给3-5%优惠
4. 非标产品可以加价10-30%

### 📝 记录要求

每次报价后，必须记录到当日 memory 文件：
- 报价日期
- 利润率
- 客户反馈
- 是否成单

**业务员问"怎么报价"时**：
```
"好的，我先帮您查一下这家公司的背景..."
"这家客户我们还没背调过，我先去了解一下..."
"根据背调结果，这家客户建议利润率在XX%-XX%之间"
```

**业务员问"价格能再低吗"**：
```
"根据利润率红线，这个价格已经是最低了"
"如果对方能提高付款比例，我们可以适当让利"
"批量到XX台，可以申请更优惠的利润率"
```

## 💡 实用技巧

1. **先问量再背调**：采购量<3台的新客户，可以先用简单背调，评分够75分再深度背调

2. **沙特客户试探**：首次报价可以比中价高5%，试探对方预算

3. **印度/孟加拉守底**：无论什么情况，印度≥20%、孟加拉≥10%、马来西亚≥20%

4. **同行客户识别**：如果对方是代理商，注意他可能倒卖。经销商可以给折扣但同行绝对不行

5. **批量阶梯报价**：
   - 1-2台：正常报价
   - 3-5台：利润率-2%
   - 6-10台：利润率-5%
   - 10台+：利润率-8%

6. **T/T预付加价**：T/T预付100%可以额外+5%利润率，因为零风险

7. **非标产品加价原则**：
   - 小改（颜色/标识）：+10%
   - 中改（尺寸调整）：+15%
   - 大改（首次开模）：+30% + ¥12,000开模费

8. **报价有效期**：建议设置7天有效期，汇率波动时重新核算

9. **含税要明确**：报价单必须注明"未税价"，开票加13%

10. **守住红线话术**：
    ```
    "这个价格已经是我们能给的最低价了"
    "再低就真的要亏本了"
    "我们可以考虑增加一些配件来平衡成本"
    ```
