## Step 3 ICP评分（6维度×100分）

| 维度 | 权重 | 评分依据 |
|------|------|----------|
| 行业匹配度 | 20% | 传送带相关采购占比 ≥50%=10分 |
| 采购能力 | 20% | 海关金额 S≥$1M/A≥$500K/B≥$100K |
| 采购频率 | 20% | 交易次数+最新日期 高频=10分 |
| 客户类型 | 15% | 制造商=10/经销商=8/终端=5 |
| 决策周期 | 15% | 企业规模推断 |
| 决策人联系 | 10% | LinkedIn/官网联系方式 |

**客户等级**：A级≥70分（重点开发，直接报价）| B级50-69分（次优先）| C级30-49分（低频跟进）| D级<30分（暂不开发）

**采购能力等级**：S级≥$1M总采购 | A级≥$500K | B级≥$100K | C/D/E级递减

## Step 1 客户发现 · 多语种搜索引擎 🌐

> **核心升级（v1.1.0）**：根据目标市场自动切换本地语言词簇，穿透英语搜索无法触达的隐形冠军
>
> **参考文档**：`references/MULTILANG-KEYWORDS.md`（完整词簇库）

### 多语种搜索三轮流程

```
第1轮（本地语言词簇）：
  从 MULTILANG-KEYWORDS.md 查找目标国家词簇
  构造查询：[本地产品词] + [本地渠道词] + [目标城市/地区]
  目的：找到只在本地语言网络存在的实力商

第2轮（英语补充搜索）：
  [English product term] + [country name] + [channel type]
  目的：找到有国际化意识的客户（更容易沟通）

第3轮（LinkedIn决策人，本地头衔）：
  "[本地决策人头衔]" site:linkedin.com + [公司名/城市]
  目的：直接定位采购决策人
```

### 语言选择速查

| 目标国家 | 搜索语言 | 典型黄金组合示例 |
|---------|---------|----------------|
| 德国/奥地利/瑞士 | 德语 | `Förderbandgurt Händler Bayern` |
| 西班牙/拉美各国 | 西班牙语 | `cinta transportadora distribuidor España` |
| 法国/北非法语区 | 法语 | `courroie transporteuse distributeur France` |
| 巴西/葡萄牙 | 葡语 | `correia transportadora distribuidor São Paulo` |
| 沙特/阿联酋/埃及 | 阿拉伯语 | `حزام ناقل موزع السعودية` |
| 印尼/越南/泰国 | 本地语 | `sabuk konveyor distributor Indonesia` |
| 土耳其 | 土耳其语 | `konveyör bant distribütör Türkiye` |
| 英语国家 | 英语（直接用）| - |

> ⚠️ **铁律**：非英语国家必须先跑本地语言第1轮，再跑英语第2轮，不可跳过！

## Step 4 LinkedIn决策人搜索 ⚠️

**第一步工具是 Exa，不是 LinkedIn MCP！**

```
1. Exa 搜索 → site:linkedin.com + 关键词
2. 提取信息 → 姓名、职位、公司、连接数
3. 优先级排序 → 高/中/低
4. 生成报告 → 决策人报告模板
5. 行动计划 → 连接消息 + 电话脚本
```

**搜索关键词组合（本地语言优先）**：

| 目标 | 英语关键词 | 本地语言示例（德语） |
|------|--------|----------------|
| 采购经理 | purchasing manager + conveyor | `Einkaufsleiter Förderband` |
| 运营经理 | operations manager + manufacturing | `Betriebsleiter Industrie` |
| 工程师 | maintenance engineer + plant | `Wartungsingenieur Anlage` |
| 老板 | owner/president/ceo + conveyor | `Geschäftsführer Riemen` |

> 💡 **提示**：在目标国家，用本地语言头衔搜索决策人，命中率比英语高 2-3 倍

## Step 6 开发信质量打磨（模式三）

**自动检查5个维度**：AI感 / 个性化 / 专业性 / 语法 / CTA

**打磨流程**：初稿 → 5维度检查 → 不合格自动修改（最多3轮）→ ✅ 评分≥7.0通过

**润色禁用15个陈词滥调**：
"In today's rapidly evolving landscape", "game-changer", "cutting-edge", "revolutionary", "state-of-the-art", "leverage", "synergy" 等

**评分规则**：
```
初始分数: 100
禁用陈词滥调: -10/个
包含数字: +5
包含问题: +5
包含列表: +5
长度适中: +5
```

## 回退方案

### 回退A：客户发现失败
```
1. 记录失败原因
2. 尝试其他关键词（变体、同义词）
3. 切换其他渠道
4. 所有渠道都失败 → 建议用户更换条件，保存失败记录
```

### 回退B：客户查重错误
```
1. 尝试本地缓存查重
2. 缓存不可用 → 跳过查重，标记"未查重"，人工确认时提醒
```

### 回退C：企业背调失败
```
1. 尝试其他背调渠道
2. 所有渠道都失败 → 标记"信息不足"，初步评分，建议人工补充
```

### 回退D：LinkedIn决策人渠道不可用
```
1. 自动切换到 Exa 搜索 LinkedIn 索引
2. Exa也失败 → Jina Reader 抓取 LinkedIn 公开页面
3. 都失败 → 标记"需人工查找"
```

### 回退E：邮件发送失败
```
1. 尝试其他邮箱
2. 无其他邮箱 → 标记"发送失败"，重试队列（1h→2h→4h），通知用户
```
