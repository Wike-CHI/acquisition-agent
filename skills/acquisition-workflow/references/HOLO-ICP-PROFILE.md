# HOLO目标客户画像

> 正确的客户定位，用于搜索和评估客户。

## 核心定位

**HOLO BELT = 输送带/工业皮带加工设备解决方案商**
**不是"卖设备"，而是站在Flexco/Almex价格与交付空档中的解决方案提供者**

## A类核心客户（必须主攻）

| 客户类型 | 典型特征 | 开发信号 |
|---------|---------|---------|
| **PVC/PU输送带加工厂** | 有技术认知、用过Flexco等欧美品牌、关注成本和灵活度 | 主动提Flexco/问效果/关心交期 |
| **同步带制造厂** | 专业性强、精度要求高、小批量多品种 | 有自己的加工车间 |
| **矿业公司（输送带维修）** | 有输送带需要硫化维修、自有维修团队 | 主动询硫化机/关心维修效率 |
| **代理商/经销商/维修服务商** | 有客户资源、看重产品线完整度、可OEM | 可贴牌/OEM |

> ⚠️ **排除**：不涉及输送带硫化维修的纯采矿作业企业

## 识别A类客户的4个信号

满足2条以上 → 高价值客户：

1. ✅ **客户主动提 Flexco / Almex**
2. ✅ **问"效果/应用"，不是"最低价"**
3. ✅ **关心交期、定制、适配**
4. ✅ **有自己的加工或维护团队**

## 典型客户痛点

| 痛点 | 描述 |
|------|------|
| 接头失败率高 | 手工或老式设备接头失败率高、返工多 |
| 换规格效率低 | 不同皮带规格需要多台设备 |
| 欧美设备价格高 | Flexco/欧洲/美国设备价格高、交期长 |
| 售后响应慢 | 售后服务响应慢，影响生产 |
| 定制不灵活 | 标准机型无法适配特殊规格 |

## 客户"真实心理"

**不是**：
> ❌ "我要买中国便宜设备"

**而是**：
> ✅ "有没有接近欧美品质，但**更灵活、交期更快、技术沟通更直接、总成本更可控**的方案"

## 关键战略洞察

> **HOLO最该争取的，不是"第一次买设备的客户"，而是"第二次重新选择设备的客户"**

## 开发信定位话术

> "获得更灵活且更具成本效益的拼接解决方案"

**不是**：
> - ❌ 中国制造商
> - ❌ 低价供应商
