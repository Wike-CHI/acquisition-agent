# 客户情报整合技能

整合多个调研技能，提供一站式客户情报服务。**支持动态ICP评分**。

## 核心功能
- 市场调研
- 客户背调
- 竞争对手分析
- **动态ICP评分**（新增）
- 客户报告生成
- 数据整合
- 洞察提炼

---

## ⭐ 动态ICP评分系统

### 初始评分（5维度）

| 维度 | 权重 | 评分标准 |
|------|------|----------|
| **采购量** | 25% | 大(10)/中(7)/小(4) |
| **产品匹配** | 25% | 高(10)/中(7)/低(4) |
| **区域** | 15% | 核心市场(10)/次要市场(7)/新兴市场(5) |
| **支付能力** | 20% | 强(10)/中(7)/弱(4) |
| **决策权** | 15% | 决策者(10)/影响者(7)/信息收集者(4) |

### 客户分层

| 分数 | 等级 | 标签 | 跟进策略 |
|------|------|------|----------|
| **≥8** | A级 | `hot_lead` | 24小时内首次联系，3天跟进 |
| **5-7** | B级 | `warm_lead` | 48小时内首次联系，5天跟进 |
| **3-4** | C级 | `cold_lead` | 存入培育池，2周联系 |
| **<3** | D级 | `disqualified` | 暂不开发 |

### 动态调整规则

ICP分数会根据互动自动调整：

| 客户行为 | 分数变化 | 原因 |
|----------|----------|------|
| 快速回复（<1小时） | **+1** | 高意向信号 |
| 主动询价 | **+2** | 购买意愿强 |
| 提及竞争对手 | **+2** | 正在比较，机会窗口 |
| 发送公司资料 | **+1** | 信任建立 |
| 询问MOQ/交期 | **+1** | 采购流程推进 |
| 从邮件转到WhatsApp | **+1** | 渠道升级 |
| 7天无回复 | **-1** | 意向降低 |
| 要求移出名单 | **-3** | 明确拒绝 |
| 标记为垃圾 | **-5** | 无效线索 |
| 邮件退信 | **-2** | 联系方式无效 |

**注意**: 每日调整上限 ±5（防止刷分）

## 使用场景
```
用户: 我需要了解德国工业皮带市场
→ 市场调研 (市场分析)
→ 客户背调 (特定公司)
→ 竞争对手分析
→ 生成综合报告

用户: 背调这5家公司
→ 客户背调 (批量)
→ 生成背调报告

用户: 分析竞争对手
→ 竞争对手分析
→ 生成竞争报告
```

## 技能组合

| 调用技能 | 用途 | 场景 |
|----------|------|------|
| market-research | 市场调研 | TAM/SAM/SOM分析 |
| company-research | 企业背调 | 深度企业信息 |
| in-depth-research | 深度研究 | 复杂分析任务 |
