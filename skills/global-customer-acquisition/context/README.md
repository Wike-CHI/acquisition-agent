# context/ — 7层上下文配置

> 每次对话开始时，按顺序加载这 5 个文件注入上下文。

## 文件清单

| 文件 | 用途 | 需手动修改 |
|------|------|-----------|
| `identity.md` | 公司身份 + 产品线 | ❌ 写死（HOLO公司信息）|
| `soul.md` | AI 人格和价值观 | ❌ 无需修改 |
| `agents.md` | 销售手册索引 | ❌ 无需修改 |
| `user.md` | 业务员个人信息 | ⚠️ **首次使用必须填写** |
| `memory.md` | 4层抗遗忘配置 + HEARTBEAT任务索引 | ❌ 无需修改 |

---

## 首次使用：填写 user.md

打开 `context/user.md`，填写以下 3 项：

```
姓名     → 你的名字（用于开发信落款）
邮箱     → 你的销售邮箱（用于发送开发信）
电话     → 你的联系电话（用于 WhatsApp/邮件签名）
职位     → 你的职位（用于开发信签名）
```

AI 每次对话开始时读取此文件。
**如果检测到"请填写"字样，AI 会主动提醒你补充信息。**

---

## AI 加载顺序

1. `identity.md` → 注入公司身份
2. `soul.md` → 注入 AI 人格
3. `agents.md` → 注入销售手册索引
4. `user.md` → 注入业务员信息（姓名/邮箱/电话）
5. `memory.md` → 注入4层抗遗忘配置

> **不在此目录的文件**：HEARTBEAT.md、TOOLS.md、ANTI-AMNESIA.md 位于技能根目录。
