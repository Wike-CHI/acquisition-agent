## 六、视频处理命令

### 6.1 提取视频帧

```bash
# 提取第10秒的画面
ffmpeg -i video.mp4 -ss 00:00:10 -frames:v 1 frame.jpg

# 每隔5秒提取一帧
ffmpeg -i video.mp4 -vf fps=1/5 frames/%04d.jpg
```

### 6.2 提取音频

```bash
# 提取音频为MP3
ffmpeg -i video.mp4 -vn -acodec mp3 audio.mp3

# 使用Whisper转录
whisper audio.mp3 --language zh --model medium
```

### 6.3 获取视频信息

```bash
# 查看视频元数据
ffprobe -v quiet -print_format json -show_format -show_streams video.mp4
```
