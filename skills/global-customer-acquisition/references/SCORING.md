# ICP评分体系（6维度×100分）

> 客户质量评分系统，用于评估潜在客户价值和优先级。
>
> 评分标准：A级≥75分（重点开发）| B级50-74分 | C级30-49分 | D级<30分（暂不开发）
>
> **权威标准**：本文件提供详细评分细则，阈值定义以 `references/ICP-STANDARDS.md` 为准。

---

## 客户等级标准

| 等级 | 分数范围 | 策略 | 预期转化率 |
|------|----------|------|-----------|

## 评分细则

### 1. 行业匹配度（20分）

**评分依据**：传送带相关采购占比

| 采购占比 | 分数 | 说明 |
|---------|------|------|
| 80-100% | 18-20分 | 核心业务完全匹配 |
| 60-79% | 15-17分 | 主要业务匹配 |
| 40-59% | 10-14分 | 部分业务匹配 |
| 20-39% | 5-9分 | 少量业务匹配 |
| <20% | 0-4分 | 业务不匹配 |

**数据来源**：海关数据、公司官网、产品目录

---

### 2. 采购能力（20分）

**评分依据**：海关金额（S/A/B/C/D/E级）

| 海关等级 | 年采购额 | 分数 | 说明 |
|---------|---------|------|------|
| **S级** | >$1M | 18-20分 | 大客户，采购能力强 |
| **A级** | $500K-$1M | 15-17分 | 中大型客户 |
| **B级** | $100K-$500K | 10-14分 | 中型客户 |
| **C级** | $50K-$100K | 5-9分 | 小型客户 |
| **D级** | $10K-$50K | 2-4分 | 微型客户 |
| **E级** | <$10K | 0-1分 | 无采购记录 |

**数据来源**：特易海关数据

---

### 3. 采购频率（20分）

**评分依据**：交易次数+最新日期

| 交易次数 | 最新日期 | 分数 | 说明 |
|---------|---------|------|------|
| ≥10次 | 6个月内 | 18-20分 | 高频活跃客户 |
| 5-9次 | 12个月内 | 15-17分 | 中频活跃客户 |
| 2-4次 | 24个月内 | 10-14分 | 低频活跃客户 |
| 1次 | 36个月内 | 5-9分 | 偶尔采购客户 |
| 0次 | - | 0-2分 | 无采购记录（新客户）|

**数据来源**：特易海关数据

---

### 4. 客户类型（15分）

**评分依据**：制造商/经销商/终端

| 客户类型 | 分数 | 说明 |
|---------|------|------|
| **制造商** | 10分 | 采购原材料，需求稳定 |
| **经销商/分销商** | 8分 | 采购成品转售，量大但利润低 |
| **终端用户** | 5分 | 采购自用，需求不稳定 |
| **加工厂** | 7分 | 采购原带加工，介于制造商和经销商之间 |
| **工程公司** | 6分 | 项目制采购，需求不连续 |

**数据来源**：公司官网、LinkedIn、工商信息

---

### 5. 决策周期（15分）

**评分依据**：企业规模推断

| 企业规模 | 员工数 | 决策周期 | 分数 | 说明 |
|---------|--------|---------|------|------|
| **大型企业** | >500人 | 3-6个月 | 12-15分 | 决策慢，但订单大 |
| **中型企业** | 100-500人 | 1-3个月 | 8-11分 | 决策中速，订单中等 |
| **小型企业** | 10-100人 | 2-4周 | 4-7分 | 决策快，订单小 |
| **微型企业** | <10人 | 1-2周 | 1-3分 | 决策很快，订单很小 |

**数据来源**：LinkedIn、工商信息

---

### 6. 决策人联系（10分）

**评分依据**：LinkedIn/官网联系方式

| 联系方式质量 | 分数 | 说明 |
|------------|------|------|
| **LinkedIn决策人 + 官网邮箱** | 9-10分 | 最优，可直接联系决策人 |
| **官网邮箱** | 7-8分 | 良好，可到达公司 |
| **LinkedIn决策人** | 5-6分 | 一般，需通过LinkedIn联系 |
| **通用邮箱（info@）** | 2-4分 | 较差，可能不是决策人 |
| **无联系方式** | 0分 | 无法联系 |

**数据来源**：LinkedIn、官网、特易易搜邮

## 评分计算示例

### 示例1：A级客户（高分）

```
客户：ABC Industrial Belts（美国输送带分销商）

1. 行业匹配度：18分（100%采购输送带）
2. 采购能力：17分（A级，年采购$800K）
3. 采购频率：16分（8次，最近6个月）
4. 客户类型：8分（经销商）
5. 决策周期：10分（中型企业，200人）
6. 决策人联系：9分（LinkedIn决策人 + 官网邮箱）

总分：18+17+16+8+10+9 = 78分 → A级（重点开发）
```

### 示例2：B级客户（中等分）

```
客户：XYZ Engineering（马来西亚工程公司）

1. 行业匹配度：12分（60%采购输送带）
2. 采购能力：12分（B级，年采购$300K）
3. 采购频率：10分（3次，最近18个月）
4. 客户类型：6分（工程公司）
5. 决策周期：6分（小型企业，50人）
6. 决策人联系：5分（LinkedIn决策人）

总分：12+12+10+6+6+5 = 51分 → B级（正常开发）
```

### 示例3：D级客户（低分）

```
客户：DEF Mining（澳大利亚矿业公司）

1. 行业匹配度：2分（10%采购输送带，主要是矿业设备）
2. 采购能力：3分（C级，年采购$80K）
3. 采购频率：5分（1次，最近30个月）
4. 客户类型：5分（终端用户）
5. 决策周期：3分（小型企业，30人）
6. 决策人联系：2分（只有info@邮箱）

总分：2+3+5+5+3+2 = 20分 → D级（暂不开发）
```

## 评分使用指南

### 评分时机

- **Step 3: 企业背调**时进行评分
- 使用 `acquisition-evaluator` 技能进行评分验证

### 评分后的行动

| 等级 | 行动 |
|------|------|
| **A级（≥75分）** | 立即开发，优先跟进，分配充足资源 |
| **B级（50-74分）** | 正常开发，中等跟进，观察反馈 |
| **C级（30-49分）** | 观察开发，低优先级，有时间再做 |
| **D级（<30分）** | 暂不开发，放弃或进入长期观察池 |

### 人工调整规则

| 情况 | 调整 | 说明 |
|------|------|------|
| 客户有特殊需求 | +5 分 | 定制化机会 |
| 客户是竞争对手 | -20 分 | 直接降到 D 级 |
| 客户有紧急项目 | +10 分 | 快速成交窗口 |
| 已有供应商但考虑更换 | +5 分 | 替代机会 |

## 常见问题

### Q1：新客户（无采购记录）怎么评分？

新客户在"采购频率"维度得 0-2 分，但可以在其他维度补回：
- 行业匹配度高：18 分
- 客户类型好：10 分（制造商）
- 决策人联系好：9 分
- 总分可能达到 37 分（C 级），值得观察开发

### Q2：为什么竞争对手要减 20 分？

竞争对手不会向你采购，开发他们是浪费时间。减 20 分确保降到 D 级。
