## 📋 平台适配规则

### Instagram
- **字数**：125-150 词（简洁）
- **格式**：Emoji + 话题标签
- **话题标签**：5-10 个
- **视觉**：产品图 + 客户案例图

### TikTok
- **时长**：15-30 秒（短视频）
- **格式**：脚本式（分镜头）
- **话题标签**：3-5 个
- **视觉**：设备操作演示

### LinkedIn
- **字数**：200-300 词（专业）
- **格式**：品牌故事 + 技术规格
- **话题标签**：3-5 个
- **视觉**：产品图 + 工厂图

### Facebook
- **字数**：150-200 词（亲切）
- **格式**：产品介绍 + 客户见证
- **话题标签**：3-5 个
- **视觉**：产品图 + 客户案例图

### X (Twitter)
- **字数**：100-120 词（极简）
- **格式**：核心卖点 + 联系方式
- **话题标签**：2-3 个
- **视觉**：产品图

### YouTube
- **字数**：300-500 词（详细）
- **格式**：视频描述 + 时间轴
- **话题标签**：5-10 个
- **视觉**：产品演示视频

### Pinterest
- **字数**：100-150 词（视觉导向）
- **格式**：标题 + 描述 + 图片建议
- **话题标签**：3-5 个
- **视觉**：产品图 + 技术参数图

### Threads
- **字数**：50-100 词（快速互动）
- **格式**：简洁有力
- **话题标签**：2-3 个
- **视觉**：产品图

### Bluesky
- **字数**：100-150 词（新兴平台）
- **格式**：简洁专业
- **话题标签**：2-3 个
- **视觉**：产品图
