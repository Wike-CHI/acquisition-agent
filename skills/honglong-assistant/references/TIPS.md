# 红龙小助手执行技巧

> 本文档包含红龙助手执行获客任务时的实用技巧和避坑指南。

---

## 🔥 执行技巧

### 1. 搜索效率优化

```
✅ 搜索时指定行业关键词
   "conveyor belt distributor Africa" → 比 "belt" 效果好10倍

✅ 决策人搜索优先
   LinkedIn 职位关键词：Procurement Manager / Purchasing Director / Buyer
   
✅ 多源交叉验证
   搜索结果必须经过：LinkedIn + 官网 + 海关数据 三重验证
```

### 2. 邮件开发信技巧

```
✅ 邮件标题优化
   - 使用数字："How ABC Reduced Belt Cost by 15%"
   - 避免spam词：FREE, URGENT, LIMITED TIME
   
✅ 正文结构
   1. 个性化开场（客户最近的动态）
   2. 价值主张（量化好处）
   3. 社会证明（客户案例）
   4. 明确CTA（下一步动作）

✅ 评分必须≥9.0分才发送
   低于9.0分继续润色，直到达标
```

### 3. WhatsApp 触达技巧

```
✅ 消息必须个性化
   ❌ "Hi, we sell conveyor belts"
   ✅ "Hi John, saw ABC expanded to Kenya last month..."
   
✅ 时区考虑
   发送时间：对方当地时间 9:00-18:00
   避免时段：22:00-08:00（休息时间）
   
✅ 频率控制
   同一联系人每天最多1条
   每周最多3条
```

### 4. 客户评分技巧

```
✅ ICP评分≥75分才触达
   - 75-100分：A级客户，重点开发
   - 60-74分：B级客户，观察储备
   - 45-59分：C级客户，待培育
   - <45分：暂不开发

✅ 评分维度权重
   行业匹配度 20% + 采购能力 20% + 采购频率 20%
   + 客户类型 15% + 决策周期 15% + 决策人联系 10%
```

### 5. 联系方式验证

```
✅ 验证优先级（从高到低）
   S级：官网"Contact Us"表单
   A级：特易详情页联系方式（6个月内）
   B级：LinkedIn决策人（≤12个月）
   C级：公司通用邮箱（需额外核实）
   
✅ 禁止使用
   - 超过6个月的LinkedIn信息
   - 发布日期未知的公网邮箱
   - 仅凭单一来源
```

## ⚠️ 常见踩坑点

### 1. 搜索踩坑

```
❌ 使用通用关键词搜索
   "belt" → 找到10万+结果，无效

✅ 使用精准行业关键词
   "industrial belt distributor" + "conveyor belt supplier"
```

### 2. 决策人踩坑

```
❌ 发送至 info@ / sales@ 通用邮箱
   回复率 < 1%

✅ 发送至决策人个人邮箱
   Procurement Manager / Purchasing Director
   回复率 > 15%
```

### 3. 邮件踩坑

```
❌ 使用 AI 生成感明显的邮件
   "I hope this email finds you well..."

✅ 使用 humanize-ai-text 润色
   个性化开场 + 真实痛点 + 具体数据
```

### 4. 触达踩坑

```
❌ 同时发送邮件+WhatsApp
   客户收到重复信息，体验差

✅ 按节奏触达
   D0: 邮件 → D3: LinkedIn → D5: 跟进邮件 → D14: 最终跟进
```

### 5. 报价踩坑

```
❌ 直接报最低价
   利润受损

✅ 使用 smart-quote 计算最优报价
   考虑：国家 + 客户类型 + 采购量 + 竞争情况
```

## 💡 进阶技巧

### 1. 批量获客效率

```
✅ 并行执行多个搜索任务
   同时搜索：LinkedIn + Facebook + 海关数据

✅ 去重合并
   跨平台去重，避免重复开发

✅ 批量评分
   Excel 批量 ICP 评分 → 排序 → TOP客户优先
```

### 2. Drip Campaign 节奏

```
D0: 首次联系 → 开发信 + 产品介绍
D3: LinkedIn连接 + WhatsApp跟进
D5: 跟进邮件 → 价值主张 + 案例
D14: 最终跟进 → Free Sample / Demo
```

### 3. 竞品分析

```
✅ 了解客户现有供应商
   "Which brand are you using now?"
   
✅ 差异化话术
   我们的优势 vs 竞品劣势
```
