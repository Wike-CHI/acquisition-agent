# 孚盟MX CRM 技能 - 快速使用指南

## 📋 功能概览

| 功能 | 命令 | 说明 |
|------|------|------|
| **登录** | `登录孚盟MX` | 使用保存的凭据自动登录 |
| **查询客户** | `查询客户 [公司名]` | 按公司名搜索客户 |
| **创建客户** | `创建客户 [信息]` | 创建新客户记录 |
| **更新客户** | `更新客户 [公司名] [字段] [值]` | 更新客户信息 |
| **客户列表** | `客户列表 [状态]` | 按状态筛选客户 |
| **添加到CRM** | `添加到CRM` | 将搜索结果添加到CRM |

## 🔧 获客流程集成

孚盟MX CRM 已集成到完整的获客流程：

```
Phase 1: 搜索
  → LinkedIn/Facebook/Instagram/海关数据
  → 找到潜在客户

Phase 1.5: 去重
  → 与孚盟CRM数据对比
  → 排除已有客户

Phase 2: 背调
  → 企业背景调查
  → 评估采购能力

Phase 3: 筛选
  → ICP评分
  → 筛选高价值客户

Phase 4: 触达
  → 发送开发信
  → 添加到孚盟CRM ← 新增
  → 记录跟进状态
```

## ⚠️ 注意事项

1. **凭据安全** - 密码使用Windows DPAPI加密存储
2. **数据备份** - CRM操作会自动记录日志
3. **批量操作** - 超过10条记录需要管理员审批
4. **查重机制** - 自动检测重复客户，避免重复录入

## 🔧 故障排查

### 问题1：登录失败

```
解决方案:
1. 检查账号密码是否正确
2. 确认网络可以访问 fumamx.com
3. 重新配置: 配置孚盟账号
```

### 问题2：客户已存在

```
解决方案:
系统会自动提示重复客户
可以选择：
- 跳过该客户
- 更新现有客户信息
```
