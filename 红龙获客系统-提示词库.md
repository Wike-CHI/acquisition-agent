# 红龙获客系统 — 提示词库

> 复制粘贴即可使用。按场景分类，从基础到高级。

---

## 一、客户搜索

### 基础搜索

```
帮我找[国家]的[行业]客户，要[数量]家
```

```
搜索[国家][城市]地区的输送带相关企业
```

### 精准搜索

```
帮我找沙特阿拉伯吉达地区的输送带经销商，要求：
1. 年采购额50万美元以上
2. 有自己的维修团队
3. 经营5年以上
要10家，按规模排序
```

```
搜索印度孟买的输送带加工设备分销商，
需要经营以下产品的：风冷拼接机、水冷拼接机、导条机、打齿机
```

### 按展会搜索

```
搜索参加过迪拜BIG5展会的输送带设备买家
```

```
搜索沙特利雅得机电展的参展商名单
```

### 多国家搜索

```
帮我分别搜索以下国家的输送带经销商，每个国家5家：
沙特、阿联酋、印度、巴西、墨西哥
```

## 二、公司调研

### 背景调查

```
帮我查一下[公司名]的背景，包括：
- 成立时间、注册资本
- 员工规模、年营业额
- 主要业务、产品线
- 采购渠道、主要供应商
- 联系方式（官网、邮箱、电话）
```

### 竞品分析

```
分析[公司名]的竞争对手有哪些，
他们的产品价格区间和采购模式是什么
```

### 采购能力评估

```
评估[公司名]作为潜在客户的采购能力：
- 年输送带设备采购预算
- 当前使用的设备品牌
- 采购决策流程
- 推荐的切入产品
```

### ICP 评分

```
给以下客户做ICP评分：
[公司A]
[公司B]
[公司C]
评分维度：企业规模、采购需求、决策人可触达性、竞争态势
```

## 三、开发信

### 首次触达

```
给[公司名]的[联系人名]写一封英文开发信，
介绍我们的风冷拼接机和水冷拼接机，
强调：中国制造、出厂价、3年质保、快速交付
语气专业友好，150词以内
```

### 跟进邮件

```
[联系人名]上周发了邮件没有回复，
写一封简短的跟进邮件，提供产品目录PDF链接
语气轻松，不要催促
```

### 报价后跟进

```
[联系人名]上个月询了风冷拼接机的价格，
写一封跟进邮件，询问是否有进一步问题，
可以提供免费样品
```

### 多语言

```
用阿拉伯语给[公司名]写一封WhatsApp消息，
介绍我们的输送带拼接设备，
要求：简短、友好、包含产品图片链接
```

### 评分优化

```
这封开发信的评分是多少？
如果低于9分，请帮我优化：
1. 开头要更有吸引力
2. 产品卖点要更具体
3. CTA要更明确
```

## 四、市场分析

### 行业分析

```
帮我做[国家][行业]的市场分析报告：
1. 市场规模和增长率
2. 主要竞争对手
3. 进口政策
4. 采购习惯
5. 推荐进入策略
```

### 展会分析

```
分析[展会名]的参展价值：
1. 展会规模和历史
2. 参展商类型
3. 专业观众画像
4. 我们应该参展吗
```

### 产品定位

```
我们的风冷拼接机在[国家]市场应该如何定位？
分析价格区间、目标客户群、销售渠道建议
```

## 五、WhatsApp 触达

### 首次消息

```
给[公司名]的[联系人名]发一条WhatsApp消息，
介绍红龙工业的输送带拼接设备，
消息要短（50词以内），语气友好
```

### 产品目录

```
给[联系人名]发一条WhatsApp，附带产品目录PDF
说明：这是我们最新的产品目录，包含全部型号和参数
```

### 批量发送

```
把搜索结果中有WhatsApp号码的客户批量发送消息，
消息内容根据行业自动匹配模板
```

## 六、数据管理

### 导出 Excel

```
把这批客户导出为Excel，包含以下列：
公司名、国家、行业、联系人、邮箱、WhatsApp、ICP评分
文件名：[国家]客户_[日期].xlsx
```

### 去重

```
把这批新搜索的客户和已有Pipeline去重，
按公司名和域名匹配，只保留新客户
```

### Pipeline 更新

```
更新Pipeline状态：
- 已发邮件的客户标记为"已触达"
- 有回复的客户标记为"已回复"
- 报价过的客户标记为"报价中"
```

## 七、组合工作流

### 完整获客（推荐）

```
帮我执行一次完整的沙特市场获客流程：
1. 搜索沙特输送带经销商（10家）
2. 对每家做背景调查和ICP评分
3. 给75分以上的生成开发信
4. 发送开发信
5. 给有WhatsApp的发送消息
6. 输出结果汇总
```

### 竞品监控

```
帮我监控[竞争对手名]的最新动态：
1. 搜索他们最近3个月的新闻
2. 检查他们的LinkedIn和Facebook
3. 看有没有参加新展会
4. 分析他们的新品发布
```

### 客户培育

```
这批客户已经发过开发信一周了，
帮我制定下一步跟进计划：
- 未回复的：怎么跟进
- 已回复的：下一步怎么推
- 高意向的：怎么促成交
```
