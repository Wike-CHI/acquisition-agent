# 红龙获客系统 — 业务员使用手册

> 版本 2.5 | 2026-04-07
> 温州红龙工业设备制造有限公司

---

## 一、这是什么？

红龙获客系统是一个 AI 助手，帮你做这些事：

| 能力 | 举例 |
|------|------|
| 找客户 | "帮我找沙特的输送带经销商" |
| 查公司 | "查一下这家公司 BEDROCK 的背景" |
| 写开发信 | "给这家公司写一封英文开发信" |
| 发邮件 | 自动通过 SMTP 发送 |
| 发 WhatsApp | 自动通过 wacli 发送 |
| 市场分析 | "分析沙特输送带市场" |

---

## 二、第一次使用

### 2.1 环境准备

确保以下内容已配置（找 Wike 协助）：

- [ ] WorkBuddy 已安装并启动
- [ ] SMTP 邮箱已配置（发送邮件用）
- [ ] WhatsApp wacli 已登录（发送 WhatsApp 用）
- [ ] NAS 已挂载（产品资料存储）

### 2.2 快速验证

在 WorkBuddy 对话框输入：

```
你好，帮我初始化获客系统
```

系统会自动检查各项配置状态。

## 三、常用场景

### 场景 1：找新客户

**目标市场 → 客户搜索 → 筛选**

```
帮我找沙特阿拉伯的输送带设备经销商，要10家
```

**更精准的搜索：**

```
搜索沙特水泥行业的输送带维修服务商，需要有以下设备的企业：
风冷拼接机、水冷拼接机、导条机
```

### 场景 2：调研一家公司

```
帮我查一下 Protech Engineering 这家公司的背景，
包括：成立时间、员工规模、主要业务、采购渠道
```

### 场景 3：生成并发送开发信

**第一步：生成开发信**

```
给 Protech Engineering 写一封开发信，
介绍我们的风冷拼接机和水冷拼接机，
语气要专业但友好，不要超过150词
```

**第二步：检查质量**

```
这封开发信的评分是多少？如果低于9分请帮我优化
```

**第三步：发送**

```
把这封开发信发送给 sergio@protech.com.sa
```

### 场景 4：批量 WhatsApp 触达

```
把搜索结果中有 WhatsApp 号码的客户，
生成阿拉伯语消息并发送
```

### 场景 5：市场分析报告

```
帮我做一份沙特阿拉伯工业皮带设备市场分析，
包括：市场规模、主要玩家、进入策略、展会信息
```

## 四、完整获客流程（推荐）

从搜索到触达的标准流程：

```
第1步：搜索客户
→ "帮我找[国家]的[行业]客户，要[数量]家"

第2步：背调筛选
→ "帮我查这10家公司的背景"
→ "给它们做ICP评分"

第3步：生成开发信
→ "给ICP评分75分以上的客户写开发信"

第4步：发送触达
→ "发送开发信"
→ "给有WhatsApp号码的发送消息"

第5步：导入孚盟MX
→ "把这批客户导出为Excel，我要导入孚盟"
```

## 五、质量铁律

发送任何内容前，系统会自动检查：

| 规则 | 说明 |
|------|------|
| ICP 评分 ≥ 75 | 低于75分不发邮件 |
| 开发信评分 ≥ 9.0 | 低于9分不发 |
| 禁用 info@ 邮箱 | 必须用决策人邮箱 |
| WhatsApp 每人每天 ≤ 3条 | 频率控制 |
| 对方当地时间 22:00-08:00 不发送 | 避免打扰 |
| 未验证联系方式不发 | 防止发错 |

## 六、提示词技巧

### 越具体越好

```
❌ "找客户"
✅ "找印度孟买地区的输送带经销商，年采购额50万美元以上"
```

### 指定输出格式

```
❌ "查这家公司"
✅ "查这家公司，输出格式：公司名/国家/成立时间/员工数/主营业务/采购渠道"
```

### 分步而不是一步到位

```
❌ "帮我找客户并发邮件"
✅ 先搜索 → 再筛选 → 最后发邮件（每步可以检查和调整）
```

### 利用评分机制

```
"给这封开发信打分"
"这批客户按ICP评分排序"
"只保留评分80分以上的"
```

## 七、孚盟 MX 导入（手动）

目前孚盟 MX 需要手动导入，操作步骤：

1. 在获客系统中导出客户为 Excel
2. 登录孚盟 MX → 客户管理 → 导入
3. 选择"批量新增客户" → 上传 Excel
4. 系统自动查重，确认导入

> 自动导入功能开发中，后续更新。

## 八、常见问题

### Q: 邮件发送失败？
检查 SMTP 配置，运行"测试邮箱连接"。

### Q: WhatsApp 发送不了？
通常是 wacli 锁问题，说"重试发送"即可自动修复。

### Q: 搜索结果不够精准？
加更多条件限制：国家、行业、产品类型、采购规模。

### Q: 开发信质量不高？
明确指定：客户行业、产品匹配点、语气风格、目标语言。
