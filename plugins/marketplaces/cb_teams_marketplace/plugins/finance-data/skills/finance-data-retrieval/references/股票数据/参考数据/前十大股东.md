# 前十大股东

> 接口：`top10_holders` | 获取上市公司前十大股东数据，包括持股数量和持股比例

## 请求参数

| 参数 | 类型 | 必选 | 说明 |
|------|------|------|------|
| ts_code | str | Y | TS股票代码 |
| period | str | N | 报告期（YYYYMMDD，一般为每季度最后一天） |
| ann_date | str | N | 公告日期 |
| start_date | str | N | 报告期开始日期 |
| end_date | str | N | 报告期结束日期 |

## 返回字段

| 字段 | 类型 | 说明 |
|------|------|------|
| ts_code | str | TS代码 |
| ann_date | str | 公告日期 |
| end_date | str | 报告期 |
| holder_name | str | 股东名称 |
| hold_amount | float | 持股数量（股） |
| hold_ratio | float | 占总股本比例（%） |
| hold_float_ratio | float | 占流通股比例（%） |
| hold_change | float | 持股变动 |
| holder_type | str | 股东类型 |

## 调用示例

```bash
curl -X POST https://www.codebuddy.cn/v2/tool/financedata \
  -H "Content-Type: application/json" \
  -d '{
    "api_name": "top10_holders",
    "params": {"ts_code": "000001.SZ", "period": "20240930"},
    "fields": ""
  }'
```
