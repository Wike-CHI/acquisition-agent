# 温州民间借贷利率

> 接口：`wz_index` | 温州民间融资综合利率指数（温州指数）数据

## 请求参数

| 参数 | 类型 | 必选 | 说明 |
|------|------|------|------|
| date | str | N | 日期 |
| start_date | str | N | 开始日期 |
| end_date | str | N | 结束日期 |

**限量：** 不限量，一次可取全部历史数据。**权限：** 用户需至少2000积分。

## 返回字段

| 字段 | 类型 | 说明 |
|------|------|------|
| date | str | 日期 |
| comp_rate | float | 综合利率指数（%） |
| center_rate | float | 民间借贷服务中心利率 |
| micro_rate | float | 小额贷款公司放款利率 |
| cm_rate | float | 民间资本管理公司融资价格 |
| sdb_rate | float | 社会直接借贷利率 |
| om_rate | float | 其他市场主体利率 |
| aa_rate | float | 农村互助会互助金费率 |
| m1_rate | float | 一月期利率 |
| m3_rate | float | 三月期利率 |
| m6_rate | float | 六月期利率 |
| m12_rate | float | 一年期利率 |
| long_rate | float | 长期利率 |

## 调用示例

```bash
curl -X POST https://www.codebuddy.cn/v2/tool/financedata \
  -H "Content-Type: application/json" \
  -d '{
    "api_name": "wz_index",
    "params": {"start_date": "20180101", "end_date": "20180131"},
    "fields": ""
  }'
```
