# 历史分钟行情

> 接口：`stk_mins` | 获取A股分钟级别历史行情数据，支持1/5/15/30/60分钟频度

## 请求参数

| 参数 | 类型 | 必选 | 说明 |
|------|------|------|------|
| ts_code | str | Y | 股票代码，如 000001.SZ |
| freq | str | Y | 分钟频度：1min/5min/15min/30min/60min |
| start_date | str | N | 开始时间，格式：2025-01-02 09:30:00 |
| end_date | str | N | 结束时间，格式：2025-01-02 15:00:00 |

## 返回字段

| 字段 | 类型 | 说明 |
|------|------|------|
| ts_code | str | 股票代码 |
| trade_time | str | 交易时间 |
| close | float | 收盘价 |
| open | float | 开盘价 |
| high | float | 最高价 |
| low | float | 最低价 |
| vol | float | 成交量（股） |
| amount | float | 成交金额（元） |

## 调用示例

```bash
curl -X POST https://www.codebuddy.cn/v2/tool/financedata \
  -H "Content-Type: application/json" \
  -d '{
    "api_name": "stk_mins",
    "params": {"ts_code": "000001.SZ", "freq": "1min", "start_date": "2025-01-02 09:30:00", "end_date": "2025-01-02 15:00:00"},
    "fields": ""
  }'
```
