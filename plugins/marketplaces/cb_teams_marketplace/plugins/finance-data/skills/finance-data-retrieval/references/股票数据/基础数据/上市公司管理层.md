# 上市公司管理层

> 接口：`stk_managers` | 获取上市公司管理层信息

## 请求参数

| 参数 | 类型 | 必选 | 说明 |
|------|------|------|------|
| ts_code | str | N | 股票代码，支持单个或多个 |
| ann_date | str | N | 公告日期（YYYYMMDD格式） |
| start_date | str | N | 公告开始日期 |
| end_date | str | N | 公告结束日期 |

## 返回字段

| 字段 | 类型 | 说明 |
|------|------|------|
| ts_code | str | TS股票代码 |
| ann_date | str | 公告日期 |
| name | str | 姓名 |
| gender | str | 性别 |
| lev | str | 岗位类别 |
| title | str | 岗位 |
| edu | str | 学历 |
| national | str | 国籍 |
| birthday | str | 出生年月 |
| begin_date | str | 上任日期 |
| end_date | str | 离任日期 |

## 调用示例

```bash
curl -X POST https://www.codebuddy.cn/v2/tool/financedata \
  -H "Content-Type: application/json" \
  -d '{
    "api_name": "stk_managers",
    "params": {"ts_code": "000001.SZ"},
    "fields": ""
  }'
```
