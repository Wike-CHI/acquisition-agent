# 新闻联播文字稿

> 接口：`cctv_news` | 获取新闻联播文字稿数据，数据从2017年起

## 请求参数

| 参数 | 类型 | 必选 | 说明 |
|------|------|------|------|
| date | str | Y | 日期（格式：YYYYMMDD） |

**权限：** 需单独开通权限。内容按段落分段处理，便于筛选过滤。

## 返回字段

| 字段 | 类型 | 说明 |
|------|------|------|
| date | str | 日期 |
| title | str | 标题 |
| content | str | 内容 |

## 调用示例

```bash
curl -X POST https://www.codebuddy.cn/v2/tool/financedata \
  -H "Content-Type: application/json" \
  -d '{
    "api_name": "cctv_news",
    "params": {"date": "20250227"},
    "fields": ""
  }'
```
