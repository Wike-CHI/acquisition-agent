# 每日筹码及胜率

> 接口：`cyq_perf` | 获取A股每日平均成本和胜率数据。数据自2018年起，每日18:00-19:00更新

## 请求参数

| 参数 | 类型 | 必选 | 说明 |
|------|------|------|------|
| ts_code | str | Y | 股票代码 |
| trade_date | str | N | 交易日期（YYYYMMDD格式） |
| start_date | str | N | 开始日期 |
| end_date | str | N | 结束日期 |

## 返回字段

| 字段 | 类型 | 说明 |
|------|------|------|
| ts_code | str | 股票代码 |
| trade_date | str | 交易日期 |
| his_low | float | 历史最低价 |
| his_high | float | 历史最高价 |
| cost_5pct | float | 5%成本分位 |
| cost_15pct | float | 15%成本分位 |
| cost_50pct | float | 50%成本分位（中位数） |
| cost_85pct | float | 85%成本分位 |
| cost_95pct | float | 95%成本分位 |
| weight_avg | float | 加权平均成本 |
| winner_rate | float | 胜率 |

## 调用示例

```bash
curl -X POST https://www.codebuddy.cn/v2/tool/financedata \
  -H "Content-Type: application/json" \
  -d '{
    "api_name": "cyq_perf",
    "params": {"ts_code": "000001.SZ", "trade_date": "20250228"},
    "fields": "ts_code,trade_date,his_low,his_high,cost_5pct,cost_50pct,cost_95pct,weight_avg,winner_rate"
  }'
```

## 返回示例

```json
{
  "code": 0,
  "msg": "",
  "data": {
    "fields": ["ts_code", "trade_date", "his_low", "his_high", "cost_5pct", "cost_15pct", "cost_50pct", "cost_85pct", "cost_95pct", "weight_avg", "winner_rate"],
    "items": [
      ["000001.SZ", "20250228", 0.2, 21.2, 9.0, 9.6, 11.0, 11.2, 12.4, 10.89, 49.03]
    ]
  }
}
```
