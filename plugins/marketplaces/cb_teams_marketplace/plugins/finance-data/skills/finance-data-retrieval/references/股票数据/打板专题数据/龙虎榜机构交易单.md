# 龙虎榜机构明细

> 接口：`top_inst` | 获取龙虎榜机构成交明细

## 请求参数

| 参数 | 类型 | 必选 | 说明 |
|------|------|------|------|
| trade_date | str | Y | 交易日期 |
| ts_code | str | N | TS代码 |

## 返回字段

| 字段 | 类型 | 说明 |
|------|------|------|
| trade_date | str | 交易日期 |
| ts_code | str | TS代码 |
| exalter | str | 营业部名称 |
| side | str | 买卖类型（0=买入前5名，1=卖出前5名） |
| buy | float | 买入额（元） |
| buy_rate | float | 买入占总成交比例 |
| sell | float | 卖出额（元） |
| sell_rate | float | 卖出占总成交比例 |
| net_buy | float | 净成交额（元） |
| reason | str | 上榜理由 |

## 调用示例

```bash
curl -X POST https://www.codebuddy.cn/v2/tool/financedata \
  -H "Content-Type: application/json" \
  -d '{
    "api_name": "top_inst",
    "params": {"trade_date": "20250228"},
    "fields": "trade_date,ts_code,exalter,buy,sell,side,reason"
  }'
```
