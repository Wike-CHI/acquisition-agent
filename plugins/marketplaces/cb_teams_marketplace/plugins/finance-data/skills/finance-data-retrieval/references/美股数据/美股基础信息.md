# 美股基础信息

> 接口：`us_basic` | 获取美股列表基础信息

## 请求参数

| 参数 | 类型 | 必选 | 说明 |
|------|------|------|------|
| ts_code | str | N | 股票代码，如AAPL |
| classify | str | N | 股票分类：ADR/GDR/EQ |
| offset | str | N | 起始行号 |
| limit | str | N | 每页最大行数 |

## 返回字段

| 字段 | 类型 | 说明 |
|------|------|------|
| ts_code | str | 美股代码 |
| name | str | 中文名称 |
| enname | str | 英文名称 |
| classify | str | 分类（ADR/GDR/EQ） |
| list_date | str | 上市日期 |
| delist_date | str | 退市日期 |

## 调用示例

```bash
curl -X POST https://www.codebuddy.cn/v2/tool/financedata \
  -H "Content-Type: application/json" \
  -d '{
    "api_name": "us_basic",
    "params": {"ts_code": "AAPL"},
    "fields": "ts_code,name,enname,classify,list_date,delist_date"
  }'
```

## 返回示例

```json
{
  "code": 0,
  "msg": "",
  "data": {
    "fields": ["ts_code", "name", "enname", "classify", "list_date", "delist_date"],
    "items": [
      ["AAPL", null, "APPLE", "EQ", "19801212", null]
    ]
  }
}
```
