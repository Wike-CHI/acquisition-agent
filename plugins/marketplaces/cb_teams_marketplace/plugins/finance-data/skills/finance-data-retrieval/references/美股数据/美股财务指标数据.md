# 美股财务指标数据

> 接口：`us_fina_indicator` | 获取美股上市公司财务指标数据，覆盖主要美股和中概股

## 请求参数

| 参数 | 类型 | 必选 | 说明 |
|------|------|------|------|
| ts_code | str | Y | 股票代码 |
| period | str | N | 报告期（YYYYMMDD，每季最后一天，如20241231） |
| report_type | str | N | 报告类型：Q1-一季报 Q2-半年报 Q3-三季报 Q4-年报 |
| start_date | str | N | 报告期开始日期 |
| end_date | str | N | 报告期结束日期 |

## 返回字段

| 字段 | 类型 | 说明 |
|------|------|------|
| ts_code | str | 股票代码 |
| end_date | str | 报告期 |
| ind_type | str | 报告期类型（Q1/Q2/Q3/Q4） |
| security_name_abbr | str | 股票名称 |
| accounting_standards | str | 会计准则 |
| notice_date | str | 公告日期 |
| start_date | str | 报告期开始日期 |
| std_report_date | str | 标准报告日期 |
| financial_date | str | 年结日期 |
| currency | str | 货币 |
| date_type | str | 报告期类型 |
| report_type | str | 报告类型 |
| operate_income | float | 营业收入 |
| operate_income_yoy | float | 营收同比增长（%） |
| gross_profit | float | 毛利润 |
| gross_profit_yoy | float | 毛利润同比增长（%） |
| parent_holder_netprofit | float | 归母净利润 |
| parent_holder_netprofit_yoy | float | 净利润同比增长（%） |
| basic_eps | float | 基本每股收益 |
| diluted_eps | float | 稀释每股收益 |
| gross_profit_ratio | float | 毛利率（%） |
| net_profit_ratio | float | 净利率（%） |
| accounts_rece_tr | float | 应收账款周转率（次） |
| inventory_tr | float | 存货周转率（次） |
| total_assets_tr | float | 总资产周转率（次） |
| roe_avg | float | 平均净资产收益率（%） |
| roa | float | 总资产收益率（%） |
| current_ratio | float | 流动比率 |
| speed_ratio | float | 速动比率 |
| debt_asset_ratio | float | 资产负债率（%） |
| equity_ratio | float | 权益乘数 |
| currency_abbr | str | 货币缩写 |

## 调用示例

```bash
curl -X POST https://www.codebuddy.cn/v2/tool/financedata \
  -H "Content-Type: application/json" \
  -d '{
    "api_name": "us_fina_indicator",
    "params": {"ts_code": "AAPL", "period": "20241231"},
    "fields": "ts_code,end_date,ind_type,security_name_abbr,operate_income,gross_profit_ratio,net_profit_ratio,roe_avg,roa,current_ratio,debt_asset_ratio"
  }'
```
