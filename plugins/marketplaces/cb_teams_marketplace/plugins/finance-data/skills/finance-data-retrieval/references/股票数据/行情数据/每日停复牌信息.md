# 每日停复牌信息

> 接口：`suspend_d` | 获取每日股票停复牌信息

## 请求参数

| 参数 | 类型 | 必选 | 说明 |
|------|------|------|------|
| ts_code | str | N | 股票代码（支持多值输入） |
| trade_date | str | N | 交易日期（YYYYMMDD） |
| start_date | str | N | 开始日期 |
| end_date | str | N | 结束日期 |
| suspend_type | str | N | 停复牌类型：S-停牌，R-复牌 |

## 返回字段

| 字段 | 类型 | 说明 |
|------|------|------|
| ts_code | str | TS代码 |
| trade_date | str | 停复牌日期 |
| suspend_timing | str | 日内停牌时间段 |
| suspend_type | str | 停复牌类型：S-停牌，R-复牌 |

## 调用示例

```bash
curl -X POST https://www.codebuddy.cn/v2/tool/financedata \
  -H "Content-Type: application/json" \
  -d '{
    "api_name": "suspend_d",
    "params": {"trade_date": "20250102", "suspend_type": "S"},
    "fields": ""
  }'
```

## 返回示例

```json
{
  "code": 0,
  "msg": "",
  "data": {
    "fields": ["ts_code", "trade_date", "suspend_timing", "suspend_type"],
    "items": [
      ["000016.SZ", "20250102", null, "S"],
      ["000421.SZ", "20250102", null, "S"],
      ["000777.SZ", "20250102", null, "S"]
    ]
  }
}
```
