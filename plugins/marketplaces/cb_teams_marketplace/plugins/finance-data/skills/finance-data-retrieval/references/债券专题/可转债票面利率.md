# 可转债票面利率

> 接口：`cb_rate` | 获取可转债票面利率信息

## 请求参数

| 参数 | 类型 | 必选 | 说明 |
|------|------|------|------|
| ts_code | str | Y | 转债代码，支持多值输入 |

## 返回字段

| 字段 | 类型 | 说明 |
|------|------|------|
| ts_code | str | 转债代码 |
| rate_freq | int | 付息频率（次/年） |
| rate_start_date | str | 付息起始日 |
| rate_end_date | str | 付息截止日 |
| coupon_rate | float | 票面利率（%） |

## 调用示例

```bash
curl -X POST https://www.codebuddy.cn/v2/tool/financedata \
  -H "Content-Type: application/json" \
  -d '{
    "api_name": "cb_rate",
    "params": {"ts_code": "113009.SH"},
    "fields": "ts_code,rate_freq,rate_start_date,rate_end_date,coupon_rate"
  }'
```

## 返回示例

```json
{
  "code": 0,
  "msg": "",
  "data": {
    "fields": ["ts_code", "rate_freq", "rate_start_date", "rate_end_date", "coupon_rate"],
    "items": [
      ["113009.SH", 1, "20160122", "20170121", 0.2],
      ["113009.SH", 1, "20170122", "20180121", 0.5],
      ["113009.SH", 1, "20180122", "20190121", 1.0]
    ]
  }
}
```
