# 可转债技术面因子（专业版）

> 接口：`cb_factor_pro` | 获取可转债日线技术因子数据，用于跟踪当前趋势状况。后缀 `_bfq` 为不复权，`_qfq` 为前复权，`_hfq` 为后复权

## 请求参数

| 参数 | 类型 | 必选 | 说明 |
|------|------|------|------|
| ts_code | str | N | 转债代码 |
| start_date | str | N | 开始日期 |
| end_date | str | N | 结束日期 |
| trade_date | str | N | 交易日期 |

## 返回字段

| 字段 | 类型 | 说明 |
|------|------|------|
| ts_code | str | 转债代码 |
| trade_date | str | 交易日期 |
| open | float | 开盘价 |
| high | float | 最高价 |
| low | float | 最低价 |
| close | float | 收盘价 |
| pre_close | float | 昨收盘价 |
| change | float | 涨跌额 |
| pct_change | float | 涨跌幅 |
| vol | float | 成交量（手） |
| amount | float | 成交金额（万元） |
| macd_bfq / macd_dea_bfq / macd_dif_bfq | float | MACD指标 |
| kdj_bfq / kdj_k_bfq / kdj_d_bfq | float | KDJ指标 |
| rsi_bfq_6 / rsi_bfq_12 / rsi_bfq_24 | float | RSI指标 |
| boll_lower_bfq / boll_mid_bfq / boll_upper_bfq | float | 布林带 |
| ma_bfq_5 / ma_bfq_10 / ma_bfq_20 / ma_bfq_60 | float | 均线 |
| cci_bfq | float | CCI指标 |
| obv_bfq | float | OBV能量潮 |
| wr_bfq / wr1_bfq | float | 威廉指标 |

> 完整字段列表包含ASI、ATR、BBI、BIAS、BRAR、CR、DFMA、DMI、EMA、EMV、EXPMA、KTN、MASS、MFI、MTM、PSY、ROC、TRIX、VR、XSII等数十个技术指标

## 调用示例

```bash
curl -X POST https://www.codebuddy.cn/v2/tool/financedata \
  -H "Content-Type: application/json" \
  -d '{
    "api_name": "cb_factor_pro",
    "params": {"ts_code": "113632.SH", "start_date": "20250220", "end_date": "20250228"},
    "fields": "ts_code,trade_date,open,high,low,close,vol,amount,macd_bfq,kdj_k_bfq,rsi_bfq_6"
  }'
```
