# 可转债转股结果

> 接口：`cb_share` | 获取可转债转股结果数据

## 请求参数

| 参数 | 类型 | 必选 | 说明 |
|------|------|------|------|
| ts_code | str | Y | 转债代码，支持多值输入 |
| ann_date | str | Y | 公告日期（YYYYMMDD） |
| start_date | str | N | 公告开始日期 |
| end_date | str | N | 公告结束日期 |

## 返回字段

| 字段 | 类型 | 说明 |
|------|------|------|
| ts_code | str | 转债代码 |
| bond_short_name | str | 转债简称 |
| publish_date | str | 公告日期 |
| end_date | str | 统计截止日 |
| issue_size | float | 可转债发行总额 |
| convert_price_initial | float | 初始转股价 |
| convert_price | float | 当期转股价 |
| convert_val | float | 当期转股金额 |
| convert_vol | float | 当期转股数量 |
| convert_ratio | float | 当期转股比例 |
| acc_convert_val | float | 累计转股金额 |
| acc_convert_vol | float | 累计转股数量 |
| acc_convert_ratio | float | 累计转股比例 |
| remain_size | float | 可转债余额 |
| total_shares | float | 转股后总股本 |

## 调用示例

```bash
curl -X POST https://www.codebuddy.cn/v2/tool/financedata \
  -H "Content-Type: application/json" \
  -d '{
    "api_name": "cb_share",
    "params": {"ts_code": "113009.SH"},
    "fields": "ts_code,end_date,convert_price,convert_val,convert_ratio,acc_convert_ratio"
  }'
```
