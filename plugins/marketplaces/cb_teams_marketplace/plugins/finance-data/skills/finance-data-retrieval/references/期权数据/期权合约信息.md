# 期权合约信息

> 接口：`opt_basic` | 获取期权合约信息

## 请求参数

| 参数 | 类型 | 必选 | 说明 |
|------|------|------|------|
| ts_code | str | N | 期权代码 |
| exchange | str | N | 交易所代码（如SSE） |
| list_date | str | N | 上市日期 |
| opt_code | str | N | 标准合约代码（OP+期货合约TS_CODE，如OPP2207.DCE） |
| call_put | str | N | 期权类型 |

## 返回字段

| 字段 | 类型 | 说明 |
|------|------|------|
| ts_code | str | 期权代码 |
| exchange | str | 交易市场 |
| name | str | 合约名称 |
| per_unit | str | 合约单位 |
| opt_code | str | 标的合约代码 |
| opt_type | str | 合约类型 |
| call_put | str | 期权类型（C-认购 P-认沽） |
| exercise_type | str | 行权方式 |
| exercise_price | float | 行权价 |
| s_month | str | 结算月 |
| maturity_date | str | 到期日 |
| list_price | float | 挂牌基准价 |
| list_date | str | 开始交易日期 |
| delist_date | str | 最后交易日期 |
| last_edate | str | 最后行权日 |
| last_ddate | str | 最后交割日 |
| quote_unit | str | 报价单位 |
| min_price_chg | str | 最小价格变动 |

## 调用示例

```bash
curl -X POST https://www.codebuddy.cn/v2/tool/financedata \
  -H "Content-Type: application/json" \
  -d '{
    "api_name": "opt_basic",
    "params": {"exchange": "SSE"},
    "fields": "ts_code,exchange,name,call_put,exercise_type,exercise_price,list_date,delist_date"
  }'
```
