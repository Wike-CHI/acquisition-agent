# 东方财富概念板块

> 接口：`dc_index` | 获取东方财富每个交易日的概念板块、行业板块、地域板块数据

## 请求参数

| 参数 | 类型 | 必选 | 说明 |
|------|------|------|------|
| ts_code | str | N | 指数代码（支持多个，逗号分隔） |
| name | str | N | 板块名称 |
| trade_date | str | N | 交易日期（YYYYMMDD格式） |
| start_date | str | N | 开始日期 |
| end_date | str | N | 结束日期 |
| idx_type | str | Y | 板块类型（行业板块、概念板块、地域板块） |

## 返回字段

| 字段 | 类型 | 说明 |
|------|------|------|
| ts_code | str | 概念代码 |
| trade_date | str | 交易日期 |
| name | str | 概念名称 |
| leading | str | 领涨股票名称 |
| leading_code | str | 领涨股票代码 |
| pct_change | float | 涨跌幅 |
| leading_pct | float | 领涨股票涨跌幅 |
| total_mv | float | 总市值（万元） |
| turnover_rate | float | 换手率 |
| up_num | int | 上涨家数 |
| down_num | int | 下降家数 |
| idx_type | str | 板块类型 |
| level | str | 行业层级 |

## 调用示例

```bash
curl -X POST https://www.codebuddy.cn/v2/tool/financedata \
  -H "Content-Type: application/json" \
  -d '{
    "api_name": "dc_index",
    "params": {"trade_date": "20250228", "idx_type": "概念板块"},
    "fields": "ts_code,name,turnover_rate,up_num,down_num"
  }'
```
