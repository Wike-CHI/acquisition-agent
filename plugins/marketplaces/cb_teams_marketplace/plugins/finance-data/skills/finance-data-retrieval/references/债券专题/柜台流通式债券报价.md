# 柜台流通式债券报价

> 接口：`bc_otcqt` | 获取柜台流通式债券报价数据

## 请求参数

| 参数 | 类型 | 必选 | 说明 |
|------|------|------|------|
| trade_date | str | N | 报价日期（YYYYMMDD） |
| start_date | str | N | 开始日期 |
| end_date | str | N | 结束日期 |
| ts_code | str | N | 债券代码 |
| bank | str | N | 报价机构 |

## 返回字段

| 字段 | 类型 | 说明 |
|------|------|------|
| trade_date | str | 报价日期 |
| qt_time | str | 报价时间 |
| bank | str | 报价机构 |
| ts_code | str | 债券代码 |
| name | str | 债券名称 |
| maturity | str | 期限 |
| remain_maturity | str | 剩余期限 |
| bond_type | str | 债券类型 |
| coupon_rate | float | 票面利率（%） |
| buy_price | float | 投资者买入价（全价） |
| sell_price | float | 投资者卖出价（全价） |
| buy_yield | float | 投资者买入到期收益率（%） |
| sell_yield | float | 投资者卖出到期收益率（%） |

## 调用示例

```bash
curl -X POST https://www.codebuddy.cn/v2/tool/financedata \
  -H "Content-Type: application/json" \
  -d '{
    "api_name": "bc_otcqt",
    "params": {"ts_code": "200013.BC", "start_date": "20240325", "end_date": "20240329"},
    "fields": "trade_date,qt_time,bank,ts_code,name,remain_maturity,buy_yield,sell_yield"
  }'
```
