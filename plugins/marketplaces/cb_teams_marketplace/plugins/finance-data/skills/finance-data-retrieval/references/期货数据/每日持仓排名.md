# 每日持仓排名

> 接口：`fut_holding` | 获取期货每日成交持仓排名数据

## 请求参数

| 参数 | 类型 | 必选 | 说明 |
|------|------|------|------|
| trade_date | str | N | 交易日期（trade_date和symbol至少输入一个） |
| symbol | str | N | 合约或品种代码 |
| start_date | str | N | 开始日期（YYYYMMDD格式） |
| end_date | str | N | 结束日期 |
| exchange | str | N | 交易所代码 |

## 返回字段

| 字段 | 类型 | 说明 |
|------|------|------|
| trade_date | str | 交易日期 |
| symbol | str | 合约代码 |
| broker | str | 期货公司会员简称 |
| vol | int | 成交量 |
| vol_chg | int | 成交量变化 |
| long_hld | int | 持买仓量 |
| long_chg | int | 持买仓变化 |
| short_hld | int | 持卖仓量 |
| short_chg | int | 持卖仓变化 |
| exchange | str | 交易所 |

## 调用示例

```bash
curl -X POST https://www.codebuddy.cn/v2/tool/financedata \
  -H "Content-Type: application/json" \
  -d '{
    "api_name": "fut_holding",
    "params": {"trade_date": "20250110", "symbol": "CU", "exchange": "SHFE"},
    "fields": "trade_date,symbol,broker,vol,vol_chg,long_hld,long_chg,short_hld,short_chg"
  }'
```

## 返回示例

```json
{
  "code": 0,
  "msg": "",
  "data": {
    "fields": ["trade_date", "symbol", "broker", "vol", "vol_chg", "long_hld", "long_chg", "short_hld", "short_chg"],
    "items": [
      ["20250110", "CU", "期货公司", 243625, -9314, 401802, -2619, 399093, -2522],
      ["20250110", "CU", "非期货公司", 2431, -338, 6147, 312, 8856, 215]
    ]
  }
}
```
