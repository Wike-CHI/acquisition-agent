# 公告原文

> 接口：`anns_d` | 获取上市公司全量公告数据，提供PDF下载链接

## 请求参数

| 参数 | 类型 | 必选 | 说明 |
|------|------|------|------|
| ts_code | str | N | 股票代码 |
| ann_date | str | N | 公告日期（YYYYMMDD格式） |
| start_date | str | N | 公告开始日期 |
| end_date | str | N | 公告结束日期 |

**限量：** 单次最大2000条。**权限：** 需单独开通权限。

## 返回字段

| 字段 | 类型 | 说明 |
|------|------|------|
| ann_date | str | 公告日期 |
| ts_code | str | 股票代码 |
| name | str | 股票名称 |
| title | str | 标题 |
| url | str | 原文下载链接 |
| rec_time | datetime | 发布时间（默认不显示） |

## 调用示例

```bash
curl -X POST https://www.codebuddy.cn/v2/tool/financedata \
  -H "Content-Type: application/json" \
  -d '{
    "api_name": "anns_d",
    "params": {"ann_date": "20230621"},
    "fields": ""
  }'
```
