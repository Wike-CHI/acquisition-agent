# 实时成交数据

> 接口：`realtime_tick` | 获取股票当日开盘以来全部分笔成交数据，爬虫接口，仅支持Python SDK调用

## 请求参数

| 参数 | 类型 | 必选 | 说明 |
|------|------|------|------|
| ts_code | str | N | 股票代码（如 000001.SZ），仅支持单只 |
| src | str | N | 数据源：sina-新浪（默认），dc-东方财富 |

## 返回字段

| 字段 | 类型 | 说明 |
|------|------|------|
| time | str | 交易时间 |
| price | float | 现价 |
| change | float | 价格变动 |
| volume | int | 成交量（手） |
| amount | int | 成交金额（元） |
| type | str | 类型：买入/卖出/中性 |

## 调用说明

此接口为爬虫接口，数据来自网络，**不支持HTTP API调用**，仅支持Python SDK。0积分开放，需tushare账号。

```python
import tushare as ts

ts.set_token('YOUR_TOKEN')

# 新浪数据
df = ts.realtime_tick(ts_code='600000.SH')

# 东财数据
df = ts.realtime_tick(ts_code='600000.SH', src='dc')
```
