# 基金列表

> 接口：`fund_basic` | 获取公募基金基础信息数据，包括场内和场外基金

## 请求参数

| 参数 | 类型 | 必选 | 说明 |
|------|------|------|------|
| ts_code | str | N | 基金代码 |
| market | str | N | 交易市场：E-场内 O-场外（默认E） |
| status | str | N | 存续状态：D-摘牌 I-发行中 L-已上市 |

## 返回字段

| 字段 | 类型 | 说明 |
|------|------|------|
| ts_code | str | 基金代码 |
| name | str | 基金名称 |
| management | str | 基金管理人 |
| custodian | str | 托管银行 |
| fund_type | str | 投资类型 |
| found_date | str | 成立日期 |
| due_date | str | 到期日期 |
| list_date | str | 上市日期 |
| issue_date | str | 发行日期 |
| delist_date | str | 退市日期 |
| issue_amount | float | 发行份额（亿） |
| m_fee | float | 管理费 |
| c_fee | float | 托管费 |
| duration_year | float | 存续期（年） |
| p_value | float | 面值 |
| min_amount | float | 起点金额（万元） |
| exp_return | float | 预期收益率 |
| benchmark | str | 业绩比较基准 |
| status | str | 存续状态（D/I/L） |
| invest_type | str | 投资风格 |
| type | str | 基金类型 |
| trustee | str | 受托人 |
| purc_startdate | str | 日常申购起始日 |
| redm_startdate | str | 日常赎回起始日 |
| market | str | E-场内 O-场外 |

## 调用示例

```bash
curl -X POST https://www.codebuddy.cn/v2/tool/financedata \
  -H "Content-Type: application/json" \
  -d '{
    "api_name": "fund_basic",
    "params": {"market": "E"},
    "fields": "ts_code,name,management,custodian,fund_type,found_date"
  }'
```
