# 股票曾用名

> 接口：`namechange` | 获取股票历史名称变更记录

## 请求参数

| 参数 | 类型 | 必选 | 说明 |
|------|------|------|------|
| ts_code | str | N | TS股票代码 |
| start_date | str | N | 公告开始日期 |
| end_date | str | N | 公告结束日期 |

## 返回字段

| 字段 | 类型 | 说明 |
|------|------|------|
| ts_code | str | TS股票代码 |
| name | str | 证券名称 |
| start_date | str | 开始日期 |
| end_date | str | 结束日期 |
| ann_date | str | 公告日期 |
| change_reason | str | 变更原因 |

## 调用示例

```bash
curl -X POST https://www.codebuddy.cn/v2/tool/financedata \
  -H "Content-Type: application/json" \
  -d '{
    "api_name": "namechange",
    "params": {"ts_code": "600848.SH"},
    "fields": ""
  }'
```

## 返回示例

```json
{
  "code": 0,
  "msg": "",
  "data": {
    "fields": ["ts_code", "name", "start_date", "end_date", "ann_date", "change_reason"],
    "items": [
      ["600848.SH", "上海临港", "20151118", null, "20151113", "改名"],
      ["600848.SH", "上海临港", "20151118", null, "20151113", "改名"],
      ["600848.SH", "自仪股份", "20070514", "20151117", null, "撤销ST"]
    ]
  }
}
```
