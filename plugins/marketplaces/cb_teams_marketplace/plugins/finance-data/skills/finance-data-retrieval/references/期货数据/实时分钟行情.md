# 实时分钟行情

> 接口：`rt_fut_min` | 获取全市场期货合约实时分钟数据，支持1/5/15/30/60分钟

## 请求参数

| 参数 | 类型 | 必选 | 说明 |
|------|------|------|------|
| ts_code | str | Y | 合约代码（如CU2501.SHF），支持多合约逗号分隔 |
| freq | str | Y | 分钟频度：1MIN/5MIN/15MIN/30MIN/60MIN |

## 返回字段

| 字段 | 类型 | 说明 |
|------|------|------|
| code | str | 合约代码 |
| freq | str | 频度 |
| time | str | 交易时间 |
| open | float | 开盘价 |
| close | float | 收盘价 |
| high | float | 最高价 |
| low | float | 最低价 |
| vol | int | 成交量 |
| amount | float | 成交金额 |
| oi | float | 持仓量 |

## 调用示例

```bash
curl -X POST https://www.codebuddy.cn/v2/tool/financedata \
  -H "Content-Type: application/json" \
  -d '{
    "api_name": "rt_fut_min",
    "params": {"ts_code": "CU2501.SHF", "freq": "1MIN"},
    "fields": ""
  }'
```

## 补充接口

> 接口：`rt_fut_min_daily` | 当日分钟快照回放，仅支持单个合约

### 请求参数

| 参数 | 类型 | 必选 | 说明 |
|------|------|------|------|
| ts_code | str | Y | 合约代码，仅支持单个 |
| freq | str | Y | 分钟频度：1MIN/5MIN/15MIN/30MIN/60MIN |
| date_str | str | N | 回放日期（格式：YYYY-MM-DD，默认当日，支持回溯一天） |
