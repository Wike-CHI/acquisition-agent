# 财报披露日期表

> 接口：`disclosure_date` | 获取上市公司财报披露计划日期数据

## 请求参数

| 参数 | 类型 | 必选 | 说明 |
|------|------|------|------|
| ts_code | str | N | TS股票代码 |
| end_date | str | N | 报告期（如20240930） |
| pre_date | str | N | 计划披露日期 |
| ann_date | str | N | 最新披露公告日 |
| actual_date | str | N | 实际披露日期 |

## 返回字段

| 字段 | 类型 | 说明 |
|------|------|------|
| ts_code | str | TS代码 |
| ann_date | str | 最新披露公告日 |
| end_date | str | 报告期 |
| pre_date | str | 计划披露日期 |
| actual_date | str | 实际披露日期 |

## 调用示例

```bash
curl -X POST https://www.codebuddy.cn/v2/tool/financedata \
  -H "Content-Type: application/json" \
  -d '{
    "api_name": "disclosure_date",
    "params": {"end_date": "20240930"},
    "fields": ""
  }'
```

## 返回示例

```json
{
  "code": 0,
  "msg": "",
  "data": {
    "fields": ["ts_code", "ann_date", "end_date", "pre_date", "actual_date"],
    "items": [
      ["920060.BJ", null, "20240930", null, "20241031"],
      ["920101.BJ", null, "20240930", null, "20241108"],
      ["920160.BJ", null, "20240930", null, "20241127"]
    ]
  }
}
```
