# 实时涨跌幅排名

> 接口：`realtime_list` | 获取全A股实时涨跌幅排名数据，爬虫接口，仅支持Python SDK调用

## 请求参数

| 参数 | 类型 | 必选 | 说明 |
|------|------|------|------|
| src | str | N | 数据源：dc-东方财富（默认），sina-新浪 |

## 返回字段（东财源）

| 字段 | 类型 | 说明 |
|------|------|------|
| ts_code | str | 股票代码 |
| name | str | 股票名称 |
| price | float | 当前价格 |
| pct_change | float | 涨跌幅 |
| change | float | 涨跌额 |
| volume | int | 成交量（手） |
| amount | int | 成交金额（元） |
| swing | float | 振幅 |
| low | float | 最低价 |
| high | float | 最高价 |
| open | float | 开盘价 |
| close | float | 收盘价 |
| vol_ratio | int | 量比 |
| turnover_rate | float | 换手率 |
| pe | int | 市盈率 |
| pb | float | 市净率 |
| total_mv | float | 总市值（元） |
| float_mv | float | 流通市值（元） |
| rise | float | 涨速 |
| 5min | float | 5分钟涨幅 |
| 60day | float | 60天涨幅 |
| 1tyear | float | 1年涨幅 |

## 返回字段（新浪源）

| 字段 | 类型 | 说明 |
|------|------|------|
| ts_code | str | 股票代码 |
| name | str | 股票名称 |
| price | float | 当前价格 |
| pct_change | float | 涨跌幅 |
| change | float | 涨跌额 |
| buy | float | 买入价 |
| sale | float | 卖出价 |
| close | float | 收盘价 |
| open | float | 开盘价 |
| high | float | 最高价 |
| low | float | 最低价 |
| volume | int | 成交量（股） |
| amount | int | 成交金额（元） |
| time | str | 当前时间 |

## 调用说明

此接口为爬虫接口，数据来自网络，**不支持HTTP API调用**，仅支持Python SDK。0积分开放，需tushare账号。

```python
import tushare as ts

# 东财数据
df = ts.realtime_list(src='dc')

# 新浪数据
df = ts.realtime_list(src='sina')
```
