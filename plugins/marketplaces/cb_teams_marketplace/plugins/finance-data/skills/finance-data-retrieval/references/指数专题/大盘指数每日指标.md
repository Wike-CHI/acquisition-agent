# 大盘指数每日指标

> 接口：`index_dailybasic` | 获取大盘指数每日指标数据，包括市值、换手率、市盈率等，数据自2004年起

## 请求参数

| 参数 | 类型 | 必选 | 说明 |
|------|------|------|------|
| trade_date | str | N | 交易日期（YYYYMMDD） |
| ts_code | str | N | 指数代码 |
| start_date | str | N | 开始日期 |
| end_date | str | N | 结束日期 |

> 注：trade_date 和 ts_code 至少需要填一个，单次最大3000行

## 返回字段

| 字段 | 类型 | 说明 |
|------|------|------|
| ts_code | str | 指数代码 |
| trade_date | str | 交易日期 |
| total_mv | float | 当日总市值（元） |
| float_mv | float | 当日流通市值（元） |
| total_share | float | 当日总股本（股） |
| float_share | float | 当日流通股本（股） |
| free_share | float | 当日自由流通股本（股） |
| turnover_rate | float | 换手率 |
| turnover_rate_f | float | 换手率（自由流通股本） |
| pe | float | 市盈率 |
| pe_ttm | float | 市盈率（TTM） |
| pb | float | 市净率 |

## 调用示例

```bash
curl -X POST https://www.codebuddy.cn/v2/tool/financedata \
  -H "Content-Type: application/json" \
  -d '{
    "api_name": "index_dailybasic",
    "params": {"ts_code": "000001.SH", "trade_date": "20250228"},
    "fields": "ts_code,trade_date,turnover_rate,pe,pe_ttm,pb"
  }'
```
