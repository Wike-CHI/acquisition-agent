# 限售股解禁

> 接口：`share_float` | 获取限售股解禁数据

## 请求参数

| 参数 | 类型 | 必选 | 说明 |
|------|------|------|------|
| ts_code | str | N | TS股票代码 |
| ann_date | str | N | 公告日期 |
| float_date | str | N | 解禁日期 |
| start_date | str | N | 解禁开始日期 |
| end_date | str | N | 解禁结束日期 |

## 返回字段

| 字段 | 类型 | 说明 |
|------|------|------|
| ts_code | str | TS代码 |
| ann_date | str | 公告日期 |
| float_date | str | 解禁日期 |
| float_share | float | 流通股份（股） |
| float_ratio | float | 流通股份占总股本比例 |
| holder_name | str | 股东名称 |
| share_type | str | 股份类型 |

## 调用示例

```bash
curl -X POST https://www.codebuddy.cn/v2/tool/financedata \
  -H "Content-Type: application/json" \
  -d '{
    "api_name": "share_float",
    "params": {"ts_code": "000001.SZ"},
    "fields": ""
  }'
```
