# 大宗交易明细

> 接口：`bond_blk_detail` | 获取沪深交易所债券大宗交易明细数据

> 注：本接口目前只有深交所的大宗交易明细，上交所明细已包含在大宗交易接口中

## 请求参数

| 参数 | 类型 | 必选 | 说明 |
|------|------|------|------|
| ts_code | str | N | 债券代码 |
| trade_date | str | N | 交易日期（YYYYMMDD） |
| start_date | str | N | 开始日期 |
| end_date | str | N | 结束日期 |

## 返回字段

| 字段 | 类型 | 说明 |
|------|------|------|
| trade_date | str | 交易日期 |
| ts_code | str | 债券代码 |
| name | str | 债券名称 |
| price | float | 成交价（元） |
| vol | float | 成交数量（万股/万份/万张/万手） |
| amount | float | 成交金额（万元） |
| buy_dp | str | 买方营业部 |
| sell_dp | str | 卖方营业部 |

## 调用示例

```bash
curl -X POST https://www.codebuddy.cn/v2/tool/financedata \
  -H "Content-Type: application/json" \
  -d '{
    "api_name": "bond_blk_detail",
    "params": {"trade_date": "20250228"},
    "fields": "trade_date,ts_code,name,price,vol,amount,buy_dp,sell_dp"
  }'
```
