# 港股通每日成交统计

> 接口：`ggt_daily` | 获取港股通每日成交信息，数据从2014年开始

## 请求参数

| 参数 | 类型 | 必选 | 说明 |
|------|------|------|------|
| trade_date | str | N | 交易日期（YYYYMMDD，支持多值输入） |
| start_date | str | N | 开始日期 |
| end_date | str | N | 结束日期 |

## 返回字段

| 字段 | 类型 | 说明 |
|------|------|------|
| trade_date | str | 交易日期 |
| buy_amount | float | 买入成交金额（亿元） |
| buy_volume | float | 买入成交笔数（万笔） |
| sell_amount | float | 卖出成交金额（亿元） |
| sell_volume | float | 卖出成交笔数（万笔） |

## 调用示例

```bash
curl -X POST https://www.codebuddy.cn/v2/tool/financedata \
  -H "Content-Type: application/json" \
  -d '{
    "api_name": "ggt_daily",
    "params": {"trade_date": "20250102"},
    "fields": ""
  }'
```

## 返回示例

```json
{
  "code": 0,
  "msg": "",
  "data": {
    "fields": ["trade_date", "buy_amount", "buy_volume", "sell_amount", "sell_volume"],
    "items": [
      ["20250102", 278.74, 51.6, 212.49, 36.72]
    ]
  }
}
```
