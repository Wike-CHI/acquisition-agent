# 最强板块统计

> 接口：`limit_cpt_list` | 获取每日涨停股票最多的概念板块，可分析板块轮动和资金流向趋势

## 请求参数

| 参数 | 类型 | 必选 | 说明 |
|------|------|------|------|
| trade_date | str | N | 交易日期（YYYYMMDD格式） |
| ts_code | str | N | 板块代码 |
| start_date | str | N | 开始日期 |
| end_date | str | N | 结束日期 |

## 返回字段

| 字段 | 类型 | 说明 |
|------|------|------|
| ts_code | str | 板块代码 |
| name | str | 板块名称 |
| trade_date | str | 交易日期 |
| days | int | 上榜天数 |
| up_stat | str | 连板高度 |
| cons_nums | int | 连板个数 |
| up_nums | int | 涨停个数 |
| pct_chg | float | 涨跌幅% |
| rank | str | 板块热点排名 |

## 调用示例

```bash
curl -X POST https://www.codebuddy.cn/v2/tool/financedata \
  -H "Content-Type: application/json" \
  -d '{
    "api_name": "limit_cpt_list",
    "params": {"trade_date": "20250228"},
    "fields": "ts_code,name,trade_date,up_nums,pct_chg,rank"
  }'
```
