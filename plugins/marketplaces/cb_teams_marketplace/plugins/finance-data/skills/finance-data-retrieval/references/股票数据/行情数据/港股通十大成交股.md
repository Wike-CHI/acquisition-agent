# 港股通十大成交股

> 接口：`ggt_top10` | 获取港股通每日十大成交数据，含沪市、深市详细数据

## 请求参数

| 参数 | 类型 | 必选 | 说明 |
|------|------|------|------|
| ts_code | str | N | 股票代码（与trade_date二选一） |
| trade_date | str | N | 交易日期（与ts_code二选一） |
| start_date | str | N | 开始日期 |
| end_date | str | N | 结束日期 |
| market_type | str | N | 市场类型：2-港股通（沪），4-港股通（深） |

## 返回字段

| 字段 | 类型 | 说明 |
|------|------|------|
| trade_date | str | 交易日期 |
| ts_code | str | 股票代码 |
| name | str | 股票名称 |
| close | float | 收盘价 |
| p_change | float | 涨跌幅 |
| rank | int | 资金排名 |
| market_type | str | 市场类型：2-港股通（沪），4-港股通（深） |
| amount | float | 累计成交金额（元） |
| net_amount | float | 净买入金额（元） |
| sh_amount | float | 沪市成交金额（元） |
| sh_net_amount | float | 沪市净买入金额（元） |
| sh_buy | float | 沪市买入金额（元） |
| sh_sell | float | 沪市卖出金额（元） |
| sz_amount | float | 深市成交金额（元） |
| sz_net_amount | float | 深市净买入金额（元） |
| sz_buy | float | 深市买入金额（元） |
| sz_sell | float | 深市卖出金额（元） |

## 调用示例

```bash
curl -X POST https://www.codebuddy.cn/v2/tool/financedata \
  -H "Content-Type: application/json" \
  -d '{
    "api_name": "ggt_top10",
    "params": {"trade_date": "20250102"},
    "fields": ""
  }'
```
