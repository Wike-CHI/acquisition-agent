# 基金管理人

> 接口：`fund_company` | 获取公募基金管理公司列表

## 请求参数

无需参数，一次性获取全部数据。

## 返回字段

| 字段 | 类型 | 说明 |
|------|------|------|
| name | str | 基金公司名称 |
| shortname | str | 简称 |
| short_enname | str | 英文缩写 |
| province | str | 省份 |
| city | str | 城市 |
| address | str | 注册地址 |
| phone | str | 电话 |
| office | str | 办公地址 |
| website | str | 公司网址 |
| chairman | str | 法人代表 |
| manager | str | 总经理 |
| reg_capital | float | 注册资本 |
| setup_date | str | 成立日期 |
| end_date | str | 公司终止日期 |
| employees | float | 员工总数 |
| main_business | str | 主要产品及业务 |
| org_code | str | 组织机构代码 |
| credit_code | str | 统一社会信用代码 |

## 调用示例

```bash
curl -X POST https://www.codebuddy.cn/v2/tool/financedata \
  -H "Content-Type: application/json" \
  -d '{
    "api_name": "fund_company",
    "params": {},
    "fields": "name,shortname,province,city,setup_date"
  }'
```
