# 周月线行情（每日更新）

> 接口：`stk_weekly_monthly` | 获取股票周/月线行情数据，每日更新

## 请求参数

| 参数 | 类型 | 必选 | 说明 |
|------|------|------|------|
| ts_code | str | N | 股票代码 |
| trade_date | str | N | 交易日期（YYYYMMDD，周/月最后一天） |
| start_date | str | N | 开始日期 |
| end_date | str | N | 结束日期 |
| freq | str | Y | 频度：week-周线，month-月线 |

## 返回字段

| 字段 | 类型 | 说明 |
|------|------|------|
| ts_code | str | 股票代码 |
| trade_date | str | 交易日期 |
| end_date | str | 计算截止日期 |
| freq | str | 频度（week/month） |
| open | float | 开盘价 |
| high | float | 最高价 |
| low | float | 最低价 |
| close | float | 收盘价 |
| pre_close | float | 上期收盘价 |
| vol | float | 成交量 |
| amount | float | 成交额 |
| change | float | 涨跌额 |
| pct_chg | float | 涨跌幅 |

## 调用示例

```bash
curl -X POST https://www.codebuddy.cn/v2/tool/financedata \
  -H "Content-Type: application/json" \
  -d '{
    "api_name": "stk_weekly_monthly",
    "params": {"ts_code": "000001.SZ", "freq": "week", "start_date": "20250101", "end_date": "20250131"},
    "fields": ""
  }'
```
