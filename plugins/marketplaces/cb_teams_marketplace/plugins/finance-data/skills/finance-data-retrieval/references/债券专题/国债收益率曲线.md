# 国债收益率曲线

> 接口：`yc_cb` | 获取中债收益率曲线，包括国债即期和到期收益率曲线数据

## 请求参数

| 参数 | 类型 | 必选 | 说明 |
|------|------|------|------|
| ts_code | str | N | 收益率曲线编码：1001.CB-国债收益率曲线 |
| curve_type | str | N | 曲线类型：0-到期，1-即期 |
| trade_date | str | N | 交易日期 |
| start_date | str | N | 查询起始日期 |
| end_date | str | N | 查询结束日期 |
| curve_term | float | N | 期限 |

## 返回字段

| 字段 | 类型 | 说明 |
|------|------|------|
| trade_date | str | 交易日期 |
| ts_code | str | 曲线编码 |
| curve_name | str | 曲线名称 |
| curve_type | str | 曲线类型：0-到期，1-即期 |
| curve_term | float | 期限（年） |
| yield | float | 收益率（%） |

## 调用示例

```bash
curl -X POST https://www.codebuddy.cn/v2/tool/financedata \
  -H "Content-Type: application/json" \
  -d '{
    "api_name": "yc_cb",
    "params": {"ts_code": "1001.CB", "curve_type": "0", "trade_date": "20250228"},
    "fields": "trade_date,ts_code,curve_name,curve_type,curve_term,yield"
  }'
```

## 返回示例

```json
{
  "code": 0,
  "msg": "",

      ["20250228", "1001.CB", "中债国债收益率曲线", "0", 0.0, 1.5],
      ["20250228", "1001.CB", "中债国债收益率曲线", "0", 0.08, 1.5578],
      ["20250228", "1001.CB", "中债国债收益率曲线", "0", 0.1, 1.5668]
    ]
  }
```
