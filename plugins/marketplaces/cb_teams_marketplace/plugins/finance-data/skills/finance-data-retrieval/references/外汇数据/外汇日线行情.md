# 外汇日线行情

> 接口：`fx_daily` | 获取外汇日线行情数据

## 请求参数

| 参数 | 类型 | 必选 | 说明 |
|------|------|------|------|
| ts_code | str | N | TS代码 |
| trade_date | str | N | 交易日期（GMT，比北京时间晚一天） |
| start_date | str | N | 开始日期（GMT） |
| end_date | str | N | 结束日期（GMT） |
| exchange | str | N | 交易商，目前只有FXCM |

## 返回字段

| 字段 | 类型 | 说明 |
|------|------|------|
| ts_code | str | 外汇代码 |
| trade_date | str | 交易日期 |
| bid_open | float | 买入开盘价 |
| bid_close | float | 买入收盘价 |
| bid_high | float | 买入最高价 |
| bid_low | float | 买入最低价 |
| ask_open | float | 卖出开盘价 |
| ask_close | float | 卖出收盘价 |
| ask_high | float | 卖出最高价 |
| ask_low | float | 卖出最低价 |
| tick_qty | int | 报价笔数 |
| exchange | str | 交易商 |

## 调用示例

```bash
curl -X POST https://www.codebuddy.cn/v2/tool/financedata \
  -H "Content-Type: application/json" \
  -d '{
    "api_name": "fx_daily",
    "params": {"ts_code": "USDCNH.FXCM", "start_date": "20250101", "end_date": "20250201"},
    "fields": "ts_code,trade_date,bid_open,bid_close,bid_high,bid_low,ask_open,ask_close,ask_high,ask_low,tick_qty"
  }'
```
