# 实时日线

> 接口：`rt_k` | 获取沪深京A股实时日线行情数据，支持通配符批量获取

## 请求参数

| 参数 | 类型 | 必选 | 说明 |
|------|------|------|------|
| ts_code | str | Y | 股票代码，支持通配符（如 `6*.SH` 全部沪市，`3*.SZ` 创业板，`688*.SH` 科创板），或单只如 `000001.SZ` |

## 返回字段

| 字段 | 类型 | 说明 |
|------|------|------|
| ts_code | str | 股票代码 |
| name | str | 股票名称 |
| pre_close | float | 昨收价 |
| high | float | 最高价 |
| open | float | 开盘价 |
| low | float | 最低价 |
| close | float | 收盘价（最新价） |
| vol | int | 成交量（股） |
| amount | int | 成交额（元） |
| num | int | 成交笔数 |

## 调用示例

```bash
curl -X POST https://www.codebuddy.cn/v2/tool/financedata \
  -H "Content-Type: application/json" \
  -d '{
    "api_name": "rt_k",
    "params": {"ts_code": "000001.SZ"},
    "fields": ""
  }'
```

## 返回示例

```json
{
  "code": 0,
  "msg": "",
  "data": {
    "fields": ["ts_code", "name", "pre_close", "high", "open", "low", "close", "vol", "amount", "num"],
    "items": [
      ["000001.SZ", "平安银行", 10.85, 10.95, 10.85, 10.8, 10.88, 102869483, 1119402076, 60388]
    ]
  }
}
```
