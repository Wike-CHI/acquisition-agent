# 基金规模

> 接口：`fund_share` | 获取基金规模数据，包含上海和深圳ETF基金

## 请求参数

| 参数 | 类型 | 必选 | 说明 |
|------|------|------|------|
| ts_code | str | N | 基金代码 |
| trade_date | str | N | 交易日期 |
| start_date | str | N | 开始日期 |
| end_date | str | N | 结束日期 |
| market | str | N | 市场代码（SH-上交所 SZ-深交所） |

## 返回字段

| 字段 | 类型 | 说明 |
|------|------|------|
| ts_code | str | 基金代码 |
| trade_date | str | 交易（变动）日期 |
| fd_share | float | 基金份额（万） |
| fund_type | str | 基金类型 |
| market | str | 市场 |

## 调用示例

```bash
curl -X POST https://www.codebuddy.cn/v2/tool/financedata \
  -H "Content-Type: application/json" \
  -d '{
    "api_name": "fund_share",
    "params": {"ts_code": "510050.SH"},
    "fields": "ts_code,trade_date,fd_share,fund_type,market"
  }'
```

## 返回示例

```json
{
  "code": 0,
  "msg": "",
  "data": {
    "fields": ["ts_code", "trade_date", "fd_share", "fund_type", "market"],
    "items": [
      ["510050.SH", "20260302", 2295156.68, "ETF", "SH"],
      ["510050.SH", "20260227", 2369046.68, "ETF", "SH"],
      ["510050.SH", "20260226", 2373366.68, "ETF", "SH"]
    ]
  }
}
```
