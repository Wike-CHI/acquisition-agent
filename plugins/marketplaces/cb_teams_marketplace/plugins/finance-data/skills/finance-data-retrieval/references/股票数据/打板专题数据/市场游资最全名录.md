# 游资名录

> 接口：`hm_list` | 获取游资分类名录信息

## 请求参数

| 参数 | 类型 | 必选 | 说明 |
|------|------|------|------|
| name | str | N | 游资名称 |

## 返回字段

| 字段 | 类型 | 说明 |
|------|------|------|
| name | str | 游资名称 |
| desc | str | 描述/备注 |
| orgs | str | 关联机构 |

## 调用示例

```bash
curl -X POST https://www.codebuddy.cn/v2/tool/financedata \
  -H "Content-Type: application/json" \
  -d '{
    "api_name": "hm_list",
    "params": {},
    "fields": "name,desc,orgs"
  }'
```
