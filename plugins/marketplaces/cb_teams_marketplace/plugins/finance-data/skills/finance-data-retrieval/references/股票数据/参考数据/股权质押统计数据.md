# 股权质押统计数据

> 接口：`pledge_stat` | 获取上市公司股权质押统计数据

## 请求参数

| 参数 | 类型 | 必选 | 说明 |
|------|------|------|------|
| ts_code | str | N | TS股票代码 |
| end_date | str | N | 截止日期 |

## 返回字段

| 字段 | 类型 | 说明 |
|------|------|------|
| ts_code | str | TS代码 |
| end_date | str | 截止日期 |
| pledge_count | int | 质押次数 |
| unrest_pledge | float | 无限售股质押数量（万股） |
| rest_pledge | float | 限售股质押数量（万股） |
| total_share | float | 总股本 |
| pledge_ratio | float | 质押比例 |

## 调用示例

```bash
curl -X POST https://www.codebuddy.cn/v2/tool/financedata \
  -H "Content-Type: application/json" \
  -d '{
    "api_name": "pledge_stat",
    "params": {"ts_code": "000001.SZ"},
    "fields": ""
  }'
```

## 返回示例

```json
{
  "code": 0,
  "msg": "",
  "data": {
    "fields": ["ts_code", "end_date", "pledge_count", "unrest_pledge", "rest_pledge", "total_share", "pledge_ratio"],
    "items": [
      ["000001.SZ", "20260227", 6, 1837.85, 0.0, 1940591.82, 0.09],
      ["000001.SZ", "20260213", 6, 1837.85, 0.0, 1940591.82, 0.09],
      ["000001.SZ", "20260206", 6, 1837.85, 0.0, 1940591.82, 0.09]
    ]
  }
}
```
