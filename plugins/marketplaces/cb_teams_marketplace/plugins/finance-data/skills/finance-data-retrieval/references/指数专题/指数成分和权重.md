# 指数成分和权重

> 接口：`index_weight` | 获取各类指数成分和权重数据，月度级别，建议输入当月首末日作为起止日期

## 请求参数

| 参数 | 类型 | 必选 | 说明 |
|------|------|------|------|
| index_code | str | Y | 指数代码 |
| trade_date | str | N | 交易日期（YYYYMMDD） |
| start_date | str | N | 开始日期 |
| end_date | str | N | 结束日期 |

## 返回字段

| 字段 | 类型 | 说明 |
|------|------|------|
| index_code | str | 指数代码 |
| con_code | str | 成分代码 |
| trade_date | str | 交易日期 |
| weight | float | 权重 |

## 调用示例

```bash
curl -X POST https://www.codebuddy.cn/v2/tool/financedata \
  -H "Content-Type: application/json" \
  -d '{
    "api_name": "index_weight",
    "params": {"index_code": "000001.SH", "trade_date": "20250228"},
    "fields": ""
  }'
```

## 返回示例

```json
{
  "code": 0,
  "msg": "",
  "data": {
    "fields": ["index_code", "con_code", "trade_date", "weight"],
    "items": [
      ["000001.SH", "600519.SH", "20250228", 3.654],
      ["000001.SH", "601398.SH", "20250228", 3.59],
      ["000001.SH", "601288.SH", "20250228", 3.174]
    ]
  }
}
```
