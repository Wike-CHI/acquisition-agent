# 美股日线行情

> 接口：`us_daily` | 获取美股日线行情数据（未复权），包含完整历史价格及重要估值指标

## 请求参数

| 参数 | 类型 | 必选 | 说明 |
|------|------|------|------|
| ts_code | str | N | 股票代码，如AAPL |
| trade_date | str | N | 交易日期（YYYYMMDD） |
| start_date | str | N | 开始日期 |
| end_date | str | N | 结束日期 |

## 返回字段

| 字段 | 类型 | 说明 |
|------|------|------|
| ts_code | str | 股票代码 |
| trade_date | str | 交易日期 |
| close | float | 收盘价 |
| open | float | 开盘价 |
| high | float | 最高价 |
| low | float | 最低价 |
| pre_close | float | 昨收价 |
| change | float | 涨跌额 |
| pct_change | float | 涨跌幅（%） |
| vol | float | 成交量 |
| amount | float | 成交金额 |
| vwap | float | 成交均价 |
| turnover_ratio | float | 换手率 |
| total_mv | float | 总市值 |
| pe | float | 市盈率 |
| pb | float | 市净率 |

## 调用示例

```bash
curl -X POST https://www.codebuddy.cn/v2/tool/financedata \
  -H "Content-Type: application/json" \
  -d '{
    "api_name": "us_daily",
    "params": {"ts_code": "AAPL", "start_date": "20250201", "end_date": "20250228"},
    "fields": "ts_code,trade_date,open,high,low,close,pct_change,vol,amount,vwap"
  }'
```
