# 基金净值

> 接口：`fund_nav` | 获取公募基金净值数据

## 请求参数

| 参数 | 类型 | 必选 | 说明 |
|------|------|------|------|
| ts_code | str | N | 基金代码（ts_code和nav_date二选一） |
| nav_date | str | N | 净值日期（ts_code和nav_date二选一） |
| market | str | N | E-场内 O-场外 |
| start_date | str | N | 净值开始日期 |
| end_date | str | N | 净值结束日期 |

## 返回字段

| 字段 | 类型 | 说明 |
|------|------|------|
| ts_code | str | 基金代码 |
| ann_date | str | 公告日期 |
| nav_date | str | 净值日期 |
| unit_nav | float | 单位净值 |
| accum_nav | float | 累计净值 |
| accum_div | float | 累计分红 |
| net_asset | float | 资产净值 |
| total_netasset | float | 合计资产净值 |
| adj_nav | float | 复权单位净值 |
| update_flag | str | 更新标志 |

## 调用示例

```bash
curl -X POST https://www.codebuddy.cn/v2/tool/financedata \
  -H "Content-Type: application/json" \
  -d '{
    "api_name": "fund_nav",
    "params": {"ts_code": "510050.SH"},
    "fields": "ts_code,ann_date,nav_date,unit_nav,accum_nav,adj_nav"
  }'
```
