# 美股交易日历

> 接口：`us_tradecal` | 获取美股交易日历信息

## 请求参数

| 参数 | 类型 | 必选 | 说明 |
|------|------|------|------|
| start_date | str | N | 开始日期（YYYYMMDD） |
| end_date | str | N | 结束日期（YYYYMMDD） |
| is_open | str | N | 交易状态：0-休市 1-交易 |

## 返回字段

| 字段 | 类型 | 说明 |
|------|------|------|
| cal_date | str | 日历日期 |
| is_open | int | 交易状态（0-休市 1-交易） |
| pretrade_date | str | 上一个交易日 |

## 调用示例

```bash
curl -X POST https://www.codebuddy.cn/v2/tool/financedata \
  -H "Content-Type: application/json" \
  -d '{
    "api_name": "us_tradecal",
    "params": {"start_date": "20250201", "end_date": "20250228"},
    "fields": "cal_date,is_open,pretrade_date"
  }'
```

## 返回示例

```json
{
  "code": 0,
  "msg": "",
  "data": {
    "fields": ["cal_date", "is_open", "pretrade_date"],
    "items": [
      ["20250228", 1, "20250227"],
      ["20250227", 1, "20250226"],
      ["20250226", 1, "20250225"]
    ]
  }
}
```
