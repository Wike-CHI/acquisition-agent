# 港股分钟行情

> 接口：`hk_mins` | 获取港股分钟级行情数据，支持1min/5min/15min/30min/60min多种频率

## 请求参数

| 参数 | 类型 | 必选 | 说明 |
|------|------|------|------|
| ts_code | str | Y | 股票代码，如00700.HK |
| freq | str | Y | 分钟频率：1min/5min/15min/30min/60min |
| start_date | datetime | N | 开始时间，格式：2025-02-28 09:00:00 |
| end_date | datetime | N | 结束时间，格式：2025-02-28 16:00:00 |

## 返回字段

| 字段 | 类型 | 说明 |
|------|------|------|
| ts_code | str | 股票代码 |
| trade_time | str | 交易时间 |
| open | float | 开盘价 |
| close | float | 收盘价 |
| high | float | 最高价 |
| low | float | 最低价 |
| vol | int | 成交量 |
| amount | float | 成交金额 |

## 调用示例

```bash
curl -X POST https://www.codebuddy.cn/v2/tool/financedata \
  -H "Content-Type: application/json" \
  -d '{
    "api_name": "hk_mins",
    "params": {"ts_code": "00700.HK", "freq": "5min", "start_date": "2025-02-28 09:00:00", "end_date": "2025-02-28 16:00:00"},
    "fields": "ts_code,trade_time,open,close,high,low,vol,amount"
  }'
```
