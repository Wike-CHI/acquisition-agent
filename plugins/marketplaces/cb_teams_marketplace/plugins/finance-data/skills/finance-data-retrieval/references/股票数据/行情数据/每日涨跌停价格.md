# 每日涨跌停价格

> 接口：`stk_limit` | 获取全市场每日涨跌停价格，含A/B股和基金

## 请求参数

| 参数 | 类型 | 必选 | 说明 |
|------|------|------|------|
| ts_code | str | N | 股票代码 |
| trade_date | str | N | 交易日期（YYYYMMDD） |
| start_date | str | N | 开始日期 |
| end_date | str | N | 结束日期 |

## 返回字段

| 字段 | 类型 | 说明 |
|------|------|------|
| trade_date | str | 交易日期 |
| ts_code | str | TS股票代码 |
| up_limit | float | 涨停价 |
| down_limit | float | 跌停价 |

## 调用示例

```bash
curl -X POST https://www.codebuddy.cn/v2/tool/financedata \
  -H "Content-Type: application/json" \
  -d '{
    "api_name": "stk_limit",
    "params": {"trade_date": "20250102"},
    "fields": ""
  }'
```

## 返回示例

```json
{
  "code": 0,
  "msg": "",
  "data": {
    "fields": ["trade_date", "ts_code", "up_limit", "down_limit"],
    "items": [
      ["20250102", "000001.SZ", 12.87, 10.53],
      ["20250102", "000002.SZ", 7.99, 6.53],
      ["20250102", "000004.SZ", 15.22, 12.46]
    ]
  }
}
```
