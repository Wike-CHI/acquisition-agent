# 每日结算参数

> 接口：`fut_settle` | 获取每日结算参数数据，包括交易和交割费率等

## 请求参数

| 参数 | 类型 | 必选 | 说明 |
|------|------|------|------|
| trade_date | str | N | 交易日期（trade_date/ts_code至少输入一个） |
| ts_code | str | N | 合约代码 |
| start_date | str | N | 开始日期（YYYYMMDD格式） |
| end_date | str | N | 结束日期 |
| exchange | str | N | 交易所代码 |

## 返回字段

| 字段 | 类型 | 说明 |
|------|------|------|
| ts_code | str | 合约代码 |
| trade_date | str | 交易日期 |
| settle | float | 结算价 |
| trading_fee_rate | float | 交易手续费率 |
| trading_fee | float | 交易手续费 |
| delivery_fee | float | 交割手续费 |
| b_hedging_margin_rate | float | 买套保交易保证金率 |
| s_hedging_margin_rate | float | 卖套保交易保证金率 |
| long_margin_rate | float | 买投机交易保证金率 |
| short_margin_rate | float | 卖投机交易保证金率 |
| offset_today_fee | float | 平今仓手续率 |
| exchange | str | 交易所 |

## 调用示例

```bash
curl -X POST https://www.codebuddy.cn/v2/tool/financedata \
  -H "Content-Type: application/json" \
  -d '{
    "api_name": "fut_settle",
    "params": {"ts_code": "CU2501.SHF"},
    "fields": "ts_code,trade_date,settle,trading_fee_rate,long_margin_rate,short_margin_rate"
  }'
```
