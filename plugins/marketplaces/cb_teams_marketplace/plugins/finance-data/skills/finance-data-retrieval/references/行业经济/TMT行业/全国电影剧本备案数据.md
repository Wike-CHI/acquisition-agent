# 全国电影剧本备案数据

> 接口：`film_record` | 获取全国电影剧本备案公示数据

## 请求参数

| 参数 | 类型 | 必选 | 说明 |
|------|------|------|------|
| ann_date | str | N | 公布日期（至少输入一个参数，格式：YYYYMMDD） |
| start_date | str | N | 开始日期 |
| end_date | str | N | 结束日期 |

**限量：** 单次最大500条，总量不限。**权限：** 用户需至少120积分。

## 返回字段

| 字段 | 类型 | 说明 |
|------|------|------|
| rec_no | str | 备案号 |
| film_name | str | 影片名称 |
| rec_org | str | 备案单位 |
| script_writer | str | 编剧 |
| rec_result | str | 备案结果 |
| rec_area | str | 备案地（备案时间） |
| classified | str | 影片分类 |
| date_range | str | 备案日期区间 |
| ann_date | str | 备案结果发布时间 |

## 调用示例

```bash
curl -X POST https://www.codebuddy.cn/v2/tool/financedata \
  -H "Content-Type: application/json" \
  -d '{
    "api_name": "film_record",
    "params": {"start_date": "20181014", "end_date": "20181214"},
    "fields": ""
  }'
```

## 返回示例

```json
{
  "code": 0,
  "msg": "",
  "data": {
    "fields": ["rec_no", "film_name", "rec_org", "script_writer", "rec_result", "rec_area", "classified", "date_range", "ann_date"],
    "items": [
      ["影剧备字[2018]第1234号", "示例电影", "某影视公司", "张三", "同意备案", "北京", "故事片", "2018.10-2018.12", "20181214"]
    ]
  }
}
```
