# 指数技术面因子（专业版）

> 接口：`idx_factor_pro` | 获取指数每日技术面因子数据，覆盖大盘指数、申万行业指数、中信指数全历史

## 请求参数

| 参数 | 类型 | 必选 | 说明 |
|------|------|------|------|
| ts_code | str | N | 指数代码（大盘/申万/中信指数） |
| trade_date | str | N | 交易日期 |
| start_date | str | N | 开始日期 |
| end_date | str | N | 结束日期 |

## 返回字段

### 基础行情

| 字段 | 类型 | 说明 |
|------|------|------|
| ts_code | str | 指数代码 |
| trade_date | str | 交易日期 |
| open | float | 开盘价 |
| high | float | 最高价 |
| low | float | 最低价 |
| close | float | 收盘价 |
| pre_close | float | 昨收价 |
| change | float | 涨跌额 |
| pct_change | float | 涨跌幅（未复权） |
| vol | float | 成交量（手） |
| amount | float | 成交额（千元） |

### 技术指标（60+）

- **趋势指标：** MACD、EMA、MA、EXPMA、TRIX等
- **动量指标：** RSI、KDJ、BRAR、CCI、ROC、MTM等
- **波动指标：** BOLL、ATR、KTN、TAQ、XSII等
- **其他指标：** OBV、MFI、PSY、EMV、VR、WR等
- **连续性指标：** updays（连涨天数）、downdays（连跌天数）

> 带 `_bfq` 后缀的为不复权数据

## 调用示例

```bash
curl -X POST https://www.codebuddy.cn/v2/tool/financedata \
  -H "Content-Type: application/json" \
  -d '{
    "api_name": "idx_factor_pro",
    "params": {"ts_code": "000001.SH", "trade_date": "20250228"},
    "fields": "ts_code,trade_date,open,high,low,close,pct_change,vol"
  }'
```

## 返回示例

```json
{
  "code": 0,
  "msg": "",
  "data": {

      ["000001.SH", "20250228", 3374.6636, 3383.1909, 3318.7162, 3320.8973, -1.9824, 568319504.0]
```
