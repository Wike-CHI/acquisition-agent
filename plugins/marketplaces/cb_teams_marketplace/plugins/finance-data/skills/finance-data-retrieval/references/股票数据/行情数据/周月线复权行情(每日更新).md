# 周月线复权行情（每日更新）

> 接口：`stk_week_month_adj` | 获取股票周/月线复权行情数据，每日更新

## 请求参数

| 参数 | 类型 | 必选 | 说明 |
|------|------|------|------|
| ts_code | str | N | 股票代码 |
| trade_date | str | N | 交易日期（YYYYMMDD，周/月最后一天） |
| start_date | str | N | 开始日期 |
| end_date | str | N | 结束日期 |
| freq | str | Y | 频度：week-周线，month-月线 |

## 返回字段

| 字段 | 类型 | 说明 |
|------|------|------|
| ts_code | str | 股票代码 |
| trade_date | str | 交易日期 |
| end_date | str | 计算截止日期 |
| freq | str | 频度（week/month） |
| open | float | 开盘价 |
| high | float | 最高价 |
| low | float | 最低价 |
| close | float | 收盘价 |
| pre_close | float | 上期收盘价（前复权价） |
| open_qfq | float | 前复权开盘价 |
| high_qfq | float | 前复权最高价 |
| low_qfq | float | 前复权最低价 |
| close_qfq | float | 前复权收盘价 |
| open_hfq | float | 后复权开盘价 |
| high_hfq | float | 后复权最高价 |
| low_hfq | float | 后复权最低价 |
| close_hfq | float | 后复权收盘价 |
| vol | float | 成交量 |
| amount | float | 成交额 |
| change | float | 涨跌额 |
| pct_chg | float | 涨跌幅 |

## 调用示例

```bash
curl -X POST https://www.codebuddy.cn/v2/tool/financedata \
  -H "Content-Type: application/json" \
  -d '{
    "api_name": "stk_week_month_adj",
    "params": {"ts_code": "000001.SZ", "freq": "week", "start_date": "20250101", "end_date": "20250131"},
    "fields": ""
  }'
```
