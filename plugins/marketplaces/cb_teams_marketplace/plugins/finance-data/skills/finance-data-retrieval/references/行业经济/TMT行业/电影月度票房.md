# 电影月度票房

> 接口：`bo_monthly` | 获取电影月度票房数据，数据从2008年1月起

## 请求参数

| 参数 | 类型 | 必选 | 说明 |
|------|------|------|------|
| date | str | Y | 日期（每月1日，格式：YYYYMMDD） |

**权限：** 用户需至少500积分

## 返回字段

| 字段 | 类型 | 说明 |
|------|------|------|
| date | str | 日期 |
| name | str | 影片名称 |
| list_date | str | 上映日期 |
| avg_price | float | 平均票价 |
| month_amount | float | 月度票房（万元） |
| list_day | int | 上映天数 |
| p_pc | int | 场均人次 |
| wom_index | float | 口碑指数 |
| m_ratio | float | 月度占比（%） |
| rank | int | 排名 |

## 调用示例

```bash
curl -X POST https://www.codebuddy.cn/v2/tool/financedata \
  -H "Content-Type: application/json" \
  -d '{
    "api_name": "bo_monthly",
    "params": {"date": "20180901"},
    "fields": ""
  }'
```
