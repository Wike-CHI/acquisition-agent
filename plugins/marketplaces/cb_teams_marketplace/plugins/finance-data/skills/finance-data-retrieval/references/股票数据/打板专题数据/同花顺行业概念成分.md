# 同花顺概念板块成分

> 接口：`ths_member` | 获取同花顺概念板块成分股列表

## 请求参数

| 参数 | 类型 | 必选 | 说明 |
|------|------|------|------|
| ts_code | str | N | 板块指数代码 |
| con_code | str | N | 股票代码 |

## 返回字段

| 字段 | 类型 | 说明 |
|------|------|------|
| ts_code | str | 指数代码 |
| con_code | str | 股票代码 |
| con_name | str | 股票名称 |
| weight | float | 权重（暂无数据） |
| in_date | str | 纳入日期（暂无数据） |
| out_date | str | 剔除日期（暂无数据） |
| is_new | str | 是否最新（Y=是，N=否） |

## 调用示例

```bash
curl -X POST https://www.codebuddy.cn/v2/tool/financedata \
  -H "Content-Type: application/json" \
  -d '{
    "api_name": "ths_member",
    "params": {"ts_code": "885800.TI"},
    "fields": "ts_code,con_code,con_name"
  }'
```

## 返回示例

```json
{
  "code": 0,
  "msg": "",
  "data": {
    "fields": ["ts_code", "con_code", "con_name"],
    "items": [
      ["885800.TI", "000012.SZ", "南玻A"],
      ["885800.TI", "000016.SZ", "深康佳A"],
      ["885800.TI", "000021.SZ", "深科技"]
    ]
  }
}
```
