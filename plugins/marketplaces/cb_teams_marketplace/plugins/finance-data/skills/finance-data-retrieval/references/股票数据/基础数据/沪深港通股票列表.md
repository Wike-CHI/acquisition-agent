# 沪深港通股票列表

> 接口：`stock_hsgt` | 获取沪深港通股票列表

## 请求参数

| 参数 | 类型 | 必选 | 说明 |
|------|------|------|------|
| ts_code | str | N | 股票代码 |
| trade_date | str | N | 交易日期（YYYYMMDD格式） |
| type | str | Y | 类型：HK_SZ-深股通 SZ_HK-港股通(深) HK_SH-沪股通 SH_HK-港股通(沪) |
| start_date | str | N | 开始日期 |
| end_date | str | N | 结束日期 |

## 返回字段

| 字段 | 类型 | 说明 |
|------|------|------|
| ts_code | str | 股票代码 |
| trade_date | str | 交易日期 |
| type | str | 类型 |
| name | str | 股票名称 |
| type_name | str | 类型名称 |

## 调用示例

```bash
curl -X POST https://www.codebuddy.cn/v2/tool/financedata \
  -H "Content-Type: application/json" \
  -d '{
    "api_name": "stock_hsgt",
    "params": {"trade_date": "20250813", "type": "HK_SZ"},
    "fields": ""
  }'
```

## 返回示例

```json
{
  "code": 0,
  "msg": "",
  "data": {
    "fields": ["ts_code", "trade_date", "type", "name", "type_name"],
    "items": [
      ["000016.SZ", "20250813", "HK_SZ", "深康佳Ａ", "深股通(港>深)"],
      ["000403.SZ", "20250813", "HK_SZ", "派林生物", "深股通(港>深)"],
      ["000401.SZ", "20250813", "HK_SZ", "冀东水泥", "深股通(港>深)"]
    ]
  }
}
```
