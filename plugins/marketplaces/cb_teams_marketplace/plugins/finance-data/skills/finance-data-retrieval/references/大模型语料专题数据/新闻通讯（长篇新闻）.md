# 新闻通讯（长篇新闻）

> 接口：`major_news` | 获取长篇新闻内容，覆盖主要财经新闻网站，提供超过8年历史数据

## 请求参数

| 参数 | 类型 | 必选 | 说明 |
|------|------|------|------|
| src | str | N | 新闻来源（新华网/凤凰财经/同花顺/新浪财经/华尔街见闻/中国证券/财新/第一财经/财联社） |
| start_date | str | N | 开始时间（如 2018-11-21 00:00:00） |
| end_date | str | N | 结束时间（如 2018-11-22 00:00:00） |

**限量：** 单次最大400条。**权限：** 需单独开通权限。

## 返回字段

| 字段 | 类型 | 说明 |
|------|------|------|
| title | str | 新闻标题 |
| content | str | 新闻内容（默认不显示，需在fields中指定） |
| pub_time | str | 发布时间 |
| src | str | 来源网站 |

## 调用示例

```bash
curl -X POST https://www.codebuddy.cn/v2/tool/financedata \
  -H "Content-Type: application/json" \
  -d '{
    "api_name": "major_news",
    "params": {"src": "新浪财经", "start_date": "2025-02-28 00:00:00", "end_date": "2025-02-28 12:00:00"},
    "fields": ""
  }'
```
