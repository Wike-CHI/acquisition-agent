# 美股复权因子

> 接口：`us_adjfactor` | 获取美股每日复权因子数据，每日美股收盘后滚动更新

## 请求参数

| 参数 | 类型 | 必选 | 说明 |
|------|------|------|------|
| ts_code | str | N | 股票代码 |
| trade_date | str | N | 交易日期（YYYYMMDD） |
| start_date | str | N | 开始日期 |
| end_date | str | N | 结束日期 |

## 返回字段

| 字段 | 类型 | 说明 |
|------|------|------|
| ts_code | str | 股票代码 |
| trade_date | str | 交易日期 |
| exchange | str | 交易所 |
| cum_adjfactor | float | 累计复权因子 |
| close_price | float | 收盘价 |

## 调用示例

```bash
curl -X POST https://www.codebuddy.cn/v2/tool/financedata \
  -H "Content-Type: application/json" \
  -d '{
    "api_name": "us_adjfactor",
    "params": {"ts_code": "AAPL", "start_date": "20250201", "end_date": "20250228"},
    "fields": "ts_code,trade_date,exchange,cum_adjfactor,close_price"
  }'
```

## 返回示例

```json
{
  "code": 0,
  "msg": "",
  "data": {
    "fields": ["ts_code", "trade_date", "exchange", "cum_adjfactor", "close_price"],
    "items": [
      ["AAPL", "20250228", "NAS", 1.0, 241.84],
      ["AAPL", "20250227", "NAS", 1.0, 237.3],
      ["AAPL", "20250226", "NAS", 1.0, 240.36]
    ]
  }
}
```
