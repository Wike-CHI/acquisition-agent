# 分红送股数据

> 接口：`dividend` | 获取上市公司分红送股数据

## 请求参数

| 参数 | 类型 | 必选 | 说明 |
|------|------|------|------|
| ts_code | str | N | TS股票代码 |
| ann_date | str | N | 公告日期 |
| record_date | str | N | 股权登记日 |
| ex_date | str | N | 除权除息日 |
| imp_ann_date | str | N | 实施公告日 |

> 以上参数至少需要输入一个

## 返回字段

| 字段 | 类型 | 说明 |
|------|------|------|
| ts_code | str | TS代码 |
| end_date | str | 分红年度 |
| ann_date | str | 预案公告日 |
| div_proc | str | 实施进度 |
| stk_div | float | 每股送转 |
| stk_bo_rate | float | 每股送股比例 |
| stk_co_rate | float | 每股转增比例 |
| cash_div | float | 每股分红（税后，元） |
| cash_div_tax | float | 每股分红（税前，元） |
| record_date | str | 股权登记日 |
| ex_date | str | 除权除息日 |
| pay_date | str | 派息日 |
| div_listdate | str | 红股上市日 |
| imp_ann_date | str | 实施公告日 |

## 调用示例

```bash
curl -X POST https://www.codebuddy.cn/v2/tool/financedata \
  -H "Content-Type: application/json" \
  -d '{
    "api_name": "dividend",
    "params": {"ts_code": "000001.SZ"},
    "fields": ""
  }'
```
