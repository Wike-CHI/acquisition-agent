# 港股列表

> 接口：`hk_basic` | 获取港股基本信息数据

## 请求参数

| 参数 | 类型 | 必选 | 说明 |
|------|------|------|------|
| ts_code | str | N | TS代码 |
| list_status | str | N | 上市状态：L-上市 D-退市 P-暂停上市（默认L） |

## 返回字段

| 字段 | 类型 | 说明 |
|------|------|------|
| ts_code | str | TS代码 |
| name | str | 股票简称 |
| fullname | str | 公司全称 |
| enname | str | 英文名称 |
| cn_spell | str | 拼音 |
| market | str | 市场类别 |
| list_status | str | 上市状态 |
| list_date | str | 上市日期 |
| delist_date | str | 退市日期 |
| trade_unit | float | 交易单位 |
| isin | str | ISIN代码 |
| curr_type | str | 货币代码 |

## 调用示例

```bash
curl -X POST https://www.codebuddy.cn/v2/tool/financedata \
  -H "Content-Type: application/json" \
  -d '{
    "api_name": "hk_basic",
    "params": {"ts_code": "00700.HK"},
    "fields": "ts_code,name,fullname,market,list_status,list_date,curr_type"
  }'
```
