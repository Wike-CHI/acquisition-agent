# 股票回购

> 接口：`repurchase` | 获取上市公司股票回购数据

## 请求参数

| 参数 | 类型 | 必选 | 说明 |
|------|------|------|------|
| ann_date | str | N | 公告日期 |
| start_date | str | N | 公告开始日期 |
| end_date | str | N | 公告结束日期 |

## 返回字段

| 字段 | 类型 | 说明 |
|------|------|------|
| ts_code | str | TS代码 |
| ann_date | str | 公告日期 |
| end_date | str | 截止日期 |
| proc | str | 进度状态 |
| exp_date | str | 过期日期 |
| vol | float | 回购数量 |
| amount | float | 回购金额 |
| high_limit | float | 回购最高价 |
| low_limit | float | 回购最低价 |

## 调用示例

```bash
curl -X POST https://www.codebuddy.cn/v2/tool/financedata \
  -H "Content-Type: application/json" \
  -d '{
    "api_name": "repurchase",
    "params": {"ann_date": "20241220"},
    "fields": ""
  }'
```
