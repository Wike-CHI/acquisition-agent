# 券商金股数据

> 接口：`broker_recommend` | 获取券商月度金股数据，一般每月1日~3日内更新当月数据

## 请求参数

| 参数 | 类型 | 必选 | 说明 |
|------|------|------|------|
| month | str | Y | 月度（YYYYMM格式） |

## 返回字段

| 字段 | 类型 | 说明 |
|------|------|------|
| month | str | 月度 |
| broker | str | 券商 |
| ts_code | str | 股票代码 |
| name | str | 股票简称 |

## 调用示例

```bash
curl -X POST https://www.codebuddy.cn/v2/tool/financedata \
  -H "Content-Type: application/json" \
  -d '{
    "api_name": "broker_recommend",
    "params": {"month": "202502"},
    "fields": "month,broker,ts_code,name"
  }'
```

## 返回示例

```json
{
  "code": 0,
  "msg": "",
  "data": {
    "fields": ["month", "broker", "ts_code", "name"],
    "items": [
      ["202502", "东兴证券", "000988.SZ", "华工科技"],
      ["202502", "东兴证券", "002387.SZ", "维信诺"],
      ["202502", "东兴证券", "002847.SZ", "盐津铺子"]
    ]
  }
}
```
