# 日线行情

> 接口：`fut_daily` | 获取期货日线行情数据

## 请求参数

| 参数 | 类型 | 必选 | 说明 |
|------|------|------|------|
| trade_date | str | N | 交易日期（YYYYMMDD格式） |
| ts_code | str | N | 合约代码 |
| exchange | str | N | 交易所代码 |
| start_date | str | N | 开始日期 |
| end_date | str | N | 结束日期 |

## 返回字段

| 字段 | 类型 | 说明 |
|------|------|------|
| ts_code | str | 合约代码 |
| trade_date | str | 交易日期 |
| pre_close | float | 昨收盘价 |
| pre_settle | float | 昨结算价 |
| open | float | 开盘价 |
| high | float | 最高价 |
| low | float | 最低价 |
| close | float | 收盘价 |
| settle | float | 结算价 |
| change1 | float | 涨跌1（收盘价-昨结算价） |
| change2 | float | 涨跌2（结算价-昨结算价） |
| vol | float | 成交量（手） |
| amount | float | 成交金额（万元） |
| oi | float | 持仓量（手） |
| oi_chg | float | 持仓变化 |
| delv_settle | float | 交割结算价 |

## 调用示例

```bash
curl -X POST https://www.codebuddy.cn/v2/tool/financedata \
  -H "Content-Type: application/json" \
  -d '{
    "api_name": "fut_daily",
    "params": {"ts_code": "CU2501.SHF", "start_date": "20250101", "end_date": "20250115"},
    "fields": "ts_code,trade_date,open,high,low,close,settle,vol,amount,oi"
  }'
```
