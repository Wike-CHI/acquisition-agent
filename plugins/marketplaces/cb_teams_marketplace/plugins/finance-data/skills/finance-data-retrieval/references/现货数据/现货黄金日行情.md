# 现货黄金日行情

> 接口：`sge_daily` | 获取上海黄金交易所现货合约日线行情数据

## 数据说明

数据由当日9:00-15:30交易及前一日20:00-2:30夜盘交易组成，成交量和成交金额按双边计算。

## 请求参数

| 参数 | 类型 | 必选 | 说明 |
|------|------|------|------|
| ts_code | str | N | 合约代码（可通过基础信息接口获取） |
| trade_date | str | N | 交易日期 |
| start_date | str | N | 开始日期 |
| end_date | str | N | 结束日期 |

## 返回字段

| 字段 | 类型 | 说明 |
|------|------|------|
| ts_code | str | 现货合约代码 |
| trade_date | str | 交易日期 |
| close | float | 收盘价（元/克） |
| open | float | 开盘价（元/克） |
| high | float | 最高价（元/克） |
| low | float | 最低价（元/克） |
| price_avg | float | 加权平均价（元/克） |
| change | float | 涨跌（元/克） |
| pct_change | float | 涨跌幅 |
| vol | float | 成交量（千克） |
| amount | float | 成交额（元） |
| oi | float | 市场持仓量 |
| settle_vol | float | 交收量 |
| settle_dire | str | 持仓方向 |

## 调用示例

```bash
curl -X POST https://www.codebuddy.cn/v2/tool/financedata \
  -H "Content-Type: application/json" \
  -d '{
    "api_name": "sge_daily",
    "params": {"trade_date": "20250110"},
    "fields": "ts_code,trade_date,close,open,high,low,vol,amount"
  }'
```
