# 债券大宗交易

> 接口：`bond_blk` | 获取沪深交易所债券大宗交易数据

## 请求参数

| 参数 | 类型 | 必选 | 说明 |
|------|------|------|------|
| ts_code | str | N | 债券代码 |
| trade_date | str | N | 交易日期（YYYYMMDD） |
| start_date | str | N | 开始日期 |
| end_date | str | N | 结束日期 |

## 返回字段

| 字段 | 类型 | 说明 |
|------|------|------|
| trade_date | str | 交易日期 |
| ts_code | str | 债券代码 |
| name | str | 债券名称 |
| price | float | 成交价（元） |
| vol | float | 累计成交量（万股/万份/万张/万手） |
| amount | float | 累计成交金额（万元） |

## 调用示例

```bash
curl -X POST https://www.codebuddy.cn/v2/tool/financedata \
  -H "Content-Type: application/json" \
  -d '{
    "api_name": "bond_blk",
    "params": {"trade_date": "20250228"},
    "fields": "trade_date,ts_code,name,price,vol,amount"
  }'
```

## 返回示例

```json
{
  "code": 0,
  "msg": "",
  "data": {
    "fields": ["trade_date", "ts_code", "name", "price", "vol", "amount"],
    "items": [
      ["20250228", "127022.SZ", "恒逸转债", 89.55, 1.12, 100.3],

      ["20250228", "127106.SZ", "伟隆转债", 119.48, 11.32, 1352.48]
    ]
  }
}
```
