# 通达信板块成分

> 接口：`tdx_member` | 获取通达信各板块成分股信息

## 请求参数

| 参数 | 类型 | 必选 | 说明 |
|------|------|------|------|
| ts_code | str | N | 板块代码（xxxxxx.TDX） |
| trade_date | str | N | 交易日期（YYYYMMDD格式） |

## 返回字段

| 字段 | 类型 | 说明 |
|------|------|------|
| ts_code | str | 板块代码 |
| trade_date | str | 交易日期 |
| con_code | str | 成分股票代码 |
| con_name | str | 成分股票名称 |

## 调用示例

```bash
curl -X POST https://www.codebuddy.cn/v2/tool/financedata \
  -H "Content-Type: application/json" \
  -d '{
    "api_name": "tdx_member",
    "params": {"trade_date": "20250513", "ts_code": "880728.TDX"},
    "fields": "ts_code,trade_date,con_code,con_name"
  }'
```

## 返回示例

```json
{
  "code": 0,
  "msg": "",
  "data": {
    "fields": ["ts_code", "trade_date", "con_code", "con_name"],
    "items": [
      ["880728.TDX", "20250513", "000039.SZ", "中集集团"],
      ["880728.TDX", "20250513", "000088.SZ", "盐田港"],
      ["880728.TDX", "20250513", "000507.SZ", "珠海港"]
    ]
  }
}
```
