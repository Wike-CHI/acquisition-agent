# 黄金现货基础信息

> 接口：`sge_basic` | 获取上海黄金交易所现货合约基础信息

## 请求参数

| 参数 | 类型 | 必选 | 说明 |
|------|------|------|------|
| ts_code | str | N | 合约代码（支持多个逗号分隔，不传获取全部） |

## 返回字段

| 字段 | 类型 | 说明 |
|------|------|------|
| ts_code | str | 品种代码 |
| ts_name | str | 品种名称 |
| trade_type | str | 交易类型 |
| t_unit | float | 交易单位（克/手） |
| p_unit | float | 报价单位 |
| min_change | float | 最小变动价位 |
| price_limit | float | 每日最大波幅限制 |
| min_vol | int | 最小单笔报价量（手） |
| max_vol | int | 最大单笔报价量（手） |
| trade_mode | str | 交易方式 |
| margin_rate | float | 保证金比例 |
| liq_rate | float | 违约金比例（%） |
| trade_time | str | 交易时间 |
| list_date | str | 上市日期 |

## 调用示例

```bash
curl -X POST https://www.codebuddy.cn/v2/tool/financedata \
  -H "Content-Type: application/json" \
  -d '{
    "api_name": "sge_basic",
    "params": {},
    "fields": "ts_code,ts_name,trade_type,t_unit,min_vol,max_vol,trade_time"
  }'
```
