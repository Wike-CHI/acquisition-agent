# 电影日度票房

> 接口：`bo_daily` | 获取电影日度票房数据，数据从2018年9月起

## 请求参数

| 参数 | 类型 | 必选 | 说明 |
|------|------|------|------|
| date | str | Y | 日期（格式：YYYYMMDD） |

**权限：** 用户需至少500积分

## 返回字段

| 字段 | 类型 | 说明 |
|------|------|------|
| date | str | 日期 |
| name | str | 影片名称 |
| avg_price | float | 平均票价 |
| day_amount | float | 当日票房（万元） |
| total | float | 累计票房（万元） |
| list_day | int | 上映天数 |
| p_pc | int | 场均人次 |
| wom_index | float | 口碑指数 |
| up_ratio | float | 环比变化（%） |
| rank | int | 排名 |

## 调用示例

```bash
curl -X POST https://www.codebuddy.cn/v2/tool/financedata \
  -H "Content-Type: application/json" \
  -d '{
    "api_name": "bo_daily",
    "params": {"date": "20181014"},
    "fields": ""
  }'
```
