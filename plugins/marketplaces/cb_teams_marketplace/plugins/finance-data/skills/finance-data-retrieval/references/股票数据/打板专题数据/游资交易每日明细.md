# 游资每日明细

> 接口：`hm_detail` | 获取游资每日交易明细数据，数据从2022年8月开始提供

## 请求参数

| 参数 | 类型 | 必选 | 说明 |
|------|------|------|------|
| trade_date | str | N | 交易日期（YYYYMMDD格式） |
| ts_code | str | N | 股票代码 |
| hm_name | str | N | 游资名称 |
| start_date | str | N | 开始日期 |
| end_date | str | N | 结束日期 |

## 返回字段

| 字段 | 类型 | 说明 |
|------|------|------|
| trade_date | str | 交易日期 |
| ts_code | str | 股票代码 |
| ts_name | str | 股票名称 |
| buy_amount | float | 买入金额（元） |
| sell_amount | float | 卖出金额（元） |
| net_amount | float | 净买卖金额（元） |
| hm_name | str | 游资名称 |
| hm_orgs | str | 关联机构 |
| tag | str | 标签 |

## 调用示例

```bash
curl -X POST https://www.codebuddy.cn/v2/tool/financedata \
  -H "Content-Type: application/json" \
  -d '{
    "api_name": "hm_detail",
    "params": {"trade_date": "20250228"},
    "fields": "trade_date,ts_code,ts_name,buy_amount,sell_amount,net_amount,hm_name"
  }'
```
