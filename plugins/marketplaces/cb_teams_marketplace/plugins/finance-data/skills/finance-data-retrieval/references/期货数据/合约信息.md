# 合约信息

> 接口：`fut_basic` | 获取期货合约列表数据

## 请求参数

| 参数 | 类型 | 必选 | 说明 |
|------|------|------|------|
| exchange | str | Y | 交易所代码：CFFEX/DCE/CZCE/SHFE/INE/GFEX |
| fut_type | str | N | 合约类型：1-普通合约 2-主力与连续合约（默认全部） |
| fut_code | str | N | 标准合约代码（如AG、AP等） |
| list_date | str | N | 上市开始日期（YYYYMMDD） |

## 返回字段

| 字段 | 类型 | 说明 |
|------|------|------|
| ts_code | str | 合约代码 |
| symbol | str | 交易标识 |
| exchange | str | 交易市场 |
| name | str | 中文简称 |
| fut_code | str | 合约产品代码 |
| multiplier | float | 合约乘数 |
| trade_unit | str | 交易计量单位 |
| per_unit | float | 交易单位（每手） |
| quote_unit | str | 报价单位 |
| quote_unit_desc | str | 最小报价单位说明 |
| d_mode_desc | str | 交割方式说明 |
| list_date | str | 上市日期 |
| delist_date | str | 最后交易日期 |
| d_month | str | 交割月份 |
| last_ddate | str | 最后交割日 |
| trade_time_desc | str | 交易时间说明 |

## 调用示例

```bash
curl -X POST https://www.codebuddy.cn/v2/tool/financedata \
  -H "Content-Type: application/json" \
  -d '{
    "api_name": "fut_basic",
    "params": {"exchange": "SHFE"},
    "fields": "ts_code,symbol,name,list_date,delist_date,per_unit,quote_unit"
  }'
```
