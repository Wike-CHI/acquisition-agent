# 资产负债表

> 接口：`balancesheet` | 获取上市公司资产负债表数据

## 请求参数

| 参数 | 类型 | 必选 | 说明 |
|------|------|------|------|
| ts_code | str | Y | TS股票代码 |
| ann_date | str | N | 公告日期 |
| start_date | str | N | 公告开始日期 |
| end_date | str | N | 公告结束日期 |
| period | str | N | 报告期（每个季度最后一天，如20240930） |
| report_type | str | N | 报告类型（1=合并报表，2=单季合并，6=母公司报表等） |
| comp_type | str | N | 公司类型（1=一般工商业，2=银行，3=保险，4=证券） |

## 返回字段

| 字段 | 类型 | 说明 |
|------|------|------|
| ts_code | str | TS代码 |
| ann_date | str | 公告日期 |
| f_ann_date | str | 实际公告日期 |
| end_date | str | 报告期 |
| report_type | str | 报告类型 |
| comp_type | str | 公司类型 |
| end_type | str | 报告期类型 |
| total_share | float | 期末总股本 |
| cap_rese | float | 资本公积金 |
| undistr_porfit | float | 未分配利润 |
| surplus_rese | float | 盈余公积金 |
| money_cap | float | 货币资金 |
| total_cur_assets | float | 流动资产合计 |
| total_nca | float | 非流动资产合计 |
| total_assets | float | 资产总计 |
| total_cur_liab | float | 流动负债合计 |
| total_ncl | float | 非流动负债合计 |
| total_liab | float | 负债合计 |
| total_hldr_eqy_exc_min_int | float | 股东权益合计（不含少数股东权益） |
| total_hldr_eqy_inc_min_int | float | 股东权益合计（含少数股东权益） |
| total_liab_hldr_eqy | float | 负债及股东权益总计 |

> 更多字段包括应收账款、存货、固定资产、应付债券等详细科目

## 调用示例

```bash
curl -X POST https://www.codebuddy.cn/v2/tool/financedata \
  -H "Content-Type: application/json" \
  -d '{
    "api_name": "balancesheet",
    "params": {"ts_code": "000001.SZ", "period": "20240930"},
    "fields": ""
  }'
```
