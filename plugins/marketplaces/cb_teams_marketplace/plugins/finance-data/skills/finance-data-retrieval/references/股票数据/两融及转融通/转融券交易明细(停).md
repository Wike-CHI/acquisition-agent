# 转融券交易明细（已停）

> 接口：`slb_sec_detail` | 转融券交易明细（该接口已停止更新）

## 请求参数

| 参数 | 类型 | 必选 | 说明 |
|------|------|------|------|
| trade_date | str | N | 交易日期（YYYYMMDD格式） |
| ts_code | str | N | 股票代码 |
| start_date | str | N | 开始日期 |
| end_date | str | N | 结束日期 |

## 返回字段

| 字段 | 类型 | 说明 |
|------|------|------|
| trade_date | str | 交易日期 |
| ts_code | str | 股票代码 |
| name | str | 股票名称 |
| tenor | str | 期限（天） |
| fee_rate | float | 费率（%） |
| lent_qnt | float | 融出数量（万股） |

## 调用示例

```bash
curl -X POST https://www.codebuddy.cn/v2/tool/financedata \
  -H "Content-Type: application/json" \
  -d '{
    "api_name": "slb_sec_detail",
    "params": {"trade_date": "20240620"},
    "fields": "trade_date,ts_code,name,tenor,fee_rate,lent_qnt"
  }'
```
