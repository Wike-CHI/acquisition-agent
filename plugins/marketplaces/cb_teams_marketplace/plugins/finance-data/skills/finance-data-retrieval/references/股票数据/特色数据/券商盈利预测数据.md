# 券商盈利预测数据

> 接口：`report_rc` | 获取券商研报每日盈利预测数据。数据自2010年起，每日19:00-22:00更新

## 请求参数

| 参数 | 类型 | 必选 | 说明 |
|------|------|------|------|
| ts_code | str | N | 股票代码 |
| report_date | str | N | 报告日期 |
| start_date | str | N | 报告开始日期 |
| end_date | str | N | 报告结束日期 |

## 返回字段

| 字段 | 类型 | 说明 |
|------|------|------|
| ts_code | str | 股票代码 |
| name | str | 股票名称 |
| report_date | str | 报告日期 |
| report_title | str | 报告标题 |
| report_type | str | 报告类型 |
| classify | str | 报告分类 |
| org_name | str | 机构名称 |
| author_name | str | 作者 |
| quarter | str | 预测季度 |
| op_rt | float | 预测营业收入（万元） |
| op_pr | float | 预测营业利润（万元） |
| tp | float | 预测利润总额（万元） |
| np | float | 预测净利润（万元） |
| eps | float | 预测每股收益（元） |
| pe | float | 预测市盈率 |
| rd | float | 预测股息率 |
| roe | float | 预测ROE |
| ev_ebitda | float | 预测EV/EBITDA |
| rating | str | 券商评级 |
| max_price | float | 预测最高目标价 |
| min_price | float | 预测最低目标价 |

## 调用示例

```bash
curl -X POST https://www.codebuddy.cn/v2/tool/financedata \
  -H "Content-Type: application/json" \
  -d '{
    "api_name": "report_rc",
    "params": {"report_date": "20250228"},
    "fields": "ts_code,name,report_date,classify,org_name,quarter,eps,pe,rating"
  }'
```
