# 东方财富板块成分

> 接口：`dc_member` | 获取东方财富板块每日成分数据，可根据概念板块代码和交易日期获取历史成分

## 请求参数

| 参数 | 类型 | 必选 | 说明 |
|------|------|------|------|
| ts_code | str | N | 板块指数代码 |
| con_code | str | N | 成分股票代码 |
| trade_date | str | N | 交易日期（YYYYMMDD格式） |

## 返回字段

| 字段 | 类型 | 说明 |
|------|------|------|
| trade_date | str | 交易日期 |
| ts_code | str | 概念代码 |
| con_code | str | 成分代码 |
| name | str | 成分股名称 |

## 调用示例

```bash
curl -X POST https://www.codebuddy.cn/v2/tool/financedata \
  -H "Content-Type: application/json" \
  -d '{
    "api_name": "dc_member",
    "params": {"trade_date": "20250228"},
    "fields": "trade_date,ts_code,con_code,name"
  }'
```

## 返回示例

```json
{
  "code": 0,
  "msg": "",
  "data": {
    "fields": ["trade_date", "ts_code", "con_code", "name"],
    "items": [
      ["20250228", "BK0490.DC", "301041.SZ", "金百泽"],
      ["20250228", "BK0490.DC", "300447.SZ", "全信股份"],
      ["20250228", "BK0490.DC", "300875.SZ", "捷强装备"]
    ]
  }
}
```
