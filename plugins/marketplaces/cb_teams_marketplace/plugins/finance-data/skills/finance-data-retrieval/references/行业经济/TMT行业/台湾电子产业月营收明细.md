# 台湾电子产业月营收明细

> 接口：`tmt_twincomedetail` | 获取台湾TMT产业上市公司各产品类别月度营收明细数据

## 请求参数

| 参数 | 类型 | 必选 | 说明 |
|------|------|------|------|
| date | str | N | 报告期 |
| item | str | N | 产品代码 |
| symbol | str | N | 公司代码 |
| start_date | str | N | 报告期开始日期 |
| end_date | str | N | 报告期结束日期 |
| source | str | N | 数据来源 |

## 返回字段

| 字段 | 类型 | 说明 |
|------|------|------|
| date | str | 报告期 |
| item | str | 产品代码 |
| symbol | str | 公司代码 |
| op_income | str | 月营收 |
| consop_income | str | 合并月营收（默认不显示） |

## 调用示例

```bash
curl -X POST https://www.codebuddy.cn/v2/tool/financedata \
  -H "Content-Type: application/json" \
  -d '{
    "api_name": "tmt_twincomedetail",
    "params": {"item": "8", "symbol": "6156"},
    "fields": ""
  }'
```

## 返回示例

```json
{
  "code": 0,
  "msg": "",
  "data": {
    "fields": ["date", "item", "symbol", "op_income"],
    "items": [
      ["20180731", "8", "6156", 429618.0],
      ["20180629", "8", "6156", 367786.0],
      ["20180531", "8", "6156", 415715.0]
    ]
  }
}
```
