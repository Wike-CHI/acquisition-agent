# 利润表

> 接口：`income` | 获取上市公司利润表数据

## 请求参数

| 参数 | 类型 | 必选 | 说明 |
|------|------|------|------|
| ts_code | str | Y | TS股票代码 |
| ann_date | str | N | 公告日期 |
| f_ann_date | str | N | 实际公告日期 |
| start_date | str | N | 公告开始日期 |
| end_date | str | N | 公告结束日期 |
| period | str | N | 报告期（每个季度最后一天，如20240930） |
| report_type | str | N | 报告类型（见下方说明） |
| comp_type | str | N | 公司类型（1=一般工商业，2=银行，3=保险，4=证券） |

### 报告类型说明

| 代码 | 说明 |
|------|------|
| 1 | 合并报表（默认） |
| 2 | 单季合并 |
| 3 | 调整单季合并 |
| 4 | 调整合并（上年同期） |
| 5 | 调整前合并 |
| 6 | 母公司报表 |
| 7 | 母公司单季 |
| 8 | 母公司调整单季 |
| 9 | 母公司调整报表 |
| 10 | 母公司调整前报表 |
| 11 | 母公司调整前合并 |
| 12 | 母公司调整前报表 |

| fv_value_chg_gain | float | 公允价值变动净收益 |

| total_cogs | float | 营业总成本 |
| oper_cost | float | 营业成本 |
| int_exp | float | 利息支出 |
| sell_exp | float | 销售费用 |

| ebitda | float | 息税折旧摊销前利润 |
| rd_exp | float | 研发费用 |

## 调用示例

```bash
curl -X POST https://www.codebuddy.cn/v2/tool/financedata \
  -H "Content-Type: application/json" \
  -d '{
    "api_name": "income",
    "params": {"ts_code": "000001.SZ", "period": "20240930"},
    "fields": ""
  }'
```
