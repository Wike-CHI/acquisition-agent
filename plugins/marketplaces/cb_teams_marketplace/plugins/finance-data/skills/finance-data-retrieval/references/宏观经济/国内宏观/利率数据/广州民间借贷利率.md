# 广州民间借贷利率

> 接口：`gz_index` | 广州民间借贷利率数据

## 请求参数

| 参数 | 类型 | 必选 | 说明 |
|------|------|------|------|
| date | str | N | 日期 |
| start_date | str | N | 开始日期 |
| end_date | str | N | 结束日期 |

**限量：** 不限量，一次可取全部历史数据。**权限：** 用户需至少2000积分。

## 返回字段

| 字段 | 类型 | 说明 |
|------|------|------|
| date | str | 日期 |
| d10_rate | float | 十天期利率（%） |
| m1_rate | float | 一月期利率 |
| m3_rate | float | 三月期利率 |
| m6_rate | float | 六月期利率 |
| m12_rate | float | 一年期利率 |
| long_rate | float | 长期利率 |

## 调用示例

```bash
curl -X POST https://www.codebuddy.cn/v2/tool/financedata \
  -H "Content-Type: application/json" \
  -d '{
    "api_name": "gz_index",
    "params": {"start_date": "20180101", "end_date": "20181231"},
    "fields": ""
  }'
```

## 返回示例

```json
{
  "code": 0,
  "msg": "",
  "data": {

    "items": [

      ["20180410", 12.00, 19.20, 19.20, 13.35, 14.40, 12.00]

  }
}
```
