# 主营业务构成

> 接口：`fina_mainbz` | 获取上市公司主营业务构成，按产品、地区、行业分类

## 请求参数

| 参数 | 类型 | 必选 | 说明 |
|------|------|------|------|
| ts_code | str | Y | TS股票代码 |
| period | str | N | 报告期（每个季度最后一天，如20240930） |
| type | str | N | 类型：P=按产品，D=按地区，I=按行业 |
| start_date | str | N | 报告期开始日期 |
| end_date | str | N | 报告期结束日期 |

## 返回字段

| 字段 | 类型 | 说明 |
|------|------|------|
| ts_code | str | TS代码 |
| end_date | str | 报告期 |
| bz_item | str | 主营业务来源 |
| bz_code | str | 类型代码 |
| bz_sales | float | 主营业务收入（元） |
| bz_profit | float | 主营业务利润（元） |
| bz_cost | float | 主营业务成本（元） |
| curr_type | str | 货币代码 |

## 调用示例

```bash
curl -X POST https://www.codebuddy.cn/v2/tool/financedata \
  -H "Content-Type: application/json" \
  -d '{
    "api_name": "fina_mainbz",
    "params": {"ts_code": "000001.SZ", "type": "P"},
    "fields": ""
  }'
```
