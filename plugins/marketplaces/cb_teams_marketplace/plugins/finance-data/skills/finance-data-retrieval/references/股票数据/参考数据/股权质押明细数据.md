# 股权质押明细数据

> 接口：`pledge_detail` | 获取上市公司股权质押明细数据

## 请求参数

| 参数 | 类型 | 必选 | 说明 |
|------|------|------|------|
| ts_code | str | Y | TS股票代码 |

## 返回字段

| 字段 | 类型 | 说明 |
|------|------|------|
| ts_code | str | TS代码 |
| ann_date | str | 公告日期 |
| holder_name | str | 股东名称 |
| pledge_amount | float | 质押数量（万股） |
| start_date | str | 质押开始日期 |
| end_date | str | 质押结束日期 |
| is_release | str | 是否已解押（Y/N） |
| release_date | str | 解押日期 |
| pledgor | str | 质押方 |
| holding_amount | float | 总持股量（万股） |
| pledged_amount | float | 总质押量（万股） |
| p_total_ratio | float | 本次质押占总股本比例 |
| h_total_ratio | float | 总持股占总股本比例 |
| is_buyback | str | 是否回购（0=否，1=是） |

## 调用示例

```bash
curl -X POST https://www.codebuddy.cn/v2/tool/financedata \
  -H "Content-Type: application/json" \
  -d '{
    "api_name": "pledge_detail",
    "params": {"ts_code": "000014.SZ"},
    "fields": ""
  }'
```

## 返回示例

```json
{
  "code": 0,
  "msg": "",
  "data": {
    "fields": ["ts_code", "ann_date", "holder_name", "pledge_amount", "start_date", "end_date", "is_release", "release_date", "pledgor", "holding_amount", "pledged_amount", "p_total_ratio", "h_total_ratio", "is_buyback"],
    "items": []
  }
}
```

> 注：部分股票可能无质押记录，返回空数据
