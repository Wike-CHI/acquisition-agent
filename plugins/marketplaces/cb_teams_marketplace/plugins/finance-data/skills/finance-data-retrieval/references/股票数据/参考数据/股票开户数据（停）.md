# 股票开户数据（停）

> 接口：`stk_account` | 获取股票开户数据（按周统计），官方已停止更新

## 请求参数

| 参数 | 类型 | 必选 | 说明 |
|------|------|------|------|
| date | str | N | 日期 |
| start_date | str | N | 开始日期 |
| end_date | str | N | 结束日期 |

## 返回字段

| 字段 | 类型 | 说明 |
|------|------|------|
| date | str | 统计周期 |
| weekly_new | float | 本周新开户数（万户） |
| total | float | 期末总账户数（万户） |
| weekly_hold | float | 本周持仓账户数（万户） |
| weekly_trade | float | 本周参与交易账户数（万户） |

> 注：自2017年2月10日起，持仓和交易账户数不再公布

## 调用示例

```bash
curl -X POST https://www.codebuddy.cn/v2/tool/financedata \
  -H "Content-Type: application/json" \
  -d '{
    "api_name": "stk_account",
    "params": {"start_date": "20180101", "end_date": "20180301"},
    "fields": ""
  }'
```

## 返回示例

```json
{
  "code": 0,
  "msg": "",
  "data": {
    "fields": ["date", "weekly_new", "total", "weekly_hold", "weekly_trade"],
    "items": [
      ["20180223", 8.35, 13575.5, null, null],
      ["20180214", 11.41, 13567.15, null, null],
      ["20180209", 25.02, 13555.74, null, null]
    ]
  }
}
```
