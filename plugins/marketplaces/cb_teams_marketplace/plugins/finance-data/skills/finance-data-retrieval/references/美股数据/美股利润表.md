# 美股利润表

> 接口：`us_income` | 获取美股上市公司利润表数据（目前覆盖主要美股和中概股）

## 请求参数

| 参数 | 类型 | 必选 | 说明 |
|------|------|------|------|
| ts_code | str | Y | 股票代码 |
| period | str | N | 报告期（YYYYMMDD，每季最后一天，如20241231） |
| ind_name | str | N | 指标名称（如：营业额） |
| report_type | str | N | 报告类型：Q1-一季报 Q2-半年报 Q3-三季报 Q4-年报 |
| start_date | str | N | 报告期开始日期 |
| end_date | str | N | 报告期结束日期 |

## 返回字段

| 字段 | 类型 | 说明 |
|------|------|------|
| ts_code | str | 股票代码 |
| end_date | str | 报告期 |
| ind_type | str | 报告期类型（Q1/Q2/Q3/Q4） |
| name | str | 股票名称 |
| ind_name | str | 财务科目名称 |
| ind_value | float | 财务科目值 |
| report_type | str | 报告类型 |

## 调用示例

```bash
curl -X POST https://www.codebuddy.cn/v2/tool/financedata \
  -H "Content-Type: application/json" \
  -d '{
    "api_name": "us_income",
    "params": {"ts_code": "NVDA", "period": "20250427"},
    "fields": "ts_code,end_date,ind_type,name,ind_name,ind_value,report_type"
  }'
```
