# 期货主力与连续合约

> 接口：`fut_mapping` | 获取期货主力（或连续）合约与月合约映射数据

## 请求参数

| 参数 | 类型 | 必选 | 说明 |
|------|------|------|------|
| ts_code | str | N | 合约代码 |
| trade_date | str | N | 交易日期（YYYYMMDD格式） |
| start_date | str | N | 开始日期 |
| end_date | str | N | 结束日期 |

## 返回字段

| 字段 | 类型 | 说明 |
|------|------|------|
| ts_code | str | 连续合约代码 |
| trade_date | str | 起始日期 |
| mapping_ts_code | str | 期货合约代码 |

## 调用示例

```bash
curl -X POST https://www.codebuddy.cn/v2/tool/financedata \
  -H "Content-Type: application/json" \
  -d '{
    "api_name": "fut_mapping",
    "params": {"ts_code": "CU.SHF"},
    "fields": "ts_code,trade_date,mapping_ts_code"
  }'
```

## 返回示例

```json
{
  "code": 0,
  "msg": "",
  "data": {
    "fields": ["ts_code", "trade_date", "mapping_ts_code"],
    "items": [
      ["CU.SHF", "20260303", "CU2604.SHF"],
      ["CU.SHF", "20260302", "CU2604.SHF"],
      ["CU.SHF", "20260227", "CU2604.SHF"]
    ]
  }
}
```
