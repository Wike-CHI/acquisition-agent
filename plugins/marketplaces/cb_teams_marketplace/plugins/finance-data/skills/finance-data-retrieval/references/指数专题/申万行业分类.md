# 申万行业分类

> 接口：`index_classify` | 获取申万行业分类列表，支持2014版和2021版

## 请求参数

| 参数 | 类型 | 必选 | 说明 |
|------|------|------|------|
| index_code | str | N | 指数代码 |
| level | str | N | 行业分级（L1/L2/L3） |
| parent_code | str | N | 父级代码（一级为0） |
| src | str | N | 来源：SW2014-申万2014版 SW2021-申万2021版 |

## 返回字段

| 字段 | 类型 | 说明 |
|------|------|------|
| index_code | str | 指数代码 |
| industry_name | str | 行业名称 |
| level | str | 行业层级 |
| industry_code | str | 行业代码 |
| is_pub | str | 是否发布指数 |
| parent_code | str | 父级代码 |
| src | str | 行业分类来源 |

## 调用示例

```bash
curl -X POST https://www.codebuddy.cn/v2/tool/financedata \
  -H "Content-Type: application/json" \
  -d '{
    "api_name": "index_classify",
    "params": {"level": "L1", "src": "SW2021"},
    "fields": ""
  }'
```

## 返回示例

```json
{
  "code": 0,
  "msg": "",
  "data": {
    "fields": ["index_code", "industry_name", "level", "industry_code", "is_pub", "parent_code", "src"],
    "items": [
      ["801010.SI", "农林牧渔", "L1", "110000", "1", "0", "SW2021"],
      ["801030.SI", "基础化工", "L1", "220000", "1", "0", "SW2021"],
      ["801040.SI", "钢铁", "L1", "230000", "1", "0", "SW2021"]
    ]
  }
}
```
