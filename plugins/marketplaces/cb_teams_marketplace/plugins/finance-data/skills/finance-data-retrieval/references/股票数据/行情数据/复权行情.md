# 复权行情

> 接口：`pro_bar` | 通过SDK获取A股复权行情，利用复权因子动态计算，不支持HTTP调用

## 请求参数

| 参数 | 类型 | 必选 | 说明 |
|------|------|------|------|
| ts_code | str | Y | 证券代码 |
| start_date | str | N | 开始日期（YYYYMMDD） |
| end_date | str | N | 结束日期（YYYYMMDD） |
| asset | str | Y | 资产类别：E-股票，I-指数，C-数字货币，FT-期货，FD-基金，O-期权（默认E） |
| adj | str | N | 复权类型：None-不复权，qfq-前复权，hfq-后复权（默认None） |
| freq | str | Y | 频度：D-日线，W-周线，M-月线，1/5/15/30/60min（默认D） |
| ma | list | N | 均线周期（如5、20、50） |

## 返回字段

具体返回字段因资产类型而异，股票日线参考 `daily` 接口。

## 调用说明

此接口为SDK级别整合接口，包含复权计算逻辑，**不支持HTTP请求**。如需HTTP方式获取复权数据，请使用 `stk_week_month_adj` 接口获取静态复权数据。

```python
import tushare as ts

# 前复权
df = ts.pro_bar(ts_code='000001.SZ', adj='qfq', start_date='20250101', end_date='20250110')

# 后复权
df = ts.pro_bar(ts_code='000001.SZ', adj='hfq', start_date='20250101', end_date='20250110')

# 周线前复权
df = ts.pro_bar(ts_code='000001.SZ', freq='W', adj='qfq', start_date='20250101', end_date='20250131')
```

## 复权说明

| 类型 | 算法 | 参数 |
|------|------|------|
| 不复权 | 无 | 空或None |
| 前复权 | 日收盘价 × 日复权因子 / 最新复权因子 | qfq |
| 后复权 | 日收盘价 × 日复权因子 | hfq |
