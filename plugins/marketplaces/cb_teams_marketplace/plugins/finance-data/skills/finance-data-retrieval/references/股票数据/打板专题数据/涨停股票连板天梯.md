# 连板天梯

> 接口：`limit_step` | 获取每天连板晋级股票，可分析连续涨停进阶个数，判断强势热度

## 请求参数

| 参数 | 类型 | 必选 | 说明 |
|------|------|------|------|
| trade_date | str | N | 交易日期（YYYYMMDD格式） |
| ts_code | str | N | 股票代码 |
| start_date | str | N | 开始日期 |
| end_date | str | N | 结束日期 |
| nums | str | N | 连板个数，支持多个（如 nums='2,3'） |

## 返回字段

| 字段 | 类型 | 说明 |
|------|------|------|
| ts_code | str | 股票代码 |
| name | str | 股票名称 |
| trade_date | str | 交易日期 |
| nums | str | 连板个数 |

## 调用示例

```bash
curl -X POST https://www.codebuddy.cn/v2/tool/financedata \
  -H "Content-Type: application/json" \
  -d '{
    "api_name": "limit_step",
    "params": {"trade_date": "20250228"},
    "fields": "ts_code,name,trade_date,nums"
  }'
```

## 返回示例

```json
{
  "code": 0,
  "msg": "",
  "data": {
    "fields": ["ts_code", "name", "trade_date", "nums"],
    "items": [
      ["605100.SH", "华丰股份", "20250228", "6"],
      ["002276.SZ", "万马股份", "20250228", "3"],
      ["603496.SH", "恒为科技", "20250228", "3"]
    ]
  }
}
```
