# 可转债行情

> 接口：`cb_daily` | 获取可转债日线行情数据

## 请求参数

| 参数 | 类型 | 必选 | 说明 |
|------|------|------|------|
| ts_code | str | N | 转债代码 |
| trade_date | str | N | 交易日期（YYYYMMDD） |
| start_date | str | N | 开始日期 |
| end_date | str | N | 结束日期 |

## 返回字段

| 字段 | 类型 | 说明 |
|------|------|------|
| ts_code | str | 转债代码 |
| trade_date | str | 交易日期 |
| pre_close | float | 昨收盘价（元） |
| open | float | 开盘价（元） |
| high | float | 最高价（元） |
| low | float | 最低价（元） |
| close | float | 收盘价（元） |
| change | float | 涨跌额（元） |
| pct_chg | float | 涨跌幅（%） |
| vol | float | 成交量（手） |
| amount | float | 成交金额（万元） |
| bond_value | float | 纯债价值 |
| bond_over_rate | float | 纯债溢价率（%） |
| cb_value | float | 转股价值 |
| cb_over_rate | float | 转股溢价率（%） |

## 调用示例

```bash
curl -X POST https://www.codebuddy.cn/v2/tool/financedata \
  -H "Content-Type: application/json" \
  -d '{
    "api_name": "cb_daily",
    "params": {"trade_date": "20250228"},
    "fields": "ts_code,trade_date,open,high,low,close,pct_chg,vol,amount"
  }'
```
