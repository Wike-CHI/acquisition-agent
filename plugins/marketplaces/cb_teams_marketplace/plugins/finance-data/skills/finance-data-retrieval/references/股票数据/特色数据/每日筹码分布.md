# 每日筹码分布

> 接口：`cyq_chips` | 获取A股每日筹码分布数据，提供价格位置百分比。数据自2018年起，每日18:00-19:00更新

## 请求参数

| 参数 | 类型 | 必选 | 说明 |
|------|------|------|------|
| ts_code | str | Y | 股票代码 |
| trade_date | str | N | 交易日期（YYYYMMDD格式） |
| start_date | str | N | 开始日期 |
| end_date | str | N | 结束日期 |

## 返回字段

| 字段 | 类型 | 说明 |
|------|------|------|
| ts_code | str | 股票代码 |
| trade_date | str | 交易日期 |
| price | float | 成本价格 |
| percent | float | 价格百分比（%） |

## 调用示例

```bash
curl -X POST https://www.codebuddy.cn/v2/tool/financedata \
  -H "Content-Type: application/json" \
  -d '{
    "api_name": "cyq_chips",
    "params": {"ts_code": "000001.SZ", "trade_date": "20250228"},
    "fields": "ts_code,trade_date,price,percent"
  }'
```

## 返回示例

```json
{
  "code": 0,
  "msg": "",
  "data": {
    "fields": ["ts_code", "trade_date", "price", "percent"],
    "items": [
      ["000001.SZ", "20250228", 0.2, 0.02],
      ["000001.SZ", "20250228", 0.4, 0.02],
      ["000001.SZ", "20250228", 0.6, 0.02]
    ]
  }
}
```
