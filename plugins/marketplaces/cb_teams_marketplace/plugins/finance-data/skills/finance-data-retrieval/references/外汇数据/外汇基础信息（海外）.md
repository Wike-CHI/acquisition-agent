# 外汇基础信息（海外）

> 接口：`fx_obasic` | 获取海外外汇基础信息，目前只有FXCM交易商的数据

## 请求参数

| 参数 | 类型 | 必选 | 说明 |
|------|------|------|------|
| exchange | str | N | 交易商 |
| classify | str | N | 分类（FX/INDEX/COMMODITY/METAL/BUND/CRYPTO/FX_BASKET） |
| ts_code | str | N | TS代码 |

### 分类说明

| 分类代码 | 名称 | 示例 |
|----------|------|------|
| FX | 外汇货币对 | USDCNH（美元人民币对） |
| INDEX | 指数 | US30（道琼斯工业平均指数） |
| COMMODITY | 大宗商品 | SOYF（大豆） |
| METAL | 金属 | XAUUSD（黄金） |
| BUND | 国库债券 | Bund（长期欧元债券） |
| CRYPTO | 加密数字货币 | BTCUSD（比特币） |
| FX_BASKET | 外汇篮子 | USDOLLAR（美元指数） |

## 返回字段

| 字段 | 类型 | 说明 |
|------|------|------|
| ts_code | str | 外汇代码 |
| name | str | 名称 |
| classify | str | 分类 |
| exchange | str | 交易商 |
| min_unit | float | 最小交易单位 |
| max_unit | float | 最大交易单位 |
| pip | float | 点 |
| pip_cost | float | 点值 |
| traget_spread | float | 目标差价 |
| min_stop_distance | float | 最小止损距离（点子） |
| trading_hours | str | 交易时间 |
| break_time | str | 休市时间 |

## 调用示例

```bash
curl -X POST https://www.codebuddy.cn/v2/tool/financedata \
  -H "Content-Type: application/json" \
  -d '{
    "api_name": "fx_obasic",
    "params": {"exchange": "FXCM", "classify": "FX"},
    "fields": "ts_code,name,classify,exchange,min_unit,max_unit,pip,pip_cost"
  }'
```
