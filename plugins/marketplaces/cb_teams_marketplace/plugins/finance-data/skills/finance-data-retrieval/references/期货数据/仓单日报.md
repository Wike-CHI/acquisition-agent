# 仓单日报

> 接口：`fut_wsr` | 获取仓单日报数据，跟踪各存放地点仓库/厂库仓单变化

## 请求参数

| 参数 | 类型 | 必选 | 说明 |
|------|------|------|------|
| trade_date | str | N | 交易日期 |
| symbol | str | N | 品种代码 |
| start_date | str | N | 开始日期（YYYYMMDD格式） |
| end_date | str | N | 结束日期 |
| exchange | str | N | 交易所代码 |

## 返回字段

| 字段 | 类型 | 说明 |
|------|------|------|
| trade_date | str | 交易日期 |
| symbol | str | 品种代码 |
| fut_name | str | 品种名称 |
| warehouse | str | 仓库名称 |
| wh_id | str | 仓库ID |
| pre_vol | int | 昨日仓单量 |
| vol | int | 今日仓单量 |
| vol_chg | int | 变化量 |
| area | str | 地区 |
| year | str | 年度 |
| grade | str | 等级 |
| brand | str | 品牌 |
| place | str | 产地 |
| pd | int | 升贴水 |
| is_ct | str | 是否为转换仓单 |
| unit | str | 单位 |
| exchange | str | 交易所 |

## 调用示例

```bash
curl -X POST https://www.codebuddy.cn/v2/tool/financedata \
  -H "Content-Type: application/json" \
  -d '{
    "api_name": "fut_wsr",
    "params": {"trade_date": "20250110", "symbol": "CU"},
    "fields": "trade_date,symbol,fut_name,warehouse,pre_vol,vol,vol_chg,unit"
  }'
```

## 返回示例

```json
{
  "code": 0,
  "msg": "",
  "data": {
    "fields": ["trade_date", "symbol", "fut_name", "warehouse", "pre_vol", "vol", "vol_chg", "unit"],
    "items": [
      ["20250110", "CU", "铜(BC)", "上海吉开伊", 0, 0, 0, "吨"],
      ["20250110", "CU", "铜(BC)", "上海国储", 850, 850, 0, "吨"],
      ["20250110", "CU", "铜", "上海裕强", 0, 0, 0, "吨"]
    ]
  }
}
```
