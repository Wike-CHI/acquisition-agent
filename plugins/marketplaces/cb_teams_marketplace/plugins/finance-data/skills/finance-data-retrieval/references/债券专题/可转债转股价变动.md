# 可转债转股价变动

> 接口：`cb_price_chg` | 获取可转债转股价变动数据

## 请求参数

| 参数 | 类型 | 必选 | 说明 |
|------|------|------|------|
| ts_code | str | Y | 转债代码，支持多值输入 |

## 返回字段

| 字段 | 类型 | 说明 |
|------|------|------|
| ts_code | str | 转债代码 |
| bond_short_name | str | 转债简称 |
| publish_date | str | 公告日期 |
| change_date | str | 变动日期 |
| convert_price_initial | float | 初始转股价格 |
| convertprice_bef | float | 修正前转股价格 |
| convertprice_aft | float | 修正后转股价格 |

## 调用示例

```bash
curl -X POST https://www.codebuddy.cn/v2/tool/financedata \
  -H "Content-Type: application/json" \
  -d '{
    "api_name": "cb_price_chg",
    "params": {"ts_code": "113009.SH"},
    "fields": "ts_code,bond_short_name,change_date,convert_price_initial,convertprice_bef,convertprice_aft"
  }'
```

    "items": [

      ["113009.SH", "广汽转债", "20161013", "20161020", 21.99, 21.87, 21.79]

  }
