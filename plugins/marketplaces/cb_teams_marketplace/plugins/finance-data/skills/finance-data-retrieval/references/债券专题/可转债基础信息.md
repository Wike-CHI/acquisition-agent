# 可转债基础信息

> 接口：`cb_basic` | 获取可转债基本信息数据，包括转债代码、名称、正股、转股价等

## 请求参数

| 参数 | 类型 | 必选 | 说明 |
|------|------|------|------|
| ts_code | str | N | 转债代码 |
| list_date | str | N | 上市日期 |
| exchange | str | N | 上市交易所 |

## 返回字段

| 字段 | 类型 | 说明 |
|------|------|------|
| ts_code | str | 转债代码 |
| bond_full_name | str | 转债名称 |
| bond_short_name | str | 转债简称 |
| cb_code | str | 转股申报代码 |
| stk_code | str | 正股代码 |
| stk_short_name | str | 正股简称 |
| maturity | float | 发行期限（年） |
| par | float | 面值 |
| issue_price | float | 发行价格 |
| issue_size | float | 发行总额（元） |
| remain_size | float | 债券余额（元） |
| value_date | str | 起息日期 |
| maturity_date | str | 到期日期 |
| rate_type | str | 利率类型 |
| coupon_rate | float | 票面利率（%） |
| add_rate | float | 补偿利率（%） |
| pay_per_year | int | 年付息次数 |
| list_date | str | 上市日期 |
| delist_date | str | 摘牌日 |
| exchange | str | 上市交易所 |
| conv_start_date | str | 转股起始日 |
| conv_end_date | str | 转股截止日 |
| conv_stop_date | str | 停止转股日（提前到期） |
| first_conv_price | float | 初始转股价 |
| conv_price | float | 最新转股价 |
| rate_clause | str | 利率说明 |
| put_clause | str | 赎回条款 |
| maturity_put_price | str | 到期赎回价格（含税） |
| call_clause | str | 回售条款 |
| reset_clause | str | 特别向下修正条款 |
| conv_clause | str | 转股条款 |
| guarantor | str | 担保人 |
| guarantee_type | str | 担保方式 |
| issue_rating | str | 发行信用等级 |
| newest_rating | str | 最新信用等级 |
| rating_comp | str | 最新评级机构 |

## 调用示例

```bash
curl -X POST https://www.codebuddy.cn/v2/tool/financedata \
  -H "Content-Type: application/json" \
  -d '{
    "api_name": "cb_basic",
    "params": {"ts_code": "113009.SH"},
    "fields": "ts_code,bond_short_name,stk_code,stk_short_name,list_date,delist_date"
  }'
```
