# 同花顺概念和行业指数

> 接口：`ths_index` | 获取同花顺板块指数列表

## 请求参数

| 参数 | 类型 | 必选 | 说明 |
|------|------|------|------|
| ts_code | str | N | 指数代码 |
| exchange | str | N | 市场类型：A-A股、HK-港股、US-美股 |
| type | str | N | 指数类型：N-概念、I-行业、R-地域、S-特色、ST-风格、TH-主题、BB-宽基 |

## 返回字段

| 字段 | 类型 | 说明 |
|------|------|------|
| ts_code | str | 代码 |
| name | str | 名称 |
| count | int | 成分个数 |
| exchange | str | 交易所 |
| list_date | str | 上市日期 |
| type | str | 指数类型 |

## 调用示例

```bash
curl -X POST https://www.codebuddy.cn/v2/tool/financedata \
  -H "Content-Type: application/json" \
  -d '{
    "api_name": "ths_index",
    "params": {},
    "fields": "ts_code,name,count,exchange,list_date,type"
  }'
```

## 返回示例

```json
{
  "code": 0,
  "msg": "",
  "data": {
    "fields": ["ts_code", "name", "count", "exchange", "list_date", "type"],
    "items": [
      ["882001.TI", "安徽", 159, "A", "20070808", "R"],
      ["882002.TI", "北京", 448, "A", "20070808", "R"],
      ["882003.TI", "重庆", 67, "A", "20070808", "R"]
    ]
  }
}
```
