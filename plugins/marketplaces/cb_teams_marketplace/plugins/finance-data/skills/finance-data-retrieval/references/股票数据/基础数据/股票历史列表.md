# 股票历史列表

> 接口：`bak_basic` | 获取备用基础列表，数据从2016年开始，包含每日股票基本面信息

## 请求参数

| 参数 | 类型 | 必选 | 说明 |
|------|------|------|------|
| trade_date | str | N | 交易日期 |
| ts_code | str | N | 股票代码 |

## 返回字段

| 字段 | 类型 | 说明 |
|------|------|------|
| trade_date | str | 交易日期 |
| ts_code | str | TS股票代码 |
| name | str | 股票名称 |
| industry | str | 行业 |
| area | str | 地域 |
| pe | float | 市盈率（动态） |
| float_share | float | 流通股本（亿） |
| total_share | float | 总股本（亿） |
| total_assets | float | 总资产（亿） |
| liquid_assets | float | 流动资产（亿） |
| fixed_assets | float | 固定资产（亿） |
| reserved | float | 公积金 |
| reserved_pershare | float | 每股公积金 |
| eps | float | 每股收益 |
| bvps | float | 每股净资产 |
| pb | float | 市净率 |
| list_date | str | 上市日期 |
| undp | float | 未分配利润 |
| per_undp | float | 每股未分配利润 |
| rev_yoy | float | 收入同比（%） |
| profit_yoy | float | 利润同比（%） |
| gpr | float | 毛利率（%） |
| npr | float | 净利率（%） |
| holder_num | int | 股东人数 |

## 调用示例

```bash
curl -X POST https://www.codebuddy.cn/v2/tool/financedata \
  -H "Content-Type: application/json" \
  -d '{
    "api_name": "bak_basic",
    "params": {"trade_date": "20250228"},
    "fields": "trade_date,ts_code,name,industry,area,pe,float_share,total_share,list_date"
  }'
```
