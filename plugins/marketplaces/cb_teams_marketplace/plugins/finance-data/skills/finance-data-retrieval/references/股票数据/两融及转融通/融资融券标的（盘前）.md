# 融资融券标的（盘前）

> 接口：`margin_secs` | 获取沪深京三大交易所融资融券标的（包括ETF），每天盘前更新

## 请求参数

| 参数 | 类型 | 必选 | 说明 |
|------|------|------|------|
| ts_code | str | N | 证券代码 |
| trade_date | str | N | 交易日期 |
| exchange | str | N | 交易所（SSE-上交所、SZSE-深交所、BSE-北交所） |
| start_date | str | N | 开始日期 |
| end_date | str | N | 结束日期 |

## 返回字段

| 字段 | 类型 | 说明 |
|------|------|------|
| trade_date | str | 交易日期 |
| ts_code | str | 证券代码 |
| name | str | 证券名称 |
| exchange | str | 交易所 |

## 调用示例

```bash
curl -X POST https://www.codebuddy.cn/v2/tool/financedata \
  -H "Content-Type: application/json" \
  -d '{
    "api_name": "margin_secs",
    "params": {"trade_date": "20250228", "exchange": "SSE"},
    "fields": "trade_date,ts_code,name,exchange"
  }'
```

## 返回示例

```json
{
  "code": 0,
  "msg": "",
  "data": {
    "fields": ["trade_date", "ts_code", "name", "exchange"],
    "items": [
      ["20250228", "510050.SH", "50ETF", "SSE"],
      ["20250228", "510100.SH", "SZ50ETF", "SSE"],
      ["20250228", "510150.SH", "消费ETF", "SSE"]
    ]
  }
}
```
