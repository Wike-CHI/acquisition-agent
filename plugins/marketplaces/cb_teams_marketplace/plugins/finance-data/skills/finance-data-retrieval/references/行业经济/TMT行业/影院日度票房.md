# 影院日度票房

> 接口：`bo_cinema` | 获取每日各影院票房数据，数据从2018年9月起

## 请求参数

| 参数 | 类型 | 必选 | 说明 |
|------|------|------|------|
| date | str | Y | 日期（格式：YYYYMMDD） |

**权限：** 用户需至少500积分

## 返回字段

| 字段 | 类型 | 说明 |
|------|------|------|
| date | str | 日期 |
| c_name | str | 影院名称 |
| aud_count | int | 观众人数 |
| att_ratio | float | 上座率 |
| day_amount | float | 当日票房 |
| day_showcount | float | 当日场次 |
| avg_price | float | 场均票价（元） |
| p_pc | float | 场均人次 |
| rank | int | 排名 |

## 调用示例

```bash
curl -X POST https://www.codebuddy.cn/v2/tool/financedata \
  -H "Content-Type: application/json" \
  -d '{
    "api_name": "bo_cinema",
    "params": {"date": "20181014"},
    "fields": ""
  }'
```
