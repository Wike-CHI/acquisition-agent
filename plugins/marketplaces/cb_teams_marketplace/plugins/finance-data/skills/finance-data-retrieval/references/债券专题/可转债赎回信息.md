# 可转债赎回信息

> 接口：`cb_call` | 获取可转债到期赎回、强制赎回等相关信息

## 请求参数

| 参数 | 类型 | 必选 | 说明 |
|------|------|------|------|
| ts_code | str | N | 转债代码，支持多值输入 |
| ann_date | str | N | 公告日期（YYYYMMDD） |
| start_date | str | N | 公告开始日期 |
| end_date | str | N | 公告结束日期 |

## 返回字段

| 字段 | 类型 | 说明 |
|------|------|------|
| ts_code | str | 转债代码 |
| call_type | str | 赎回类型：到赎、强赎 |
| is_call | str | 赎回状态：公告强赎、公告不强赎、公告到期赎回等 |
| ann_date | str | 公告/通知日期 |
| call_date | str | 赎回日期 |
| call_price | float | 赎回价格含税（元/张） |
| call_price_tax | float | 赎回价格扣税后（元/张） |
| call_vol | float | 赎回债券数量（张） |
| call_amount | float | 赎回金额（万元） |
| payment_date | str | 行权后款项到账日 |
| call_reg_date | str | 赎回登记日 |

## 调用示例

```bash
curl -X POST https://www.codebuddy.cn/v2/tool/financedata \
  -H "Content-Type: application/json" \
  -d '{
    "api_name": "cb_call",
    "params": {"ts_code": "113009.SH"},
    "fields": "ts_code,call_type,is_call,ann_date,call_date,call_price"
  }'
```
