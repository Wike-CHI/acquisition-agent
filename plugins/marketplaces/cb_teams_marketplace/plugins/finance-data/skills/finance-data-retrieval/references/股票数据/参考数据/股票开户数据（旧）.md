# 股票开户数据（旧）

> 接口：`stk_account_old` | 获取历史股票开户数据（2008年1月至2015年5月），更新数据请使用 `stk_account`

## 请求参数

| 参数 | 类型 | 必选 | 说明 |
|------|------|------|------|
| start_date | str | N | 开始日期 |
| end_date | str | N | 结束日期 |

## 返回字段

| 字段 | 类型 | 说明 |
|------|------|------|
| date | str | 统计周期 |
| new_sh | int | 本周新开户数（沪市，户） |
| new_sz | int | 本周新开户数（深市，户） |
| active_sh | float | 活跃账户数（沪市，万户） |
| active_sz | float | 活跃账户数（深市，万户） |
| total_sh | float | 总账户数（沪市，万户） |
| total_sz | float | 总账户数（深市，万户） |
| trade_sh | float | 交易账户数（沪市，万户） |
| trade_sz | float | 交易账户数（深市，万户） |

## 调用示例

```bash
curl -X POST https://www.codebuddy.cn/v2/tool/financedata \
  -H "Content-Type: application/json" \
  -d '{
    "api_name": "stk_account_old",
    "params": {"start_date": "20140101", "end_date": "20140301"},
    "fields": ""
  }'
```
