# 申万实时行情

> 接口：`rt_sw_k` | 获取申万行业指数最新截面数据（需单独权限）

## 请求参数

| 参数 | 类型 | 必选 | 说明 |
|------|------|------|------|
| ts_code | str | N | 指数代码，支持逗号分隔多个（如801005.SI,801001.SI），不填则返回全部 |

## 返回字段

| 字段 | 类型 | 说明 |
|------|------|------|
| ts_code | str | 指数代码 |
| name | str | 指数名称 |
| trade_time | str | 交易时间 |
| close | float | 最新价 |
| pre_close | float | 昨收 |
| high | float | 最高价 |
| open | float | 开盘价 |
| low | float | 最低价 |
| vol | float | 成交量（股） |
| amount | float | 成交额（元） |
| pct_change | float | 涨跌幅 |

## 调用示例

```bash
curl -X POST https://www.codebuddy.cn/v2/tool/financedata \
  -H "Content-Type: application/json" \
  -d '{
    "api_name": "rt_sw_k",
    "params": {"ts_code": "801010.SI"},
    "fields": ""
  }'
```
