## 合约代码规则

| 主力合约 | 连续合约 | 普通合约 |
|------|------|------|
| XX（如CU.SHF） | XXL（如CUL.SHF） | XXMMDD（如CU1811.SHF） |

> 例外：TL0表示30年期国债主力合约，TL表示T合约主连，TLL表示TL合约主连

中金所连续合约规则：IFL.CFX（当月）、IFL1.CFX（次月）、IFL2.CFX（当季）、IFL3.CFX（下季）

## 数据规则

- 行情类数据（日线、结算参数等）：使用带交易所后缀的合约代码，如CU1811.SHF
- 品种类数据（持仓排名、仓单等）：使用品种代码，如CU

## 数据接口列表

| 数据名称 | 接口 | 描述 |
|------|------|------|
| 期货合约列表 | `fut_basic` | 全部历史合约信息 |
| 期货交易日历 | `trade_cal` | 1996年至今，定期更新 |
| 期货日线行情 | `fut_daily` | 1996年至今，每日更新 |
| 每日持仓排名 | `fut_holding` | 2002年至今，每日更新 |
| 仓单日报 | `fut_wsr` | 2006年至今，每日更新 |
| 结算参数 | `fut_settle` | 2012年至今，每日更新 |
| 期货周月线行情 | `fut_weekly_monthly` | 周线/月线行情，每日更新 |
| 历史分钟行情 | `ft_mins` | 支持1/5/15/30/60分钟 |
| 实时分钟行情 | `rt_fut_min` | 实时分钟数据 |
| 主力与连续合约 | `fut_mapping` | 主力合约映射 |
| 品种交易周报 | `fut_weekly_detail` | 2010年至今 |
| 涨跌停价格 | `ft_limit` | 2005年至今 |
| 南华期货指数 | `index_daily` | 南华指数行情 |
