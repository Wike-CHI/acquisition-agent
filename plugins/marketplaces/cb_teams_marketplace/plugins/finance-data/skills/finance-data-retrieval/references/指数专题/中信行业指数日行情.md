# 中信行业指数日行情

> 接口：`ci_daily` | 获取中信行业指数日线行情，单次最大4000行

## 请求参数

| 参数 | 类型 | 必选 | 说明 |
|------|------|------|------|
| ts_code | str | N | 行业代码 |
| trade_date | str | N | 交易日期（YYYYMMDD） |
| start_date | str | N | 开始日期 |
| end_date | str | N | 结束日期 |

## 返回字段

| 字段 | 类型 | 说明 |
|------|------|------|
| ts_code | str | 指数代码 |
| trade_date | str | 交易日期 |
| open | float | 开盘点位 |
| low | float | 最低点位 |
| high | float | 最高点位 |
| close | float | 收盘点位 |
| pre_close | float | 昨收点位 |
| change | float | 涨跌点位 |
| pct_change | float | 涨跌幅 |
| vol | float | 成交量（万股） |
| amount | float | 成交额（万元） |

## 调用示例

```bash
curl -X POST https://www.codebuddy.cn/v2/tool/financedata \
  -H "Content-Type: application/json" \
  -d '{
    "api_name": "ci_daily",
    "params": {"trade_date": "20250228"},
    "fields": "ts_code,trade_date,open,low,high,close"
  }'
```
