# 期货周月线行情

> 接口：`fut_weekly_monthly` | 获取期货周线/月线行情数据，每日更新

## 请求参数

| 参数 | 类型 | 必选 | 说明 |
|------|------|------|------|
| ts_code | str | N | 合约代码 |
| trade_date | str | N | 交易日期 |
| start_date | str | N | 开始日期 |
| end_date | str | N | 结束日期 |
| freq | str | Y | 频率：week-周线 month-月线 |
| exchange | str | N | 交易所 |

## 返回字段

| 字段 | 类型 | 说明 |
|------|------|------|
| ts_code | str | 期货代码 |
| trade_date | str | 交易日期（周五或月末） |
| end_date | str | 计算截止日期 |
| freq | str | 频率（week/month） |
| open | float | 开盘价 |
| high | float | 最高价 |
| low | float | 最低价 |
| close | float | 收盘价 |
| pre_close | float | 上期收盘价 |
| settle | float | 结算价 |
| pre_settle | float | 上期结算价 |
| vol | float | 成交量（手） |
| amount | float | 成交金额（万元） |
| oi | float | 持仓量（手） |
| oi_chg | float | 持仓变化 |
| exchange | str | 交易所 |
| change1 | float | 涨跌1（收盘价-上期结算价） |
| change2 | float | 涨跌2（结算价-上期结算价） |

## 调用示例

```bash
curl -X POST https://www.codebuddy.cn/v2/tool/financedata \
  -H "Content-Type: application/json" \
  -d '{
    "api_name": "fut_weekly_monthly",
    "params": {"ts_code": "CU2501.SHF", "freq": "week"},
    "fields": "ts_code,trade_date,open,high,low,close,settle,vol,amount"
  }'
```
