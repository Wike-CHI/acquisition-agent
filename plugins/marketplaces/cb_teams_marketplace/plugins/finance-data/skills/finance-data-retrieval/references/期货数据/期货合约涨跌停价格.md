# 期货合约涨跌停价格

> 接口：`ft_limit` | 获取所有期货合约每天的涨跌停价格及最低保证金率，数据自2005年起

## 请求参数

| 参数 | 类型 | 必选 | 说明 |
|------|------|------|------|
| ts_code | str | N | 合约代码 |
| trade_date | str | N | 交易日期（YYYYMMDD格式） |
| start_date | str | N | 开始日期 |
| end_date | str | N | 结束日期 |
| cont | str | N | 品种代码（如CU） |
| exchange | str | N | 交易所代码（如DCE） |

## 返回字段

| 字段 | 类型 | 说明 |
|------|------|------|
| trade_date | str | 交易日期 |
| ts_code | str | 合约代码 |
| name | str | 合约名称 |
| up_limit | float | 涨停价 |
| down_limit | float | 跌停价 |
| m_ratio | float | 最低交易保证金率（%） |
| cont | str | 品种代码 |
| exchange | str | 交易所代码 |

## 调用示例

```bash
curl -X POST https://www.codebuddy.cn/v2/tool/financedata \
  -H "Content-Type: application/json" \
  -d '{
    "api_name": "ft_limit",
    "params": {"ts_code": "CU2501.SHF"},
    "fields": "trade_date,ts_code,name,up_limit,down_limit,m_ratio,exchange"
  }'
```
