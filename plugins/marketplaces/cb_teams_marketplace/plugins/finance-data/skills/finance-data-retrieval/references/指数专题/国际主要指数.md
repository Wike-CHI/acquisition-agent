# 国际主要指数

> 接口：`index_global` | 获取国际主要指数日线行情，单次最大4000行

## 请求参数

| 参数 | 类型 | 必选 | 说明 |
|------|------|------|------|
| ts_code | str | N | 指数代码（见下表） |
| trade_date | str | N | 交易日期（YYYYMMDD） |
| start_date | str | N | 开始日期 |
| end_date | str | N | 结束日期 |

## 返回字段

| 字段 | 类型 | 说明 |
|------|------|------|
| ts_code | str | 指数代码 |
| trade_date | str | 交易日期 |
| open | float | 开盘点位 |
| close | float | 收盘点位 |
| high | float | 最高点位 |
| low | float | 最低点位 |
| pre_close | float | 昨收 |
| change | float | 涨跌点位 |
| pct_chg | float | 涨跌幅 |
| swing | float | 振幅 |
| vol | float | 成交量（大部分无数据） |
| amount | float | 成交额（大部分无数据） |

## 调用示例

```bash
curl -X POST https://www.codebuddy.cn/v2/tool/financedata \
  -H "Content-Type: application/json" \
  -d '{
    "api_name": "index_global",
    "params": {"ts_code": "SPX", "trade_date": "20250228"},
    "fields": ""
  }'
```
