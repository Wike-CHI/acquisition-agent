# 交易日历

> 接口：`trade_cal` | 获取各大交易所交易日历数据，默认上交所

## 请求参数

| 参数 | 类型 | 必选 | 说明 |
|------|------|------|------|
| exchange | str | N | 交易所：SSE-上交所 SZSE-深交所 CFFEX-中金所 SHFE-上期所 CZCE-郑商所 DCE-大商所 INE-上海能源 |
| start_date | str | N | 开始日期（YYYYMMDD格式） |
| end_date | str | N | 结束日期 |
| is_open | str | N | 是否交易：0-休市 1-交易 |

## 返回字段

| 字段 | 类型 | 说明 |
|------|------|------|
| exchange | str | 交易所代码 |
| cal_date | str | 日历日期 |
| is_open | int | 是否交易：0-休市 1-交易 |
| pretrade_date | str | 上一个交易日 |

## 调用示例

```bash
curl -X POST https://www.codebuddy.cn/v2/tool/financedata \
  -H "Content-Type: application/json" \
  -d '{
    "api_name": "trade_cal",
    "params": {"exchange": "SSE", "start_date": "20250101", "end_date": "20250110"},
    "fields": ""
  }'
```

## 返回示例

```json
{
  "code": 0,
  "msg": "",
  "data": {
    "fields": ["exchange", "cal_date", "is_open", "pretrade_date"],
    "items": [
      ["SSE", "20250110", 1, "20250109"],
      ["SSE", "20250109", 1, "20250108"],
      ["SSE", "20250108", 1, "20250107"]
    ]
  }
}
```
