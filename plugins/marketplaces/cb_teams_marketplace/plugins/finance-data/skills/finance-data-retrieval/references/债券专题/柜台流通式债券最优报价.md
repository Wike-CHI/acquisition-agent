# 柜台流通式债券最优报价

> 接口：`bc_bestotcqt` | 获取柜台流通式债券最优报价数据

## 请求参数

| 参数 | 类型 | 必选 | 说明 |
|------|------|------|------|
| trade_date | str | N | 报价日期（YYYYMMDD） |
| start_date | str | N | 开始日期 |
| end_date | str | N | 结束日期 |
| ts_code | str | N | 债券代码 |

## 返回字段

| 字段 | 类型 | 说明 |
|------|------|------|
| trade_date | str | 报价日期 |
| ts_code | str | 债券编码 |
| name | str | 债券简称 |
| remain_maturity | str | 剩余期限 |
| bond_type | str | 债券类型 |
| best_buy_bank | str | 最优报买价方 |
| best_buy_yield | float | 投资者最优买入到期收益率（%） |
| best_buy_price | float | 投资者最优买入全价 |
| best_sell_bank | str | 最优报卖价方 |
| best_sell_yield | float | 投资者最优卖出到期收益率（%） |
| best_sell_price | float | 投资者最优卖出全价 |

## 调用示例

```bash
curl -X POST https://www.codebuddy.cn/v2/tool/financedata \
  -H "Content-Type: application/json" \
  -d '{
    "api_name": "bc_bestotcqt",
    "params": {"ts_code": "200013.BC", "start_date": "20240325", "end_date": "20240329"},
    "fields": "trade_date,ts_code,name,remain_maturity,best_buy_bank,best_buy_yield,best_sell_bank,best_sell_yield"
  }'
```
