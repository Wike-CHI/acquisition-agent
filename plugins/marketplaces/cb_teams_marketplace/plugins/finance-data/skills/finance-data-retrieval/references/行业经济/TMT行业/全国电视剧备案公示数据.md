# 全国电视剧备案公示数据

> 接口：`teleplay_record` | 获取2009年以来全国拍摄制作电视剧备案公示数据

## 请求参数

| 参数 | 类型 | 必选 | 说明 |
|------|------|------|------|
| report_date | str | N | 备案月份（YYYYMM） |
| start_date | str | N | 备案开始月份（YYYYMM） |
| end_date | str | N | 备案结束月份（YYYYMM） |
| org | str | N | 备案机构 |
| name | str | N | 电视剧名称 |

**限量：** 单次最大1000条，总量不限。**权限：** 用户需至少600积分。

## 返回字段

| 字段 | 类型 | 说明 |
|------|------|------|
| name | str | 电视剧名称 |
| classify | str | 题材 |
| types | str | 体裁 |
| org | str | 报备机构 |
| report_date | str | 报备时间 |
| license_key | str | 许可证号 |
| episodes | str | 集数 |
| shooting_date | str | 拍摄时间 |
| prod_cycle | str | 制作周期 |
| content | str | 内容提要 |
| pro_opi | str | 省级管理部门备案意见 |
| dept_opi | str | 相关部门意见 |
| remarks | str | 备注 |

## 调用示例

```bash
curl -X POST https://www.codebuddy.cn/v2/tool/financedata \
  -H "Content-Type: application/json" \
  -d '{
    "api_name": "teleplay_record",
    "params": {"report_date": "201905"},
    "fields": ""
  }'
```
