# 美股复权行情

> 接口：`us_daily_adj` | 获取美股复权日线行情数据，包含复权因子、股本、市值等信息

> 复权逻辑：价格 x 复权因子 = 复权价格（如 close x adj_factor = 前复权收盘价）。历史复权因子可能因公司行为而刷新，需动态更新

## 请求参数

| 参数 | 类型 | 必选 | 说明 |
|------|------|------|------|
| ts_code | str | N | 股票代码，如AAPL |
| trade_date | str | N | 交易日期（YYYYMMDD） |
| start_date | str | N | 开始日期 |
| end_date | str | N | 结束日期 |
| exchange | str | N | 交易所代码（NAS/NYS/OTC） |
| offset | int | N | 起始行号 |
| limit | int | N | 每页行数 |

## 返回字段

| 字段 | 类型 | 说明 |
|------|------|------|
| ts_code | str | 股票代码 |
| trade_date | str | 交易日期 |
| close | float | 收盘价 |
| open | float | 开盘价 |
| high | float | 最高价 |
| low | float | 最低价 |
| pre_close | float | 昨收价 |
| change | float | 涨跌额 |
| pct_change | float | 涨跌幅（%） |
| vol | int | 成交量 |
| amount | float | 成交金额 |
| vwap | float | 成交均价 |
| adj_factor | float | 复权因子 |
| turnover_ratio | float | 换手率 |
| free_share | int | 流通股本 |
| total_share | int | 总股本 |
| free_mv | float | 流通市值 |
| total_mv | float | 总市值 |
| exchange | str | 交易所代码 |

## 调用示例

```bash
curl -X POST https://www.codebuddy.cn/v2/tool/financedata \
  -H "Content-Type: application/json" \
  -d '{
    "api_name": "us_daily_adj",
    "params": {"ts_code": "AAPL", "start_date": "20250201", "end_date": "20250228"},
    "fields": "ts_code,trade_date,close,pct_change,vol,amount,adj_factor,turnover_ratio,total_mv"
  }'
```
