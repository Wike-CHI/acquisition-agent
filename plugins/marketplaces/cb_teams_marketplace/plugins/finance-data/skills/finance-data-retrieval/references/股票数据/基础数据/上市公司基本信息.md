# 上市公司基本信息

> 接口：`stock_company` | 获取上市公司基础信息，可按交易所分批提取

## 请求参数

| 参数 | 类型 | 必选 | 说明 |
|------|------|------|------|
| ts_code | str | N | 股票代码 |
| exchange | str | N | 交易所：SSE-上交所 SZSE-深交所 BSE-北交所 |

## 返回字段

| 字段 | 类型 | 说明 |
|------|------|------|
| ts_code | str | 股票代码 |
| com_name | str | 公司全称 |
| com_id | str | 统一社会信用代码 |
| exchange | str | 交易所代码 |
| chairman | str | 法人代表 |
| manager | str | 总经理 |
| secretary | str | 董秘 |
| reg_capital | float | 注册资本（万元） |
| setup_date | str | 注册日期 |
| province | str | 所在省份 |
| city | str | 所在城市 |
| introduction | str | 公司介绍 |
| website | str | 公司主页 |
| email | str | 电子邮件 |
| office | str | 办公室 |
| employees | int | 员工人数 |
| main_business | str | 主要业务及产品 |
| business_scope | str | 经营范围 |

## 调用示例

```bash
curl -X POST https://www.codebuddy.cn/v2/tool/financedata \
  -H "Content-Type: application/json" \
  -d '{
    "api_name": "stock_company",
    "params": {"exchange": "SSE"},
    "fields": "ts_code,com_name,chairman,manager,secretary,reg_capital,setup_date,province"
  }'
```
