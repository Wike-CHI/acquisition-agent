# 沪深市场每日交易统计

> 接口：`daily_info` | 获取沪深交易所每日交易统计数据，包括各板块详细信息

## 请求参数

| 参数 | 类型 | 必选 | 说明 |
|------|------|------|------|
| trade_date | str | N | 交易日期（YYYYMMDD） |
| ts_code | str | N | 板块代码（见下表） |
| exchange | str | N | 交易所：SH-上交所 SZ-深交所 |
| start_date | str | N | 开始日期 |
| end_date | str | N | 结束日期 |
| fields | str | N | 指定返回字段 |

### 深交所板块代码

| 代码 | 说明 |
|------|------|
| SZ_MARKET | 深圳市场 |
| SZ_MAIN | 深市主板 |
| SZ_A | 深市A股 |
| SZ_B | 深市B股 |
| SZ_GEM | 创业板 |
| SZ_SME | 中小企业板 |
| SZ_FUND | 深市基金 |
| SZ_FUND_ETF | 深市ETF |
| SZ_BOND | 深市债券 |

### 上交所板块代码

| 代码 | 说明 |
|------|------|
| SH_MARKET | 上海市场 |
| SH_A | 沪市A股 |
| SH_B | 沪市B股 |
| SH_STAR | 科创板 |
| SH_FUND | 沪市基金 |
| SH_FUND_ETF | 沪市ETF |

## 返回字段

| 字段 | 类型 | 说明 |
|------|------|------|
| trade_date | str | 交易日期 |
| ts_code | str | 市场代码 |
| ts_name | str | 市场名称 |
| com_count | int | 上市公司数 |
| total_share | float | 总股本（亿股） |
| float_share | float | 流通股本（亿股） |
| total_mv | float | 总市值（亿元） |
| float_mv | float | 流通市值（亿元） |
| amount | float | 成交额（亿元） |
| vol | float | 成交量（亿股） |
| trans_count | int | 成交笔数（万笔） |
| pe | float | 平均市盈率 |
| tr | float | 换手率（%），深交所无数据 |
| exchange | str | 交易所 |

## 调用示例

```bash
curl -X POST https://www.codebuddy.cn/v2/tool/financedata \
  -H "Content-Type: application/json" \
  -d '{
    "api_name": "daily_info",
    "params": {"trade_date": "20250228", "exchange": "SH"},
    "fields": ""
  }'
```
