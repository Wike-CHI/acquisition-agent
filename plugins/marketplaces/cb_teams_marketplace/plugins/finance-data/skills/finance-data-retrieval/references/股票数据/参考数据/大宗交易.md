# 大宗交易

> 接口：`block_trade` | 获取大宗交易数据

## 请求参数

| 参数 | 类型 | 必选 | 说明 |
|------|------|------|------|
| ts_code | str | N | TS股票代码 |
| trade_date | str | N | 交易日期 |
| start_date | str | N | 开始日期 |
| end_date | str | N | 结束日期 |

## 返回字段

| 字段 | 类型 | 说明 |
|------|------|------|
| ts_code | str | TS代码 |
| trade_date | str | 交易日期 |
| price | float | 成交价 |
| vol | float | 成交量（万股） |
| amount | float | 成交金额 |
| buyer | str | 买方营业部 |
| seller | str | 卖方营业部 |

## 调用示例

```bash
curl -X POST https://www.codebuddy.cn/v2/tool/financedata \
  -H "Content-Type: application/json" \
  -d '{
    "api_name": "block_trade",
    "params": {"trade_date": "20241220"},
    "fields": ""
  }'
```
