# 新闻快讯

> 接口：`news` | 获取主流新闻网站快讯数据，提供超过6年历史新闻

## 请求参数

| 参数 | 类型 | 必选 | 说明 |
|------|------|------|------|
| start_date | datetime | Y | 开始日期（格式：2018-11-20 09:00:00） |
| end_date | datetime | Y | 结束日期 |
| src | str | Y | 新闻来源（见下表） |

**限量：** 单次最大1500条。**权限：** 需单独开通权限。

### 数据来源

| 来源 | src标识 |
|------|---------|
| 新浪财经 | sina |
| 华尔街见闻 | wallstreetcn |
| 同花顺 | 10jqka |
| 东方财富 | eastmoney |
| 云财经 | yuncaijing |
| 凤凰新闻 | fenghuang |
| 金融界 | jinrongjie |
| 财联社 | cls |
| 第一财经 | yicai |

## 返回字段

| 字段 | 类型 | 说明 |
|------|------|------|
| datetime | str | 新闻时间戳 |
| content | str | 内容 |
| title | str | 标题 |
| channels | str | 分类（默认不显示） |

## 调用示例

```bash
curl -X POST https://www.codebuddy.cn/v2/tool/financedata \
  -H "Content-Type: application/json" \
  -d '{
    "api_name": "news",
    "params": {"src": "sina", "start_date": "2025-02-28 09:00:00", "end_date": "2025-02-28 12:00:00"},
    "fields": ""
  }'
```
