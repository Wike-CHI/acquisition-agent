# 北交所新旧代码对照

> 接口：`bse_mapping` | 获取北交所股票代码变更后新旧代码映射表

## 请求参数

| 参数 | 类型 | 必选 | 说明 |
|------|------|------|------|
| o_code | str | N | 旧代码 |
| n_code | str | N | 新代码 |

## 返回字段

| 字段 | 类型 | 说明 |
|------|------|------|
| name | str | 股票名称 |
| o_code | str | 原代码 |
| n_code | str | 新代码 |
| list_date | str | 上市日期 |

## 调用示例

```bash
curl -X POST https://www.codebuddy.cn/v2/tool/financedata \
  -H "Content-Type: application/json" \
  -d '{
    "api_name": "bse_mapping",
    "params": {},
    "fields": ""
  }'
```

## 返回示例

```json
{
  "code": 0,
  "msg": "",
  "data": {
    "fields": ["name", "o_code", "n_code", "list_date"],
    "items": [
      ["永顺生物", "839729.BJ", "920729.BJ", "20200727"],
      ["同享科技", "839167.BJ", "920167.BJ", "20200727"],
      ["方大新材", "838163.BJ", "920163.BJ", "20200727"]
    ]
  }
}
```
