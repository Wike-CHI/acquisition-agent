# 深圳市场每日交易情况

> 接口：`sz_daily_info` | 获取深圳市场每日交易概况，单次最大2000行

## 请求参数

| 参数 | 类型 | 必选 | 说明 |
|------|------|------|------|
| trade_date | str | N | 交易日期（YYYYMMDD） |
| ts_code | str | N | 板块代码（见下表） |
| start_date | str | N | 开始日期 |
| end_date | str | N | 结束日期 |

### 板块代码

| 代码 | 说明 |
|------|------|
| 股票 | 深圳市场股票总和 |
| 主板A股 | 深圳主板A股 |
| 主板B股 | 深圳主板B股 |
| 创业板A股 | 深圳创业板 |
| 基金 | 深圳基金总和 |
| ETF | 深圳ETF |
| LOF | 深圳LOF |
| 封闭式基金 | 深圳封闭式基金 |
| 基础设施基金 | 深圳REITs基金 |
| 债券 | 深圳债券总和 |
| 债券现券 | 深圳现券 |
| 债券回购 | 深圳债券回购 |
| ABS | 深圳ABS |
| 期权 | 深圳期权 |

## 返回字段

| 字段 | 类型 | 说明 |
|------|------|------|
| trade_date | str | 交易日期 |
| ts_code | str | 市场类型 |
| count | int | 股票个数 |
| amount | float | 成交金额 |
| vol | float | 成交量 |
| total_share | float | 总股本 |
| total_mv | float | 总市值 |
| float_share | float | 流通股本 |
| float_mv | float | 流通市值 |

## 调用示例

```bash
curl -X POST https://www.codebuddy.cn/v2/tool/financedata \
  -H "Content-Type: application/json" \
  -d '{
    "api_name": "sz_daily_info",
    "params": {"trade_date": "20250228"},
    "fields": ""
  }'
```
