# 财务审计意见

> 接口：`fina_audit` | 获取上市公司定期财务审计意见数据

## 请求参数

| 参数 | 类型 | 必选 | 说明 |
|------|------|------|------|
| ts_code | str | Y | TS股票代码 |
| ann_date | str | N | 公告日期 |
| start_date | str | N | 公告开始日期 |
| end_date | str | N | 公告结束日期 |
| period | str | N | 报告期（每个季度最后一天，如20241231） |

## 返回字段

| 字段 | 类型 | 说明 |
|------|------|------|
| ts_code | str | TS代码 |
| ann_date | str | 公告日期 |
| end_date | str | 报告期 |
| audit_result | str | 审计结果 |
| audit_fees | float | 审计费用（元） |
| audit_agency | str | 会计师事务所 |
| audit_sign | str | 签字会计师 |

## 调用示例

```bash
curl -X POST https://www.codebuddy.cn/v2/tool/financedata \
  -H "Content-Type: application/json" \
  -d '{
    "api_name": "fina_audit",
    "params": {"ts_code": "000001.SZ"},
    "fields": ""
  }'
```
