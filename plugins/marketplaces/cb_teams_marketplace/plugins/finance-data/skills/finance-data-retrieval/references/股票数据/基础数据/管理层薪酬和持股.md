# 管理层薪酬和持股

> 接口：`stk_rewards` | 获取上市公司管理层薪酬和持股

## 请求参数

| 参数 | 类型 | 必选 | 说明 |
|------|------|------|------|
| ts_code | str | Y | TS股票代码，支持单个或多个 |
| end_date | str | N | 报告期 |

## 返回字段

| 字段 | 类型 | 说明 |
|------|------|------|
| ts_code | str | TS股票代码 |
| ann_date | str | 公告日期 |
| end_date | str | 截止日期 |
| name | str | 姓名 |
| title | str | 职务 |
| reward | float | 报酬（元） |
| hold_vol | float | 持股数 |

## 调用示例

```bash
curl -X POST https://www.codebuddy.cn/v2/tool/financedata \
  -H "Content-Type: application/json" \
  -d '{
    "api_name": "stk_rewards",
    "params": {"ts_code": "000001.SZ"},
    "fields": ""
  }'
```
