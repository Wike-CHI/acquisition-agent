# 通用行情接口

> 接口：`pro_bar` | SDK级别整合接口，支持股票、指数、期货、基金、期权等多种资产行情获取，不支持HTTP调用

## 请求参数

| 参数 | 类型 | 必选 | 说明 |
|------|------|------|------|
| ts_code | str | Y | 证券代码，不支持多值输入 |
| start_date | str | N | 开始日期（日线：YYYYMMDD，分钟：2025-01-01 09:00:00） |
| end_date | str | N | 结束日期 |
| asset | str | Y | 资产类别：E-股票，I-指数，C-数字货币，FT-期货，FD-基金，O-期权，CB-可转债（默认E） |
| adj | str | N | 复权类型（仅股票）：None-不复权，qfq-前复权，hfq-后复权（默认None） |
| freq | str | Y | 频度：D-日线，W-周线，M-月线，1/5/15/30/60min（默认D） |
| ma | list | N | 均线周期，支持任意整数 |
| factors | list | N | 股票因子：tor-换手率，vr-量比（仅asset=E有效） |
| adjfactor | str | N | 是否返回复权因子（默认False） |

## 返回字段

具体返回字段因资产类型而异：
- 股票日线：参考 `daily` 接口
- 基金日线：参考 `fund_daily` 接口
- 期货日线：参考 `fut_daily` 接口
- 期权日线：参考 `opt_daily` 接口
- 指数日线：参考 `index_daily` 接口

## 调用说明

此接口为SDK级别整合接口，包含复权计算等逻辑处理，**不支持HTTP请求**。

```python
import tushare as ts

# 前复权股票数据
df = ts.pro_bar(ts_code='000001.SZ', adj='qfq', start_date='20250101', end_date='20250110')

# 上证指数数据
df = ts.pro_bar(ts_code='000001.SH', asset='I', start_date='20250101', end_date='20250110')

# 带均线数据
df = ts.pro_bar(ts_code='000001.SZ', start_date='20250101', end_date='20250110', ma=[5, 20, 50])

# 带换手率和量比
df = ts.pro_bar(ts_code='000001.SZ', start_date='20250101', end_date='20250110', factors=['tor', 'vr'])
```
