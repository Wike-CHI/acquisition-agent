# 基金持仓

> 接口：`fund_portfolio` | 获取公募基金持仓数据，季度更新

## 请求参数

| 参数 | 类型 | 必选 | 说明 |
|------|------|------|------|
| ts_code | str | N | 基金代码（ts_code、ann_date、period至少输入一个） |
| symbol | str | N | 股票代码 |
| ann_date | str | N | 公告日期（YYYYMMDD格式） |
| period | str | N | 季度（每个季度最后一天，如20231231表示2023年年报） |
| start_date | str | N | 报告期开始日期 |
| end_date | str | N | 报告期结束日期 |

## 返回字段

| 字段 | 类型 | 说明 |
|------|------|------|
| ts_code | str | 基金代码 |
| ann_date | str | 公告日期 |
| end_date | str | 截止日期 |
| symbol | str | 股票代码 |
| mkv | float | 持有股票市值（元） |
| amount | float | 持有股票数量（股） |
| stk_mkv_ratio | float | 占股票市值比 |
| stk_float_ratio | float | 占流通股本比例 |

## 调用示例

```bash
curl -X POST https://www.codebuddy.cn/v2/tool/financedata \
  -H "Content-Type: application/json" \
  -d '{
    "api_name": "fund_portfolio",
    "params": {"ts_code": "510050.SH"},
    "fields": "ts_code,ann_date,end_date,symbol,mkv,amount,stk_mkv_ratio"
  }'
```
