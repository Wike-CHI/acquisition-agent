# 转融资交易汇总

> 接口：`slb_len` | 获取转融通融资交易汇总数据

## 请求参数

| 参数 | 类型 | 必选 | 说明 |
|------|------|------|------|
| trade_date | str | N | 交易日期（YYYYMMDD格式） |
| start_date | str | N | 开始日期 |
| end_date | str | N | 结束日期 |

## 返回字段

| 字段 | 类型 | 说明 |
|------|------|------|
| trade_date | str | 交易日期 |
| ob | float | 期初余额（亿元） |
| auc_amount | float | 竞价成交金额（亿元） |
| repo_amount | float | 续借成交金额（亿元） |
| repay_amount | float | 偿还金额（亿元） |
| cb | float | 期末余额（亿元） |

## 调用示例

```bash
curl -X POST https://www.codebuddy.cn/v2/tool/financedata \
  -H "Content-Type: application/json" \
  -d '{
    "api_name": "slb_len",
    "params": {"trade_date": "20250228"},
    "fields": "trade_date,ob,auc_amount,repo_amount,repay_amount,cb"
  }'
```

## 返回示例

```json
{
  "code": 0,
  "msg": "",
  "data": {
    "fields": ["trade_date", "ob", "auc_amount", "repo_amount", "repay_amount", "cb"],
    "items": [
      ["20250228", 1436.6, null, null, null, 1436.6]
    ]
  }
}
```
