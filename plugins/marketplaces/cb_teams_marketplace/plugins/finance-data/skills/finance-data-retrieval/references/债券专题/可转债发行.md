# 可转债发行

> 接口：`cb_issue` | 获取可转债发行数据

## 请求参数

| 参数 | 类型 | 必选 | 说明 |
|------|------|------|------|
| ts_code | str | N | 转债代码 |
| ann_date | str | N | 发行公告日 |
| start_date | str | N | 公告开始日期 |
| end_date | str | N | 公告结束日期 |

## 返回字段

| 字段 | 类型 | 说明 |
|------|------|------|
| ts_code | str | 转债代码 |
| ann_date | str | 发行公告日 |
| res_ann_date | str | 发行结果公告日 |
| plan_issue_size | float | 计划发行总额（元） |
| issue_size | float | 实际发行总额（元） |
| issue_price | float | 发行价格 |
| issue_type | str | 发行方式 |
| issue_cost | float | 发行费用（元） |
| onl_code | str | 网上申购代码 |
| onl_name | str | 网上申购简称 |
| onl_date | str | 网上发行日期 |
| onl_size | float | 网上发行数量（张） |
| onl_pch_vol | float | 网上有效申购数量（张） |
| onl_pch_num | int | 网上有效申购户数 |
| onl_pch_excess | float | 网上超额认购倍数 |
| onl_winning_rate | float | 网上中签率（%） |
| shd_ration_code | str | 股东配售代码 |
| shd_ration_name | str | 股东配售简称 |
| shd_ration_date | str | 股东配售日期 |
| shd_ration_record_date | str | 股东配售股权登记日 |
| shd_ration_pay_date | str | 股东配售缴款日 |
| shd_ration_price | float | 股东配售价格 |
| shd_ration_ratio | float | 股东配售比例 |
| shd_ration_size | float | 股东配售数量（张） |
| shd_ration_vol | float | 股东有效申购数量（张） |
| shd_ration_num | int | 股东有效申购户数 |
| shd_ration_excess | float | 股东超额认购倍数 |
| offl_size | float | 网下发行数量（张） |
| offl_deposit | float | 网下保证金比例（%） |
| offl_pch_vol | float | 网下有效申购数量（张） |
| offl_pch_num | int | 网下有效申购户数 |
| offl_pch_excess | float | 网下超额认购倍数 |
| offl_winning_rate | float | 网下中签率 |
| lead_underwriter | str | 主承销商 |
| lead_underwriter_vol | float | 主承销商包销数量（张） |

## 调用示例

```bash
curl -X POST https://www.codebuddy.cn/v2/tool/financedata \
  -H "Content-Type: application/json" \
  -d '{
    "api_name": "cb_issue",
    "params": {"ts_code": "113009.SH"},
    "fields": "ts_code,ann_date,issue_size,issue_price,issue_type"
  }'
```
