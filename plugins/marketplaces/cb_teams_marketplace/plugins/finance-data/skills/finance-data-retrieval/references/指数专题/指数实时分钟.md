# 指数实时分钟

> 接口：`rt_idx_min` | 获取交易所指数实时分钟数据，支持1~60分钟频率（需单独权限）

## 请求参数

| 参数 | 类型 | 必选 | 说明 |
|------|------|------|------|
| ts_code | str | Y | 指数代码，支持逗号分隔多个（如000001.SH,399300.SZ） |
| freq | str | Y | 频率：1MIN/5MIN/15MIN/30MIN/60MIN |

## 返回字段

| 字段 | 类型 | 说明 |
|------|------|------|
| ts_code | str | 指数代码 |
| time | str | 交易时间 |
| open | float | 开盘价 |
| close | float | 收盘价 |
| high | float | 最高价 |
| low | float | 最低价 |
| vol | float | 成交量（股） |
| amount | float | 成交额（元） |

## 调用示例

```bash
curl -X POST https://www.codebuddy.cn/v2/tool/financedata \
  -H "Content-Type: application/json" \
  -d '{
    "api_name": "rt_idx_min",
    "params": {"ts_code": "000001.SH", "freq": "5MIN"},
    "fields": ""
  }'
```

## 返回示例

```json
{
  "code": 0,
  "msg": "",
  "data": {
    "fields": ["ts_code", "time", "open", "close", "high", "low", "vol", "amount"],
    "items": [
      ["000001.SH", "2025-02-28 15:00:00", 3320.12, 3320.89, 3321.50, 3318.71, 56831950.0, 75805489.0]
    ]
  }
}
```
