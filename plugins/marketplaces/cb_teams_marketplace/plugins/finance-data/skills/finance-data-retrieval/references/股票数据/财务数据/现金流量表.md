# 现金流量表

> 接口：`cashflow` | 获取上市公司现金流量表数据

## 请求参数

| 参数 | 类型 | 必选 | 说明 |
|------|------|------|------|
| ts_code | str | Y | TS股票代码 |
| ann_date | str | N | 公告日期 |
| f_ann_date | str | N | 实际公告日期 |
| start_date | str | N | 公告开始日期 |
| end_date | str | N | 公告结束日期 |
| period | str | N | 报告期（每个季度最后一天，如20240930） |
| report_type | str | N | 报告类型（1=合并报表，2=单季合并，6=母公司报表等） |
| comp_type | str | N | 公司类型（1=一般工商业，2=银行，3=保险，4=证券） |
| is_calc | int | N | 是否计算报表 |

## 返回字段

| 字段 | 类型 | 说明 |
|------|------|------|
| ts_code | str | TS代码 |
| ann_date | str | 公告日期 |
| f_ann_date | str | 实际公告日期 |
| end_date | str | 报告期 |
| comp_type | str | 公司类型 |
| report_type | str | 报告类型 |
| end_type | str | 报告期类型 |
| net_profit | float | 净利润 |
| c_fr_sale_sg | float | 销售商品、提供劳务收到的现金 |
| n_cashflow_act | float | 经营活动产生的现金流量净额 |
| n_cashflow_inv_act | float | 投资活动产生的现金流量净额 |
| n_cash_flows_fnc_act | float | 筹资活动产生的现金流量净额 |
| free_cashflow | float | 企业自由现金流量 |
| c_cash_equ_end_period | float | 期末现金及现金等价物余额 |
| update_flag | str | 更新标识（1=最新） |

## 调用示例

```bash
curl -X POST https://www.codebuddy.cn/v2/tool/financedata \
  -H "Content-Type: application/json" \
  -d '{
    "api_name": "cashflow",
    "params": {"ts_code": "000001.SZ", "period": "20240930"},
    "fields": ""
  }'
```
