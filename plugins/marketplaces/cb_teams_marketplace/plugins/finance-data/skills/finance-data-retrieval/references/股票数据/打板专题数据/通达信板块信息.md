# 通达信板块信息

> 接口：`tdx_index` | 获取通达信板块基本信息，包括概念、行业、风格和地域板块

## 请求参数

| 参数 | 类型 | 必选 | 说明 |
|------|------|------|------|
| ts_code | str | N | 板块代码（xxxxxx.TDX） |
| trade_date | str | N | 交易日期（YYYYMMDD格式） |
| idx_type | str | N | 板块类型：概念板块、行业板块、风格板块、地域板块 |

## 返回字段

| 字段 | 类型 | 说明 |
|------|------|------|
| ts_code | str | 板块代码 |
| trade_date | str | 交易日期 |
| name | str | 板块名称 |
| idx_type | str | 板块类型 |
| idx_count | int | 成分个数 |
| total_share | float | 总股本（亿） |
| float_share | float | 流通股本（亿） |
| total_mv | float | 总市值（亿） |
| float_mv | float | 流通市值（亿） |

## 调用示例

```bash
curl -X POST https://www.codebuddy.cn/v2/tool/financedata \
  -H "Content-Type: application/json" \
  -d '{
    "api_name": "tdx_index",
    "params": {"trade_date": "20250513"},
    "fields": "ts_code,name,idx_type,idx_count"
  }'
```
