# 周线行情

> 接口：`weekly` | 获取A股周线行情数据，每周最后一个交易日更新

## 请求参数

| 参数 | 类型 | 必选 | 说明 |
|------|------|------|------|
| ts_code | str | N | 股票代码（与trade_date二选一） |
| trade_date | str | N | 交易日期（每周最后一个交易日，YYYYMMDD） |
| start_date | str | N | 开始日期 |
| end_date | str | N | 结束日期 |

## 返回字段

| 字段 | 类型 | 说明 |
|------|------|------|
| ts_code | str | 股票代码 |
| trade_date | str | 交易日期 |
| close | float | 周收盘价 |
| open | float | 周开盘价 |
| high | float | 周最高价 |
| low | float | 周最低价 |
| pre_close | float | 上周收盘价 |
| change | float | 周涨跌额 |
| pct_chg | float | 周涨跌幅 |
| vol | float | 周成交量 |
| amount | float | 周成交额 |

## 调用示例

```bash
curl -X POST https://www.codebuddy.cn/v2/tool/financedata \
  -H "Content-Type: application/json" \
  -d '{
    "api_name": "weekly",
    "params": {"ts_code": "000001.SZ", "start_date": "20250101", "end_date": "20250131"},
    "fields": ""
  }'
```
