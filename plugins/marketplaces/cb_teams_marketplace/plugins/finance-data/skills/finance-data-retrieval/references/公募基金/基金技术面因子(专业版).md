# 基金技术面因子（专业版）

> 接口：`fund_factor_pro` | 获取场内基金每日技术面因子数据，覆盖全历史

## 请求参数

| 参数 | 类型 | 必选 | 说明 |
|------|------|------|------|
| ts_code | str | N | 基金代码 |
| start_date | str | N | 开始日期 |
| end_date | str | N | 结束日期 |
| trade_date | str | N | 交易日期 |

## 返回字段

### 基础行情

| 字段 | 类型 | 说明 |
|------|------|------|
| ts_code | str | 基金代码 |
| trade_date | str | 交易日期 |
| open | float | 开盘价 |
| high | float | 最高价 |
| low | float | 最低价 |
| close | float | 收盘价 |
| pre_close | float | 昨收价 |
| change | float | 涨跌额 |
| pct_change | float | 涨跌幅 |
| vol | float | 成交量（手） |
| amount | float | 成交额（千元） |

### 技术因子（50+个，均以`_bfq`后缀表示不复权）

| 类别 | 因子 |
|------|------|
| 趋势指标 | EMA、MA、MACD、TRIX、EXPMA、BBI、DFMA等 |
| 动量指标 | RSI、KDJ、BRAR、MTM、ROC、PSY等 |
| 波动指标 | ATR、BOLL、KTN、TAQ等 |
| 成交量指标 | OBV、MFI、EMV、VR等 |
| 其他指标 | CCI、CR、WR、DMI、DPO、MASS、XSII等 |

## 调用示例

```bash
curl -X POST https://www.codebuddy.cn/v2/tool/financedata \
  -H "Content-Type: application/json" \
  -d '{
    "api_name": "fund_factor_pro",
    "params": {"ts_code": "510050.SH"},
    "fields": "ts_code,trade_date,open,high,low,close,vol,macd_bfq,rsi_bfq_6"
  }'
```
