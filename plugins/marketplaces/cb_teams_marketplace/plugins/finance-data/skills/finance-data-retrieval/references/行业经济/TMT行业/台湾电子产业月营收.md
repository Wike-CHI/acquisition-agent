# 台湾电子产业月营收

> 接口：`tmt_twincome` | 获取台湾TMT电子产业各产品类别月度营收数据

## 请求参数

| 参数 | 类型 | 必选 | 说明 |
|------|------|------|------|
| date | str | N | 报告期 |
| item | str | Y | 产品代码 |
| start_date | str | N | 报告期开始日期 |
| end_date | str | N | 报告期结束日期 |

**说明：** 单次最多返回30个月数据，可通过分段日期多次请求获取更多数据。

## 返回字段

| 字段 | 类型 | 说明 |
|------|------|------|
| date | str | 报告期 |
| item | str | 产品代码 |
| op_income | str | 月营收 |

## 调用示例

```bash
curl -X POST https://www.codebuddy.cn/v2/tool/financedata \
  -H "Content-Type: application/json" \
  -d '{
    "api_name": "tmt_twincome",
    "params": {"item": "8"},
    "fields": ""
  }'
```

## 返回示例

```json
{
  "code": 0,
  "msg": "",
  "data": {
    "fields": ["date", "item", "op_income"],
    "items": [
      ["20180731", "8", 35144753.0],
      ["20180629", "8", 30940090.0],
      ["20180531", "8", 30982240.0]
    ]
  }
}
```

## 产品代码说明

平台支持65个产品类别，包括：PC、笔记本电脑、主板、PCB、IC基板、显卡、LED、太阳能、LCD面板、半导体、内存模组、手机、相机、软件服务等。
