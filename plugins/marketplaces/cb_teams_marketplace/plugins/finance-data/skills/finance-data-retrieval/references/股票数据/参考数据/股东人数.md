# 股东人数

> 接口：`stk_holdernumber` | 获取上市公司股东户数数据

## 请求参数

| 参数 | 类型 | 必选 | 说明 |
|------|------|------|------|
| ts_code | str | N | TS股票代码 |
| ann_date | str | N | 公告日期 |
| enddate | str | N | 截止日期 |
| start_date | str | N | 公告开始日期 |
| end_date | str | N | 公告结束日期 |

## 返回字段

| 字段 | 类型 | 说明 |
|------|------|------|
| ts_code | str | TS代码 |
| ann_date | str | 公告日期 |
| end_date | str | 截止日期 |
| holder_num | int | 股东户数 |

## 调用示例

```bash
curl -X POST https://www.codebuddy.cn/v2/tool/financedata \
  -H "Content-Type: application/json" \
  -d '{
    "api_name": "stk_holdernumber",
    "params": {"ts_code": "000001.SZ", "start_date": "20240101", "end_date": "20241231"},
    "fields": ""
  }'
```

## 返回示例

```json
{
  "code": 0,
  "msg": "",
  "data": {
    "fields": ["ts_code", "ann_date", "end_date", "holder_num"],
    "items": [
      ["000001.SZ", "20241019", "20240930", 517695],
      ["000001.SZ", "20240816", "20240630", 546413],
      ["000001.SZ", "20240420", "20240331", 567902]
    ]
  }
}
```
