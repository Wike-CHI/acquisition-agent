# 期货主要品种交易周报

> 接口：`fut_weekly_detail` | 获取期货交易所主要品种每周交易统计信息，数据自2010年3月起

## 请求参数

| 参数 | 类型 | 必选 | 说明 |
|------|------|------|------|
| week | str | N | 周期（每年第几周，如202001表示2020第1周） |
| prd | str | N | 期货品种（支持多品种逗号分隔） |
| start_week | str | N | 开始周期 |
| end_week | str | N | 结束周期 |
| exchange | str | N | 交易所 |
| fields | str | N | 提取字段（如fields='prd,name,vol'） |

## 返回字段

| 字段 | 类型 | 说明 |
|------|------|------|
| exchange | str | 交易所代码 |
| prd | str | 期货品种代码 |
| name | str | 品种名称 |
| vol | int | 成交量（手） |
| vol_yoy | float | 同比增减（%） |
| amount | float | 成交金额（亿元） |
| amout_yoy | float | 同比增减（%） |
| cumvol | int | 年累计成交总量（手） |
| cumvol_yoy | float | 同比增减（%） |
| cumamt | float | 年累计成交金额（亿元） |
| cumamt_yoy | float | 同比增减（%） |
| open_interest | int | 持仓量（手） |
| interest_wow | float | 环比增减（%） |
| mc_close | float | 本周主力合约收盘价 |
| close_wow | float | 环比涨跌（%） |
| week | str | 周期 |
| week_date | str | 周日期 |

## 调用示例

```bash
curl -X POST https://www.codebuddy.cn/v2/tool/financedata \
  -H "Content-Type: application/json" \
  -d '{
    "api_name": "fut_weekly_detail",
    "params": {"prd": "CU", "start_week": "202401", "end_week": "202403"},
    "fields": "exchange,prd,name,vol,amount,open_interest,mc_close,week"
  }'
```
