# 金融数据智能检索

根据用户的自然语言需求，自动检索金融数据。

## 工作流程

### Step 1: 理解用户意图

分析用户的自然语言请求，提取关键信息：
- **数据类型**：股票行情、财务报表、宏观指标、基金净值等
- **具体标的**：股票代码、指数名称、期货品种等
- **时间范围**：日期、时间段
- **数据粒度**：日线、分钟线、周线、月线等

### Step 2: 查找目标 API

根据提取的意图，在下方 **API 索引** 中找到匹配的接口名称（api_name）。

**匹配策略**：
1. 先根据大类缩小范围（如“沪深300成分股”→ 指数专题）
2. 再在子分类中定位具体接口（如“成分和权重”→ `index_weight`）
3. 如果用户意图模糊，选择最常用的接口并在返回结果时说明

**索引中找不到时的降级策略**：

如果在 API 索引中没有找到明确匹配的接口，不要直接放弃。按以下顺序尝试：

1. **浏览相关分类目录**：用 Glob 工具扫描可能相关的 references 子目录，查看是否有遍漏的接口文档
   ```
   Glob: references/股票数据/特色数据/*.md
   Glob: references/指数专题/*.md
   ```
2. **读取完整索引**：读取 `references/MAP.md` 查看完整的接口目录树，可能有索引未列出的接口
3. **按关键词搜索文档内容**：用 Grep 工具在 references 目录中搜索用户描述的关键词
   ```
   Grep: pattern="涨停" path="references/"
   Grep: pattern="北向资金" path="references/"
   ```
4. 如果以上方法都无法找到匹配接口，才告知用户该数据不在当前接口覆盖范围内

### Step 3: 查阅 API 详细文档

找到目标 api_name 后，**必须先读取对应的接口文档，再构造调用**。不要凭猜测填写参数。

**具体操作**：
1. 根据下方 **文档查阅路径映射**，确定文档所在目录
2. 用 Read 工具读取该文档，例如：
   ```
   Read: references/股票数据/行情数据/历史日线.md
   ```
3. 从文档中提取以下关键信息：
   - **输入参数**：哪些必填、哪些可选、参数类型和格式
   - **输出字段**：可返回的字段列表，用于按需选择 fields 参数
   - **参数格式约束**：日期格式（YYYYMMDD）、代码格式（000001.SZ）、枚举值等
   - **调用限制**：单次返回上限、权限要求等

### Step 4: 构造并执行 API 调用

使用 Bash 工具执行 curl 请求：

```bash
curl -s -X POST https://www.codebuddy.cn/v2/tool/financedata \
  -H "Content-Type: application/json" \
  -d '{
    "api_name": "<接口名>",
    "params": {<根据文档填写参数>},
    "fields": "<根据需求选择返回字段，留空返回全部>"
  }'
```

### Step 5: 解析和展示结果

API 返回格式为：
```json
{
  "code": 0,
  "msg": "",
  "data": {
    "fields": ["字段1", "字段2", "..."],
    "items": [["值1", "值2", "..."], ["..."]]
  }
}
```

- 将 fields + items 转换为可读表格展示
- 如果 code 不为 0，说明 msg 中的错误信息
- 数据量大时，摘要展示并说明总条数

## 重要注意事项

1. **代码格式**：股票代码带交易所后缀（如 `000001.SZ`、`600000.SH`、`430047.BJ`）
2. **日期格式**：大多数接口使用 `YYYYMMDD`（如 `20250301`），新闻类接口使用 `YYYY-MM-DD HH:mm:ss`
3. **必填参数**：部分接口有必填参数（如期货合约信息需要 exchange），务必从文档确认
4. **数据限量**：部分接口有单次返回上限，需要分页请求
5. **已停用接口**：标记"停"的接口不可调用，索引中已标注
6. **仅SDK接口**：标记"仅SDK"的接口不支持 HTTP 调用（如 `pro_bar` 复权行情、实时爬虫接口），改用对应的 HTTP 接口
7. **已下线接口**：标记"已下线"的接口服务端已移除，无法调用。reference 文档保留供参考
